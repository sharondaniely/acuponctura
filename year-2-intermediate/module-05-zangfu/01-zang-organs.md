# איברי זאנג — האיברים המלאים (脏)

## Zang Organs — The Solid (Yin) Organs

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לתאר את חמשת איברי הזאנג ותפקודיהם ברפואה הסינית
2. להבין את הרקמות, איברי החישה, הרגשות והנוזלים המשויכים לכל איבר
3. לזהות פתולוגיות נפוצות של כל איבר זאנג
4. להסביר את ההבדל המהותי בין איברי זאנג לאיברי פו

---

## מבוא

חמשת איברי הזאנג (五脏 Wu Zang) הם הלב, הכבד, הטחול, הריאות והכליות. בניגוד לאנטומיה המערבית, ברפואה הסינית כל "איבר" מייצג **מערכת פונקציונלית שלמה** הכוללת תפקודים פיזיים, רגשיים ורוחניים.

איברי הזאנג הם **איברים ין** — מלאים, מאחסנים את החומרים החיוניים (צ'י, דם, ג'ינג, שן), ואינם מפרישים. הם עומדים במרכז הפיזיולוגיה הסינית.

> "五脏者，藏精气而不泻也" (Wu Zang Zhe, Cang Jing Qi Er Bu Xie Ye)
> "חמשת איברי הזאנג מאחסנים תמציות וצ'י ואינם מפרישים"
> — הואנג די ניי ג'ינג

## 1. הלב (心 Xin)

הלב נחשב ל**"קיסר של כל האיברים"** (君主之官 Jun Zhu Zhi Guan). הוא מכוון את כל שאר האיברים ומהווה את מרכז התודעה.

### 1.1 תפקודים עיקריים

| תפקוד | הסבר |
|--------|-------|
| שולט בדם ובכלי הדם (主血脉 Zhu Xue Mai) | הלב מניע את הדם בכלי הדם; תנועה, קצב וחיוּת הדם תלויים בצ'י הלב |
| מאכלס את השן (藏神 Cang Shen) | השן (神) — התודעה, המחשבה, הזיכרון והשינה — כולם שוכנים בלב |
| נפתח ללשון (开窍于舌 Kai Qiao Yu She) | מצב הלב משתקף בלשון — צבעה, צורתה ויכולת הדיבור |
| מתבטא במראה הפנים (其华在面 Qi Hua Zai Mian) | פנים ורודות ובוהקות = לב בריא; חיוורון או כחלון = בעיה בלב |

### 1.2 שיוכים

- **רגש:** שמחה (喜 Xi) — שמחה מוגזמת מפזרת את צ'י הלב
- **נוזל:** זיעה (汗 Han) — "הזיעה היא נוזל הלב"
- **רקמה:** כלי הדם (脉 Mai)
- **יסוד:** אש (火 Huo)
- **עונה:** קיץ
- **צבע:** אדום
- **טעם:** מר

### 1.3 פתולוגיות נפוצות

- **חסר צ'י הלב (心气虚):** דפיקות, קוצר נשימה, עייפות, הזעה ספונטנית
- **חסר דם הלב (心血虚):** חרדה, נדודי שינה, חלומות מרובים, חיוורון, לשון חיוורת
- **חסר ין הלב (心阴虚):** חום בחמש, הזעות לילה, אי-שקט, לשון אדומה ללא ציפוי
- **אש הלב בוערת (心火亢盛):** כיבים בפה, אי-שקט חמור, שתן כהה, לשון אדומה עם קצה אדום בוהק
- **קיפאון דם הלב (心血瘀阻):** כאבי חזה דוקרים, שפתיים סגולות, לשון כהה עם כתמים

## 2. הכבד (肝 Gan)

הכבד נקרא **"הגנרל"** (将军之官 Jiang Jun Zhi Guan) — אחראי על תכנון אסטרטגי ועל זרימה חופשית.

### 2.1 תפקודים עיקריים

| תפקוד | הסבר |
|--------|-------|
| מבטיח זרימה חלקה של צ'י (疏泄 Shu Xie) | התפקוד החשוב ביותר — כאשר הזרימה חלקה, הרגשות מאוזנים, העיכול תקין והמחזור סדיר |
| מאחסן דם (藏血 Cang Xue) | מווסת את כמות הדם לפי הצורך — בפעילות: דם לשרירים; במנוחה: דם חוזר לכבד |
| שולט בגידים (主筋 Zhu Jin) | הגמישות, התנועתיות וכוח השרירים תלויים בהזנת הגידים ע"י דם הכבד |
| נפתח לעיניים (开窍于目 Kai Qiao Yu Mu) | הראייה תלויה בהזנת העיניים ע"י דם הכבד |
| מאכלס את ההון (藏魂 Cang Hun) | ההון (魂) — הנשמה האתרית, הדמיון, החזון והחלומות |

### 2.2 שיוכים

- **רגש:** כעס (怒 Nu) — כולל תסכול, מרירות, דיכאון מודחק
- **נוזל:** דמעות (泪 Lei)
- **רקמה:** גידים ורצועות (筋 Jin)
- **יסוד:** עץ (木 Mu)
- **עונה:** אביב
- **צבע:** ירוק/תכלת
- **טעם:** חמוץ

### 2.3 פתולוגיות נפוצות

- **קיפאון צ'י הכבד (肝气郁结):** הדפוס הנפוץ ביותר — תחושת גוש בגרון, נפיחות בצדדים, עצבנות, מחזור לא סדיר
- **עליית יאנג הכבד (肝阳上亢):** כאבי ראש, סחרחורת, טנטון, פנים אדומות, כעס פתאומי
- **רוח הכבד (肝风内动):** רעד, עוויתות, שבץ, סחרחורת חמורה
- **חסר דם הכבד (肝血虚):** ראייה מטושטשת, עיניים יבשות, ציפורניים שבירות, מחזור מועט, חיוורון

## 3. הטחול (脾 Pi)

הטחול הוא **"השר האחראי על התחבורה"** (仓廪之官 Cang Lin Zhi Guan) — מרכז העיכול וייצור הצ'י.

### 3.1 תפקודים עיקריים

| תפקוד | הסבר |
|--------|-------|
| שולט בהמרה והובלה (运化 Yun Hua) | מעכל את המזון, מפיק ממנו את תמציות המזון (谷气 Gu Qi) ומעביר אותן לכל הגוף |
| מעלה את הצ'י הטהור (升清 Sheng Qing) | מעלה את התמציות הטהורות כלפי מעלה לריאות וללב |
| שולט בדם (统血 Tong Xue) | שומר על הדם בתוך כלי הדם ומונע דימומים |
| שולט בשרירים ובגפיים (主肌肉/四肢) | שרירים חזקים וגפיים חמות = טחול בריא |
| נפתח לפה (开窍于口 Kai Qiao Yu Kou) | חוש הטעם וצורת השפתיים משקפים את מצב הטחול |
| מאכלס את הי (藏意 Cang Yi) | הי (意) — הכוונה, הריכוז, הלמידה והזיכרון |

### 3.2 שיוכים

- **רגש:** דאגה/הרהור יתר (思 Si)
- **נוזל:** רוק (涎 Xian)
- **רקמה:** שרירים (肌肉 Ji Rou)
- **יסוד:** אדמה (土 Tu)
- **עונה:** סוף קיץ / תקופות מעבר
- **צבע:** צהוב
- **טעם:** מתוק

### 3.3 פתולוגיות נפוצות

- **חסר צ'י הטחול (脾气虚):** עייפות, תיאבון ירוד, נפיחות, שלשול, שרירים חלשים
- **חסר יאנג הטחול (脾阳虚):** כנ"ל + קור בבטן, גפיים קרות, שלשול עם מזון לא מעוכל
- **הטחול לא שולט בדם (脾不统血):** דימום כרוני, וסת מרובה, חבורות, דם בצואה
- **לחות טחול (脾虚生湿):** כבדות בגוף, ציפוי לשון שמן, בחילה, הפרשות מרובות

## 4. הריאות (肺 Fei)

הריאות נקראות **"שר ממשלה ראשי"** (相傅之官 Xiang Fu Zhi Guan) — אחראיות על קצב וסדר.

### 4.1 תפקודים עיקריים

| תפקוד | הסבר |
|--------|-------|
| שולטות בצ'י ובנשימה (主气司呼吸) | קולטות את צ'י הטבע (清气) ומשלבות אותו עם צ'י המזון ליצירת צ'י אמיתי |
| שולטות בפיזור (宣发 Xuan Fa) | מפזרות ווי צ'י (צ'י מגן) ונוזלים לפני השטח של הגוף |
| שולטות בהורדה (肃降 Su Jiang) | מורידות צ'י ונוזלים כלפי מטה — לכליות ולשלפוחית |
| מווסתות מעברי מים (通调水道 Tong Tiao Shui Dao) | "המקור העליון של המים" — מפזרות ומורידות נוזלים |
| שולטות בעור ובשערות גוף (主皮毛 Zhu Pi Mao) | עור בריא ולח = ריאות חזקות |
| נפתחות לאף (开窍于鼻 Kai Qiao Yu Bi) | חוש הריח ונשימה תקינה דרך האף |
| מאכלסות את הפּוֹ (藏魄 Cang Po) | הפּוֹ (魄) — הנשמה הגופנית, האינסטינקטים, תחושות הגוף |

### 4.2 שיוכים

- **רגש:** עצב/אבל (悲 Bei / 忧 You)
- **נוזל:** ריר אף (涕 Ti)
- **רקמה:** עור ושערות גוף (皮毛 Pi Mao)
- **יסוד:** מתכת (金 Jin)
- **עונה:** סתיו
- **צבע:** לבן
- **טעם:** חריף

### 4.3 פתולוגיות נפוצות

- **חסר צ'י הריאות (肺气虚):** קוצר נשימה, קול חלש, שיעול חלש, הזעה ספונטנית, חיסון חלש
- **חסר ין הריאות (肺阴虚):** שיעול יבש, גרון יבש, קול צרוד, חום בחמש, הזעות לילה
- **רוח-קור תוקפת את הריאות (风寒犯肺):** שיעול, ריר שקוף, גודש באף, צמרמורת
- **רוח-חום תוקפת את הריאות (风热犯肺):** שיעול, ריר צהוב, כאב גרון, חום
- **ליחה חוסמת את הריאות (痰湿阻肺):** שיעול עם כיח רב, לחות בחזה, קוצר נשימה

## 5. הכליות (肾 Shen)

הכליות נקראות **"שורש החיים"** (先天之本 Xian Tian Zhi Ben) — שורש ין ויאנג של כל הגוף.

### 5.1 תפקודים עיקריים

| תפקוד | הסבר |
|--------|-------|
| מאחסנות ג'ינג (藏精 Cang Jing) | מאחסנות את התמצית — הן של "לפני השמיים" (תורשתית) והן של "אחרי השמיים" (נרכשת) |
| שולטות במים (主水 Zhu Shui) | מווסתות את חילוף החומרים של נוזלי הגוף, מפרידות טהור מעכור |
| קולטות צ'י (纳气 Na Qi) | "אוחזות" בצ'י שהריאות שולחות כלפי מטה — חיוניות לנשימה עמוקה |
| שולטות בעצמות ומייצרות מוח עצם (主骨生髓) | עצמות חזקות ומוח תקין תלויים בג'ינג הכליות |
| נפתחות לאוזניים (开窍于耳 Kai Qiao Yu Er) | שמיעה טובה = כליות חזקות; טנטון = חסר כליות |
| מתבטאות בשיער הראש (其华在发 Qi Hua Zai Fa) | שיער עבה ובוהק = ג'ינג מלא; שיער דליל ושיבה מוקדמת = חסר ג'ינג |
| מאכלסות את הג'י (藏志 Cang Zhi) | הג'י (志) — כוח הרצון, ההתמדה והמוטיבציה |
| שולטות בשערי החיים (命门 Ming Men) | שער החיים — מקור אש הכליות המחממת את כל הגוף |

### 5.2 שיוכים

- **רגש:** פחד (恐 Kong) ובהלה (惊 Jing)
- **נוזל:** רוק עבה (唾 Tuo)
- **רקמה:** עצמות (骨 Gu)
- **יסוד:** מים (水 Shui)
- **עונה:** חורף
- **צבע:** שחור / כהה
- **טעם:** מלוח

### 5.3 פתולוגיות נפוצות

- **חסר ין הכליות (肾阴虚):** חום בחמש, הזעות לילה, סחרחורת, טנטון, כאבי גב, לשון אדומה
- **חסר יאנג הכליות (肾阳虚):** קור בגפיים, כאבי גב, תכיפות שתן, אימפוטנציה, לשון חיוורת נפוחה
- **חסר ג'ינג הכליות (肾精不足):** התפתחות איטית, אינפרטיליות, הזדקנות מוקדמת, ירידה בזיכרון
- **הכליות לא אוחזות בצ'י (肾不纳气):** קוצר נשימה חמור (קשה לשאוף), אסתמה כרונית

## נקודות חשובות לזכירה

1. **כל איבר זאנג מאחסן** ואינו מפריש — זו תכונת הין שלו
2. **כל איבר מזווג** עם איבר פו (ראו שיעור הבא) ועם יסוד מחמשת היסודות
3. **הקשר רגש-איבר הוא דו-כיווני**: רגש מוגזם פוגע באיבר, ופגיעה באיבר גורמת לחוסר איזון רגשי
4. **ברפואה הסינית "איבר" אינו רק איבר אנטומי** — הוא מערכת תפקודית רחבה
5. **הכליות הן השורש** של כל האיברים, כי הן מאחסנות את הג'ינג שממנו נובעים ין ויאנג של כל הגוף
