# מסימפטומים לדפוס - שלבי ניתוח

## From Symptoms to Pattern - Analysis Steps

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לסווג ממצאים לפי שמונת העקרונות (בא גאנג — 八纲)
2. לזהות את האיבר או הערוץ המעורב
3. להגיע לדפוס ספציפי בצורה שיטתית
4. ליישם את התהליך על דוגמאות קליניות

---

## 1. מבוא — ארבעת שלבי הניתוח

```
ממצאים גולמיים
      │
      ▼
[שלב 1] סיווג לפי שמונת העקרונות (八纲辨证)
         פנים/חוץ? חום/קור? עודף/חסר? ין/יאנג?
      │
      ▼
[שלב 2] זיהוי איבר/ערוץ מעורב
         איזה זאנג-פו? איזה ערוץ?
      │
      ▼
[שלב 3] קביעת דפוס ספציפי
         שם הסינדרום המדויק
      │
      ▼
[שלב 4] אימות — האם הכל מתאים?
```

## 2. שלב 1: שמונת העקרונות (八纲辨证 Ba Gang Bian Zheng)

### 2.1 ציר 1: פנים או חוץ (里/表 Li/Biao)

**שאלה**: האם הבעיה **שטחית** (ברמת העור, השרירים, הערוצים) או **עמוקה** (באיברים הפנימיים)?

| סימנים חיצוניים (表证) | סימנים פנימיים (里证) |
|---|---|
| התחלה פתאומית | התחלה הדרגתית |
| צמרמורת + חום | חום בלבד, או קור בלבד |
| כאב גוף כללי | סימפטומים של איבר ספציפי |
| דופק צף (浮) | דופק עמוק (沉) |
| ציפוי לשון דק | ציפוי לשון עבה |
| קשור לחשיפה חיצונית | לא קשור למזג אוויר |

### 2.2 ציר 2: חום או קור (热/寒 Re/Han)

**שאלה**: האם הדפוס **חם** או **קר**?

| סימני חום (热) | סימני קור (寒) |
|---|---|
| פנים אדומות | פנים חיוורות |
| תחושת חום | תחושת קור |
| צמא למים קרים | העדפה לשתייה חמה |
| שתן כהה | שתן בהיר |
| עצירות יבשה | שלשול מימי |
| לשון אדומה, ציפוי צהוב | לשון חיוורת, ציפוי לבן |
| דופק מהיר | דופק איטי |

### 2.3 ציר 3: עודף או חסר (实/虚 Shi/Xu)

**שאלה**: האם יש **יותר מדי** (פתוגן, סטגנציה) או **חסר** (צ'י, דם, ין, יאנג)?

| סימני עודף (实) | סימני חסר (虚) |
|---|---|
| מחלה חריפה | מחלה כרונית |
| קול חזק | קול חלש |
| כאב שמחמיר בלחיצה | כאב שמוקל בלחיצה |
| נשימה כבדה | קוצר נשימה, עייפות |
| עצירות חזקה | צואה רכה |
| דופק מלא | דופק ריק |
| לשון עם ציפוי עבה | לשון ללא ציפוי |

### 2.4 ציר 4: ין או יאנג (阴/阳 Yin/Yang)

זהו **סיכום כולל**:
- **דפוס ין** = פנימי + קור + חסר (שקט, קר, חלש)
- **דפוס יאנג** = חיצוני + חום + עודף (סוער, חם, חזק)

## 3. שלב 2: זיהוי האיבר המעורב

### 3.1 מילות מפתח לפי איבר

| איבר | סימפטומים מובילים |
|---|---|
| **לב (心)** | פלפיטציות, נדודי שינה, חרדה, כיבים בלשון, שכחה |
| **ריאות (肺)** | שיעול, קוצר נשימה, בעיות עור, נזלת, קול חלש |
| **כבד (肝)** | כאב בצלעות, כעס, עצבנות, עיניים, ציפורניים, מחזור |
| **טחול (脾)** | עייפות, נפיחות, חוסר תיאבון, צואה רכה, כבדות |
| **כליות (肾)** | גב תחתון, ברכיים, שמיעה, שתן, רבייה, עצמות, שיער |
| **שק מרה (胆)** | כאב בצלעות, טעם מר, חוסר החלטיות |
| **קיבה (胃)** | כאב אפיגסטרי, בחילות, הקאות, תיאבון |
| **מעי גס (大肠)** | עצירות, שלשול, כאב בטן תחתון |
| **שלפוחית (膀胱)** | בעיות שתן, כאב גב תחתון |

### 3.2 סימנים מאשרים מהלשון

| אזור לשון | איבר |
|---|---|
| קצה אדום | לב |
| צדדים אדומים | כבד |
| מרכז עם ציפוי עבה | טחול/קיבה |
| שורש ללא ציפוי | כליות |

### 3.3 סימנים מאשרים מהדופק

| עמדה | איבר | ממצא |
|---|---|---|
| צון שמאל חלש | לב | חסר דם/צ'י לב |
| גואן שמאל מתוח | כבד | סטגנציית צ'י כבד |
| גואן ימין חלש | טחול | חסר צ'י טחול |
| צ'י חלש | כליות | חסר כליות |

## 4. שלב 3: קביעת דפוס ספציפי

שלב זה משלב את ממצאי שלבים 1 ו-2 לשם דפוס מדויק.

### 4.1 דוגמה: בעיה בכבד

```
שלב 1: פנים + (חום או סטגנציה) + (עודף או מעורב)
שלב 2: כבד (צלעות, כעס, מחזור, עיניים)
שלב 3: מה בדיוק?
```

| דפוס | ממצאים מבדילים |
|---|---|
| סטגנציית צ'י כבד (肝气郁结) | כאב צלעות, אנחות, מתח, דופק מתוח, לשון רגילה |
| אש כבד (肝火上炎) | כאב ראש, עיניים אדומות, כעס, מר בפה, לשון אדומה צדדים, דופק מתוח מהיר |
| רוח כבד (肝风内动) | רעד, סחרחורת, עוויתות, לשון סוטה |
| חסר דם כבד (肝血虚) | עיניים יבשות, ציפורניים שבירות, חלומות, לשון חיוורת |
| חסר ין כבד (肝阴虚) | סחרחורת, עיניים יבשות, חום אחה"צ, לשון אדומה |

---

## 5. שלב 4: אימות — בדיקת עקביות

### 5.1 שאלות לבדיקה עצמית

1. **האם הלשון תואמת את הדופק?** אם לא — מה הסיבה?
2. **האם כל הסימפטומים מוסברים?** אם נשארו סימפטומים "יתומים" — ייתכן שיש דפוס שני
3. **האם הדפוס הגיוני?** האם ניתן להסביר את השתלשלות המחלה?
4. **האם יש סתירות?** סתירות מעידות על מורכבות — ראו מודול על דפוסים מורכבים

---

## 6. דוגמאות מלאות

### דוגמה א: כאב ראש כרוני

**ממצאים**:
- כאב ראש בצדדים (רקות), 3 שנים
- מחמיר בעת מתח ולחץ
- מוקל במנוחה ובטיול
- עצבנות, אנחות תכופות, כאב בצלעות
- מחזור לא סדיר עם כאב לפני
- **לשון**: צדדים אדומים מעט, ציפוי דק
- **דופק**: מתוח (弦), במיוחד בגואן שמאל

**ניתוח**:
1. שמונת העקרונות: פנימי, לא חום ולא קור ברורים, מעורב
2. איבר: **כבד** (צדדים, צלעות, כעס, מחזור, דופק מתוח)
3. דפוס: **סטגנציית צ'י כבד** (肝气郁结)
4. אימות: כאב ראש בצדדים = ערוץ שאו יאנג (שק מרה = צמד הכבד) ✓

### דוגמה ב: עייפות כרונית

**ממצאים**:
- עייפות כבדה, 6 חודשים
- גרוע יותר אחרי אכילה
- נפיחות בטנית, צואה רכה
- חוסר תיאבון, כבדות בגפיים
- **לשון**: חיוורת, נפוחה, סימני שיניים, ציפוי לבן
- **דופק**: ריק, חלש בגואן ימין

**ניתוח**:
1. שמונת העקרונות: פנימי, לא חם (חיוור, לבן), חסר
2. איבר: **טחול** (עיכול, נפיחות, צואה רכה, כבדות, דופק גואן ימין)
3. דפוס: **חסר צ'י טחול** (脾气虚)
4. אימות: לשון חיוורת+נפוחה+סימני שיניים = קלאסי לחסר טחול ✓

### דוגמה ג: נדודי שינה

**ממצאים**:
- קושי להירדם, ומתעוררת ב-3:00 בלילה
- חלומות רבים, חרדה
- פלפיטציות, שכחה
- סחרחורת קלה, ראייה מטושטשת
- **לשון**: חיוורת, דקה, קצה אדום מעט
- **דופק**: דק (细), חלש בצון שמאל

**ניתוח**:
1. שמונת העקרונות: פנימי, מעורב (חיוור אבל קצה אדום), חסר
2. איברים: **לב** (שינה, חרדה, פלפיטציות) + **כבד** (3:00 = שעת הכבד, ראייה)
3. דפוס: **חסר דם לב** (心血虚) עם **חסר דם כבד** (肝血虚)
4. אימות: לשון חיוורת ודקה = חסר דם ✓, קצה אדום = חום קל מחסר ✓

## 7. מלכודות בתהליך הניתוח

| מלכודת | הסבר | כיצד להימנע |
|---|---|---|
| **קיבוע מוקדם** | "אבחנתי" אחרי הסימפטום הראשון | אספו הכל לפני שמנתחים |
| **התעלמות מהלשון** | "הלשון לא התאימה אז התעלמתי" | הלשון לא משקרת — אם לא מתאימה, האבחנה שלכם אולי שגויה |
| **הטיה לדפוס מוכר** | "אני תמיד מוצא חסר טחול" | בדקו — האם באמת? או שאתם פשוט מכירים אותו הכי טוב? |
| **דפוס יחיד** | "חייב להיות דפוס אחד שמסביר הכל" | רוב המטופלים מציגים 2-3 דפוסים |

---

## 8. שאלות לחזרה

1. פרט את ארבעת הצירים של שמונת העקרונות ומה כל ציר שואל.
2. כיצד מזהים איזה איבר מעורב?
3. מה לעשות כאשר הלשון והדופק סותרים?
4. נתחו: מטופל עם כאב גב תחתון, שתן תכוף בלילה, ברכיים חלשות, לשון חיוורת ורטובה, דופק עמוק ואיטי.
5. מהי "קיבוע מוקדם" וכיצד להימנע ממנו?
