    מעקרון טיפולי לבחירת נקודות¶

    From Treatment Principle to Point Selection¶

    מטרות למידה¶

    בסיום שיעור זה, הסטודנט יוכל: 1. לבחור נקודות דיקור בהתאם לעקרון הטיפולי 2. להשתמש באסטרטגיות בחירה שונות (מקומי, דיסטלי, שילובים) 3. לקבוע כמות נקודות מתאימה 4. לשלב קטגוריות נקודות מיוחדות לטיפול יעיל

    1. מבוא — מעקרון לנקודות¶

    עקרון הטיפול אומר לנו מה לעשות. בחירת הנקודות אומרת איפה ואיך לעשות את זה.

    עקרון: "החלק כבד, הנע צ'י"
         │
         ▼
שאלה: אילו נקודות מחליקות את הכבד ומניעות צ'י?
         │
         ▼
תשובה: LR3, LR14, PC6, GB34...

    2. אסטרטגיות בחירת נקודות¶

    נקודות באזור הבעיה — מטפלות בענף (ביאו):

    בעיה נקודות מקומיות
    כאב ראש מצחי Yintang, ST8, GV24
    כאב ברך ST35, Xiyan, SP9
    כאב בטן REN12, ST25, REN6
    כאב כתף LI15, SI9, SJ14
    כאב גב תחתון BL23, BL25, GV4

    נקודות רחוקות מאזור הבעיה, על הערוץ הרלוונטי — מטפלות בשורש (בן):

    בעיה ערוץ נקודות דיסטליות

    כאב אוזן SJ17, SI19, GB2

    3. שילובי נקודות קלאסיים¶

    3.1 יואן + לואו (Yuan Luo Pei Xue)¶

    שילוב נקודת יואן (מקור) של האיבר הפגוע עם נקודת לואו (חיבור) של הערוץ המצומד:

    איבר פגוע יואן + לואו של הצמד
    ריאות LU9 + LI6
    מעי גס LI4 + LU7
    כבד LR3 + GB37
    שק מרה GB40 + LR5
    טחול SP3 + ST40
    קיבה ST42 + SP4

    3.2 שו-גב + מו-קדמי (Shu Mu Pei Xue)¶

    שילוב נקודת שו (גב) עם נקודת מו (קדמי) של אותו איבר — חיזוק כפול:

    איבר שו (גב) מו (קדמי)
    ריאות BL13 LU1
    לב BL15 REN14
    כבד BL18 LR14
    שק מרה BL19 GB24
    טחול BL20 LR13
    קיבה BL21 REN12
    כליות BL23 GB25

    3.3 שמאל-ימין¶

    דקירה בצד הנגדי לבעיה, או בשני הצדדים לאיזון: - כאב בצד ימין → דקירה בצד שמאל (או בשני הצדדים)

    3.4 מעלה-מטה¶

    שילוב נקודה בחלק העליון עם נקודה בחלק התחתון:

    שילוב שימוש
    LI4 + LR3 ("ארבע שערים") הנעת צ'י ודם כללית
    GV20 + REN1 חיבור מעלה-מטה, העלאת יאנג
    PC6 + SP4 בעיות חזה ובטן (ערוצים מיוחדים)
    LU7 + KI6 בעיות גרון, ריאות (ערוצים מיוחדים)
    SJ5 + GB41 בעיות צד הגוף, אוזניים (ערוצים מיוחדים)
    SI3 + BL62 בעיות עורף, עמוד שדרה (ערוצים מיוחדים)

    4. כמה נקודות לבחור?¶

    4.1 כללי אצבע¶

    גישה מספר נקודות מתי
    מינימליסטית 3-5 דפוס ברור, מטופל חלש, ילדים, קשישים
    סטנדרטית 6-10 רוב המקרים
    מקסימלית 10-15 דפוסים מורכבים, מטופל חזק, מטופל שלא מגיב

    4.2 עקרון¶

    "פחות זה יותר" — פחות נקודות = אנרגיה ממוקדת יותר לכל נקודה

    מטפלים מתחילים נוטים להשתמש ביותר מדי נקודות. עם הניסיון, המספר יורד.

    5. דוגמאות שלמות — מדפוס לנקודות¶

    דוגמה 1: סטגנציית צ'י כבד¶

    • עקרון: החלק כבד, הנע צ'י
    • נקודות:
    • LR3 (טאי צ'ונג) — יואן כבד, מחליקה צ'י כבד [דיסטלי + פעולה]
    • LR14 (צ'י מן) — מו כבד, פותחת חסימות מקומיות [מו]
    • PC6 (ניי גואן) — מרגיעה את החזה, מניעה צ'י [פעולה]
    • GB34 (יאנג לינג צ'ואן) — מחליקה צ'י שק מרה (צמד הכבד) [דיסטלי]
    • שילוב: LR3 (יואן כבד) + GB37 (לואו שק מרה) = יואן-לואו

    דוגמה 2: חסר צ'י טחול¶

    • עקרון: חזק צ'י טחול
    • נקודות:
    • ST36 (צו סאן לי) — מחזקת צ'י וטחול/קיבה [דיסטלי + פעולה]
    • SP6 (סאן ין ג'יאו) — מחזקת טחול, תזן דם [דיסטלי]
    • BL20 (פי שו) — שו טחול, חיזוק ישיר [שו]
    • REN12 (ג'ונג ואן) — מו קיבה, חיזוק עיכול [מו]
    • REN6 (צ'י האי) — מחזקת צ'י כללי [פעולה]
    • שילוב: BL20 (שו טחול) + LR13 (מו טחול) = שו-מו
    • טכניקה: בו (חיזוק) + מוקסה על ST36 ו-REN6

    דוגמה 3: חסר ין כליות¶

    • עקרון: תזן ין כליות
    • נקודות:
    • KI3 (טאי שי) — יואן כליות, תזן ין כליות [דיסטלי + יואן]
    • KI6 (ג'או האי) — מחזקת ין, פותחת ין צ'יאו מאי [פעולה]
    • SP6 (סאן ין ג'יאו) — מחזקת שלושת הין [דיסטלי]
    • BL23 (שן שו) — שו כליות [שו]
    • REN4 (גואן יואן) — מחזקת כליות, תזן ג'ינג [פעולה]
    • טכניקה: בו (חיזוק) בעדינות. ללא מוקסה (חום עלול להחמיר חסר ין)

    6. כללים חשובים¶

    6.1 לכל נקודה צריכה סיבה¶

    אל תוסיפו נקודות "למקרה". לכל נקודה שנבחרת, צריך להיות נימוק ברור: למה דווקא נקודה זו? מה היא תורמת לעקרון הטיפולי?

    6.2 התאמה אישית¶

    שני מטופלים עם אותו דפוס עשויים לקבל נקודות שונות — בגלל מבנה גוף, גיל, מצב כללי, או תלונות נלוות.

    6.3 שינוי מטיפול לטיפול¶

    אל תחזרו על אותן נקודות בדיוק בכל טיפול. התאימו בהתאם לשינויים שמטופל מדווח, ובהתאם לממצאי לשון ודופק עדכניים.

    7. שאלות לחזרה¶

    1. מהם ארבע אסטרטגיות בחירת נקודות (מקומי, דיסטלי, סמוך, פעולה)?
    2. הסבירו את שילוב יואן-לואו. בחרו זוג אחד ותנו דוגמה קלינית.
    3. הסבירו את שילוב שו-מו. מתי נשתמש בו?
    4. מדוע "פחות זה יותר" בבחירת נקודות?
    5. בחרו נקודות לדפוס: אש כבד. נמקו כל בחירה.
