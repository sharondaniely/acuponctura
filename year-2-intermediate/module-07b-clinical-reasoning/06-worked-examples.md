# עשר דוגמאות מפורטות — מתחילה ועד סוף

## Ten Complete Worked Examples — From Start to Finish

---

## מטרות למידה

בסיום לימוד שיעור זה, הסטודנט יוכל:
1. לעבור את כל התהליך האבחוני מאיסוף ממצאים ועד בחירת נקודות
2. לראות כיצד התיאוריה מיושמת בפרקטיקה
3. לזהות דפוסים נפוצים ואת הטיפול המתאים להם
4. להשתמש בדוגמאות כמודל לטיפולים עצמאיים

---

## כיצד להשתמש בדוגמאות אלו

לכל דוגמה מוצגים שבעה שלבים:

1. **הצגת המטופל** — מי, מה, מתי
2. **ממצאי אבחון** — ארבע שיטות + לשון + דופק
3. **ניתוח לפי שמונת העקרונות** — פנימי/חיצוני, חום/קור, עודף/חסר
4. **זיהוי דפוס** — הדפוס הספציפי
5. **עקרון טיפולי** — מה צריך לעשות
6. **בחירת נקודות** — אילו נקודות ומדוע
7. **נוסחת צמחים** — רפרנס לנוסחה קלאסית מתאימה

## דוגמה 1: כאב ראש מתחי כרוני

### הצגת המטופל
גבר בן 42, מנהל בחברת הייטק. כאב ראש כרוני כבר שנתיים, מחמיר בלחץ בעבודה.

### ממצאים
- כאב ראש בצדדים (רקות), לפעמים מקרין לעיניים
- עצבנות, נטייה לכעוס על דברים קטנים
- טעם מר בפה, במיוחד בבוקר
- שינה לא טובה, קושי להירדם
- כאב ונוקשות בצלעות
- לשון: צדדים אדומים, קצת נפוחים, ציפוי צהוב דק
- דופק: מתוח (弦 Xian), מהיר מעט, חזק בגואן שמאל

### ניתוח — שמונת העקרונות
- **פנימי/חיצוני**: פנימי (כרוני, ללא צמרמורת)
- **חום/קור**: חום (טעם מר, ציפוי צהוב, דופק מהיר)
- **עודף/חסר**: עודף (כאב חזק, דופק מתוח, קול רם)
- **ין/יאנג**: יאנג

### דפוס מזוהה
**אש כבד עולה** (肝火上炎 Gan Huo Shang Yan)

### עקרון טיפולי
נקה אש כבד, הורד עלייה (清肝泻火, 降逆)

### נקודות
| נקודה | נימוק |
|---|---|
| LR2 | נקודת ינג (荥) — מנקה אש כבד (נקודת הבחירה הראשונה לאש כבד) |
| LR3 | יואן כבד — מווסתת צ'י כבד |
| GB20 | מרגיעה עליית יאנג, מקלה כאב ראש צדדי |
| GB34 | נקודת הוי גידים, מרפה מתח |
| LI4 | עוצרת כאב, במיוחד בראש |
| Taiyang | נקודת אקסטרה — כאב רקות |
| **טכניקה**: פיזור (泻 Xie) בכל הנקודות |

### נוסחת צמחים (רפרנס)
**לונג דאן שיה גאן טאנג** (龙胆泻肝汤) — מנקה אש ולחות-חום מהכבד
