# מדפוס לעקרון טיפולי

## From Pattern to Treatment Principle

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לגזור עקרון טיפולי מדפוס מזוהה
2. לנסח עקרונות טיפול בשפה המקצועית של הרפואה הסינית
3. להבין את ההיגיון מאחורי כל עקרון
4. להבחין בין טיפול בשורש (בן — 本) לטיפול בענף (ביאו — 标)

---

## 1. מבוא — מדפוס לעקרון

לאחר שזיהינו את הדפוס, השלב הבא הוא לקבוע **עקרון טיפולי** (治则 Zhi Ze). העקרון קובע את **הכיוון** של הטיפול — מה אנחנו רוצים להשיג.

**חוק יסוד**:
> 寒者热之，热者寒之 (Han Zhe Re Zhi, Re Zhe Han Zhi)
> "את הקר — חממו, את החם — קררו"

> 虚则补之，实则泻之 (Xu Ze Bu Zhi, Shi Ze Xie Zhi)
> "את החסר — חזקו, את העודף — פנו"

## 2. עקרונות טיפול כלליים

### 2.1 לפי שמונת העקרונות

| דפוס | עקרון טיפולי | סינית |
|---|---|---|
| חיצוני (表) | שחרר את השטח | 解表 (Jie Biao) |
| פנימי (里) | טפל בפנים (לפי הדפוס הספציפי) | 治里 (Zhi Li) |
| חום (热) | נקה חום | 清热 (Qing Re) |
| קור (寒) | חמם | 温 (Wen) |
| עודף (实) | פנה/הפחת | 泻 (Xie) |
| חסר (虚) | חזק/תזן | 补 (Bu) |

| עקרון | סינית | מתי |

| תזן דם | 补血 (Bu Xue) | חסר דם |
| תזן ין | 滋阴 (Zi Yin) | חסר ין |
| חמם יאנג | 温阳 (Wen Yang) | חסר יאנג |
| הנע צ'י | 行气 (Xing Qi) | סטגנציית צ'י |
| הנע דם | 活血 (Huo Xue) | סטגנציית דם |
| נקה חום | 清热 (Qing Re) | חום פנימי |

## 3. עקרונות לפי דפוסי איברים — טבלה מקיפה

### 3.1 דפוסי לב (心)

| דפוס | עקרון טיפולי |
|---|---|
| חסר דם לב (心血虚) | תזן דם, הרגע רוח הלב (养血安神) |
| חסר ין לב (心阴虚) | תזן ין לב, נקה חום מחסר (滋阴清热) |
| חסר צ'י לב (心气虚) | חזק צ'י הלב (补心气) |
| חסר יאנג לב (心阳虚) | חמם יאנג הלב (温心阳) |
| אש לב (心火亢盛) | נקה אש לב, הרגע רוח (清心火, 安神) |
| סטגנציית דם לב (心血瘀阻) | הנע דם, פתח חסימות (活血化瘀, 通络) |
| פלגמה-אש חוסמת לב (痰火扰心) | נקה חום, המס פלגמה, פתח פתחי הלב (清热化痰, 开窍) |

### 3.2 דפוסי ריאות (肺)

| דפוס | עקרון טיפולי |
|---|---|
| רוח-קור בריאות (风寒犯肺) | שחרר שטח, פזר קור, פתח ריאות (解表散寒, 宣肺) |
| רוח-חום בריאות (风热犯肺) | שחרר שטח, נקה חום, פתח ריאות (解表清热, 宣肺) |
| חסר צ'י ריאות (肺气虚) | חזק צ'י ריאות (补肺气) |
| חסר ין ריאות (肺阴虚) | תזן ין ריאות, הלח (滋阴润肺) |
| פלגמה-לחות בריאות (痰湿阻肺) | ייבש לחות, המס פלגמה, הורד צ'י ריאות (燥湿化痰, 降肺气) |

### 3.3 דפוסי כבד (肝)

| דפוס | עקרון טיפולי |
|---|---|
| סטגנציית צ'י כבד (肝气郁结) | החלק כבד, הנע צ'י (疏肝理气) |
| אש כבד (肝火上炎) | נקה אש כבד (清肝泻火) |
| רוח פנימית כבד (肝风内动) | הרגע כבד, הרגע רוח (平肝熄风) |
| חסר דם כבד (肝血虚) | תזן דם כבד (补肝血) |
| חסר ין כבד (肝阴虚) | תזן ין כבד (滋肝阴) |
| יאנג כבד עולה (肝阳上亢) | הרגע כבד, הורד יאנג (平肝潜阳) |
| לחות-חום כבד/שק מרה (肝胆湿热) | נקה חום, ייבש לחות בכבד/שק מרה (清利肝胆湿热) |

### 3.4 דפוסי טחול (脾)

| דפוס | עקרון טיפולי |
|---|---|
| חסר צ'י טחול (脾气虚) | חזק צ'י טחול (健脾益气) |
| חסר יאנג טחול (脾阳虚) | חמם יאנג טחול, חזק טחול (温脾阳, 健脾) |
| צ'י טחול שוקע (脾气下陷) | העלה יאנג טחול (升提脾阳) |
| טחול לא מחזיק דם (脾不统血) | חזק טחול, החזק דם (补脾统血) |
| לחות-קור טחול (寒湿困脾) | חמם ויבש, חזק טחול (温中燥湿, 健脾) |
| לחות-חום טחול (脾胃湿热) | נקה חום, ייבש לחות (清热燥湿) |

### 3.5 דפוסי כליות (肾)

| דפוס | עקרון טיפולי |
|---|---|
| חסר ין כליות (肾阴虚) | תזן ין כליות (滋肾阴) |
| חסר יאנג כליות (肾阳虚) | חמם יאנג כליות (温肾阳) |
| חסר ג'ינג כליות (肾精不足) | חזק ג'ינג, תזן כליות (补肾填精) |
| כליות לא אוחזות צ'י (肾不纳气) | חזק כליות, אחוז צ'י (补肾纳气) |
| כליות לא שולטות במים (肾虚水泛) | חמם כליות, הנע מים (温肾利水) |

## 4. שורש וענף (本标 Ben Biao)

### 4.1 עיקרון

- **בן (本)** = השורש, הסיבה העמוקה
- **ביאו (标)** = הענף, הסימפטום הגלוי

**כלל**: בדרך כלל, טפלו **בשורש** (בן). אך במצב חריף — טפלו **בענף** (ביאו) קודם.

### 4.2 דוגמאות

| מצב | שורש (בן) | ענף (ביאו) | מה לטפל קודם? |
|---|---|---|---|
| חסר טחול + שלשול חריף | חסר טחול | שלשול | ענף — עצרו שלשול, אז חזקו טחול |
| חסר ין כליות + כאב גב כרוני | חסר ין | כאב | שורש — תזנו ין, הכאב ייפסק |
| סטגנציית כבד + כאב ראש חריף | סטגנציית כבד | כאב ראש | ענף — הקלו כאב ראש, אז טפלו בכבד |
| חסר דם + נדודי שינה כרוניים | חסר דם | נדודי שינה | שורש — תזנו דם, השינה תשתפר |

### 4.3 כלל מעשי

> 急则治标，缓则治本 (Ji Ze Zhi Biao, Huan Ze Zhi Ben)
> "בחריף — טפלו בענף; בכרוני — טפלו בשורש"

לעיתים: **טפלו בשניהם בו-זמנית** (标本同治 Biao Ben Tong Zhi) — כאשר גם השורש וגם הענף דחופים, או כאשר הם קשורים הדדית.

## 5. ניסוח עקרון טיפולי — מבנה מומלץ

**פורמט**: [פעולה] + [מה] + [של מה]

דוגמאות:
- **חזק** צ'י **של הטחול** (健脾益气)
- **תזן** ין **של הכליות** (滋肾阴)
- **החלק** צ'י **של הכבד** (疏肝理气)
- **נקה** חום **מהקיבה** (清胃热)
- **הנע** דם, **פתח** חסימות (活血化瘀)

**עקרון מורכב**: כאשר יש שני דפוסים:
- **חזק** צ'י טחול + **החלק** צ'י כבד (健脾疏肝)
- **תזן** ין כליות + **הורד** אש כבד (滋肾清肝)

---

## 6. דוגמאות שלמות

### דוגמה 1
- **דפוס**: סטגנציית צ'י כבד (肝气郁结)
- **עקרון**: החלק כבד, הנע צ'י (疏肝理气)
- **הסבר**: הצ'י של הכבד "תקוע" → צריך לגרום לו לזרום שוב

### דוגמה 2
- **דפוס**: חסר צ'י טחול + לחות (脾气虚 + 湿)
- **עקרון**: חזק טחול, ייבש לחות (健脾燥湿)
- **הסבר**: הטחול חלש → לחות מצטברת → חזק את הטחול והוא ינהל את הלחות

### דוגמה 3
- **דפוס**: חסר ין כליות עם אש עולה (肾阴虚, 虚火上炎)
- **עקרון**: תזן ין כליות, נקה חום מחסר, הורד אש (滋肾阴, 清虚热, 降火)
- **הסבר**: ין חסר → אין מה לאזן את האש → האש עולה → תזנו ין וגם הורידו אש

## 7. שאלות לחזרה

1. מהו ה"חוק היסודי" של עקרונות הטיפול ברפואה סינית?
2. מהו ההבדל בין שורש (בן) לענף (ביאו)?
3. מתי נטפל בענף לפני השורש?
4. נסחו עקרון טיפולי לכל אחד: א) חסר יאנג כליות, ב) אש כבד, ג) פלגמה-לחות בריאות.
5. כיצד ננסח עקרון טיפולי כאשר יש שני דפוסים?
