# מודול 6ב: קומפנדיום סינדרומים (辨证纲要)

## תיאור כללי

קומפנדיום זה הוא מדריך מקיף לכל הסינדרומים (דפוסים פתולוגיים) ברפואה הסינית, מאורגן לפי מערכת איברים. כל סינדרום כולל: תיאור המנגנון, גורמים, סימפטומים מפורטים, ממצאי בדיקה (לשון ודופק), אבחנה מבדלת, עקרון טיפולי, מרשם דיקור עם הסבר לכל נקודה, פורמולת צמחים, והערות קליניות מעשיות.

## מטרות למידה

בסיום מודול זה, הסטודנט יוכל:
1. לזהות ולהבדיל בין סינדרומים של כל מערכת איברים
2. לקשר בין סימפטומים, ממצאי לשון ודופק לסינדרום ספציפי
3. לגבש עקרון טיפולי מתאים לכל סינדרום
4. לבחור נקודות דיקור ופורמולות צמחים בהתאם לסינדרום
5. להבין את הקשרים בין סינדרומים (התפתחות, טרנספורמציה, שילובים)

## איך להשתמש בקומפנדיום

1. **ללימוד שיטתי**: לימדו לפי סדר — כבד, לב, טחול, ריאות, כליות. בכל מערכת, התחילו מהדפוסים הבסיסיים (חוסר צ'י, חוסר דם) לפני המורכבים (רוח, ליחה-אש)
2. **לעיון מהיר בקליניקה**: חפשו לפי סימפטום מוביל → עברו לטבלת האבחנה המבדלת
3. **לתרגול**: השתמשו בטבלאות ההשוואה כדי לתרגל הבחנה בין סינדרומים דומים
4. **לבחינות**: שימו לב לממצאי הלשון והדופק — אלה המפתח לאבחון נכון

## מבנה כל סינדרום

כל קובץ סינדרום כולל את המקטעים הבאים:
- **תיאור כללי** — מנגנון והתפתחות
- **גורמים אטיולוגיים** — מה גורם לסינדרום
- **סימפטומים ומאפיינים** — עיקריים, משניים, ומבדילים
- **ממצאי בדיקה** — לשון ודופק
- **אבחנה מבדלת** — טבלת השוואה לסינדרומים דומים
- **עקרון טיפולי** — בסינית ובעברית
- **מרשם דיקור** — נקודות עם הסבר לבחירה
- **פורמולת צמחים** — פורמולה ראשית עם שינויים
- **הערות קליניות** — טיפים מעשיים

## מפת קשרים בין סינדרומי כבד

```
אגירת צ'י כבד (肝气郁结)
├── → אש כבד (肝火上炎) ← [אגירה הופכת לחום]
│     └── → חוסר ין כבד (肝阴虚) ← [אש שורפת ין]
│           └── → עליית יאנג כבד (肝阳上亢) ← [ין לא מעגן יאנג]
│                 └── → רוח כבד (肝风内动) ← [יאנג קיצוני הופך לרוח]
├── → קיפאון דם ← [אגירת צ'י ממושכת מעכבת דם]
├── → התקפת כבד על טחול/קיבה ← [כבד תוקף עיכול]
└── → ליחה/שקעים ← [אגירה פוגעת בחילוף נוזלים]

חוסר דם כבד (肝血虚)
├── → חוסר ין כבד (肝阴虚) ← [דם הוא חלק מהין]
├── → רוח מחוסר (血虚生风) ← [דם לא מזין גידים]
└── → חוסר דם לב (心血虚) ← [כבד ולב חולקים דם]
```

## מפת קשרים בין סינדרומי לב

```
חוסר צ'י לב (心气虚)
└── → חוסר יאנג לב (心阳虚) ← [צ'י חלש → יאנג חלש]
      └── → קריסת יאנג (亡阳) ← [מצב חירום!]

חוסר דם לב (心血虚)
└── → חוסר ין לב (心阴虚) ← [דם הוא חלק מהין]
      └── → אש ריקה (虚火) ← [ין לא מאזן אש]

אש לב (心火亢盛) → שורפת ין → חוסר ין לב

חוסר צ'י/יאנג → דם לא זורם → קיפאון דם לב (心血瘀阻)

חולשת טחול → ליחה → ליחה מעממת לב (痰蒙心窍)
```

## סיכום מהיר: לשון ודופק

### סינדרומי כבד

| סינדרום | לשון | דופק |
|---------|------|------|
| אגירת צ'י כבד | רגילה/אדמדמה בצדדים | חוטי (弦) |
| אש כבד | אדומה, ציפוי צהוב יבש | חוטי + מהיר |
| חוסר דם כבד | חיוורת בצדדים | דק + חוטי |
| חוסר ין כבד | אדומה, ללא ציפוי | דק + מהיר |
| רוח כבד | תלוי בסוג | תלוי בסוג |
| עליית יאנג כבד | אדומה, ציפוי מועט | חוטי, חזק למעלה חלש למטה |
| קיפאון קור בכבד | חיוורת-סגלגלה, ציפוי לבן לח | חוטי + איטי/הדוק |
| לחות-חום כבד/מרה | אדומה, ציפוי צהוב שמנוני | חוטי + מהיר + חלקלק |

### סינדרומי לב

| סינדרום | לשון | דופק |
|---------|------|------|
| חוסר צ'י לב | חיוורת | ריק + חלש |
| חוסר יאנג לב | חיוורת-סגלגלה, לחה | עמוק + חלש + איטי |
| חוסר ין לב | אדומה בקצה, ללא ציפוי | דק + מהיר |
| חוסר דם לב | חיוורת, דקה | דק + חלש |
| אש לב | אדומה מאוד בקצה, ציפוי צהוב | מלא + מהיר |
| קיפאון דם לב | סגולה עם כתמים | חוטי + מחוספס |
| ליחה מעממת לב | ציפוי שמנוני (לבן או צהוב) | חלקלק (±מהיר) |
