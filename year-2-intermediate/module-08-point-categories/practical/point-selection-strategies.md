# אסטרטגיות בחירת נקודות — מדריך מעשי

## Point Selection Strategies — Practical Guide

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. ליישם את העקרונות המרכזיים לבחירת נקודות
2. לבנות מרשם טיפולי מאוזן ויעיל
3. לשלב קטגוריות נקודות שונות במרשם אחד
4. להבין את הפילוסופיה של "פחות זה יותר"
5. ליישם שילובים אמפיריים קלאסיים למצבים שכיחים

---

## 1. עקרונות יסוד בבחירת נקודות

### 1.1 שלושת השלבים

כל בחירת נקודות עוברת שלושה שלבים:

```
שלב 1: זיהוי דפוס (辨证 Bian Zheng) → מהו הדפוס?
שלב 2: קביעת עקרון טיפולי (治法 Zhi Fa) → מה רוצים לעשות?
שלב 3: בחירת נקודות (选穴 Xuan Xue) → אילו נקודות מממשות את העקרון?
```

**דוגמה**:
- דפוס: סטגנציית צ'י-כבד (肝气郁结)
- עקרון טיפולי: הרגעת הכבד, פיזור סטגנציה (疏肝理气)
- נקודות: LR3, LR14, PC6, GB34

### 1.2 ששת הכללים לבחירת נקודות

| כלל | הסבר | דוגמה |
|---|---|---|
| **מקומי + דיסטלי** | שלב נקודות ליד הבעיה + נקודות רחוקות | כאב ברך: ST35 (מקומי) + ST36 (דיסטלי) |
| **מעלה + מטה** | שלב נקודות בחלק עליון + תחתון | כאב ראש: GB20 (למעלה) + LR3 (למטה) |
| **ימין + שמאל** | דקרו בשני הצדדים, או דקרו בצד הנגדי לכאב | כאב כתף ימין: LI4 שמאל |
| **קדמי + אחורי** | שלב חזה/בטן + גב | חולשת ריאות: LU1 (קדמי) + BL13 (אחורי) |
| **ין + יאנג** | שלב ערוצי ין + ערוצי יאנג | כאבי גב: BL40 (יאנג) + KI3 (ין) |
| **שורש + ענף** | טפל בסיבה (שורש) + בסימפטום (ענף) | שיעול מחסר: LU9 + ST36 (שורש) + LU7 (ענף) |

## 2. גישת מקומי-דיסטלי (远近配穴法)

### 2.1 נקודות מקומיות (近穴 Jin Xue)

- ממוקמות **באזור הבעיה** או קרוב אליו
- משפיעות ישירות על הרקמה המקומית
- מטפלות בסימפטום

**כללים**:
- לפחות 1-2 נקודות מקומיות ברוב הטיפולים
- בדלקת חריפה — היזהרו מדיקור ישיר באזור נפוח/אדום
- נקודות אה-שי (阿是穴) — רגישות = נקודת טיפול

### 2.2 נקודות דיסטליות (远穴 Yuan Xue)

- ממוקמות **רחוק מהבעיה**, בדרך כלל מתחת למרפק/ברך
- משפיעות דרך הערוץ וההיגיון האנרגטי
- מטפלות בשורש הבעיה

**כללים**:
- בחרו ערוץ שעובר באזור הבעיה
- נקודות דיסטליות **חזקות יותר** כשהן רחוקות יותר
- הנקודות הדיסטליות החשובות ביותר הן מ**מרפק/ברך כלפי מטה**

### 2.3 דוגמאות

| בעיה | מקומי | דיסטלי | ערוץ מוביל |
|---|---|---|---|
| כאב ראש עורפי | GB20, BL10 | BL60, SI3 | שלפוחית, מעי דק |
| כאב ראש קדמי | ין טאנג, ST8 | LI4, ST44 | קיבה, מעי גס |
| כאב כתף | LI15, TE14 | LI4, TE5 | מעי גס, שאן ג'יאו |
| כאב ברך | ST35, שי יאן | ST36, SP9 | קיבה, טחול |
| כאב גב תחתון | BL23, יאו יאן | BL40, BL60 | שלפוחית |
| כאב בטן | REN12, ST25 | ST36, SP4 | קיבה, טחול |

## 3. בחירת נקודות לפי קטגוריה

### 3.1 מתי להשתמש בכל קטגוריה

| קטגוריה | מתי | דוגמה |
|---|---|---|
| **יואן** (原穴) | חסר כרוני, חיזוק איבר | KI3 לחסר כליות |
| **לואו** (络穴) | חיבור צמדי פנים-חוץ | SP4 + ST42 (יואן-לואו) |
| **שי** (郄穴) | מצב חריף, כאב חד | ST34 לכאב קיבה חריף |
| **שו-גב** (背俞穴) | חסר כרוני, מוקסה | BL20 + מוקסה לחולשת טחול |
| **מו-קדמי** (募穴) | עודף, אבחון | REN12 לכאב קיבה חריף |
| **חמש שו** (五输穴) | טיפול לפי אלמנט, עונה | LR2 לניקוי חום כבד |
| **הוי** (八会穴) | בעיית רקמה/חומר ספציפי | GB34 לבעיות גידים |
| **נקודות מפגש** (交会穴) | כלים יוצאי דופן | SI3+BL62 לדו מאי |

### 3.2 "עץ ההחלטות" לבחירת קטגוריה

```
האם הבעיה חריפה?
├── כן → נקודת שי (郄穴) + מו-קדמי
└── לא → האם הבעיה כרונית?
    ├── כן → נקודת יואן (原穴) + שו-גב (+ מוקסה)
    └── מעורב → שילוב יואן + שי + שו-מו
```

## 4. שילובי נקודות אמפיריים (经验配穴)

### 4.1 שילובים קלאסיים שכל מטפל צריך לדעת

| שילוב | נקודות | שימוש |
|---|---|---|
| **ארבעת השערים** (四关穴) | LI4 + LR3 (דו-צדדי) | כאב כללי, מתח, סטגנציה |
| **שלוש נקודות הרגל** | ST36 + SP6 + LR3 | בסיס לרוב הטיפולים — עיכול, דם, צ'י |
| **שלישיית הדם** | BL17 + SP10 + LR3 | כל בעיות הדם |
| **שלישיית השינה** | HT7 + SP6 + אן מיאן | נדודי שינה |
| **שלישיית העיכול** | ST36 + REN12 + SP3 | חולשת קיבה-טחול |
| **זוג הכליות** | KI3 + BL23 | חסר כליות כרוני |
| **זוג האלרגיה** | LI4 + LI20 | נזלת, סינוסיטיס |
| **זוג החרדה** | PC6 + HT7 | חרדה, דפיקות, פאניקה |

### 4.2 שילובים למצבים שכיחים

**כאב ראש**:
- עורפי: GB20 + BL10 + BL60 + SI3
- צדדי: טאי יאנג + GB8 + TE5 + GB41
- קדמי: ין טאנג + ST8 + LI4 + ST44
- קודקודי: GV20 + סי שן צ'ונג + LR3 + KI1

**כאבי גב תחתון**:
- BL23 + BL25 + BL40 + BL60 + KI3
- + ג'יא ג'י מקומיים + יאו יאן

**בעיות עיכול**:
- REN12 + ST25 + ST36 + SP6 + SP3
- + מוקסה על ST36 ו-REN12

**חרדה ודיכאון**:
- HT7 + PC6 + LR3 + ין טאנג + GV20
- + אן מיאן (אם יש נדודי שינה)

**מחזור לא סדיר**:
- SP6 + SP8 + LR3 + REN4 + KI3
- + BL23 + BL32

## 5. הפילוסופיה של "פחות זה יותר" (少而精)

### 5.1 העיקרון

> "刺之要，气至而有效" — "עיקר הדקירה — כשהצ'י מגיע, יש יעילות"
> — לינג שו

**אל תדקרו יותר מדי נקודות.** מרשם טיפולי טוב הוא:
- **ממוקד**: 6-12 נקודות ברוב הטיפולים
- **הגיוני**: כל נקודה משרתת מטרה ברורה
- **מאוזן**: שילוב מקומי+דיסטלי, ין+יאנג, שורש+ענף

### 5.2 הנחיות כמותיות

| סוג טיפול | מספר נקודות מומלץ |
|---|---|
| בעיה מקומית פשוטה | 4-6 |
| בעיה פנימית ממוקדת | 6-10 |
| דפוס מורכב | 8-12 |
| מצב חירום | 1-3 (ממוקד) |

### 5.3 שגיאות נפוצות

- **יותר מדי נקודות** → מפזר את הצ'י, מחליש את הטיפול
- **חוסר לוגיקה** → נקודות שסותרות זו את זו (חיזוק + ניקוז באותו ערוץ)
- **רק מקומי** → מטפל בסימפטום בלבד
- **רק דיסטלי** → מפספס את ההשפעה המקומית

## 6. בניית מרשם טיפולי — שלב אחר שלב

### 6.1 תבנית לבניית מרשם

1. **נקודה ראשית** (主穴 Zhu Xue) — הנקודה שמגדירה את כיוון הטיפול
2. **נקודות תומכות** (配穴 Pei Xue) — מחזקות ומשלימות
3. **נקודות מקומיות** — לטיפול בסימפטום
4. **נקודות איזון** — לאיזון ין-יאנג, ימין-שמאל

### 6.2 דוגמה מפורטת — מטופלת עם דיסמנוריאה מסטגנציית דם

**דפוס**: סטגנציית דם ברחם (瘀血阻滞)
**עקרון**: הנעת דם, פיזור סטגנציה, עצירת כאב (活血化瘀, 止痛)

| תפקיד | נקודה | הנמקה |
|---|---|---|
| ראשית | **SP6** | מפגש שלושת ערוצי הין — מפעילה דם |
| תומכת | **SP8** (שי) | נקודת שי של טחול — כאב חריף גינקולוגי |
| תומכת | **LR3** (יואן) | מפזרת סטגנציית כבד |
| תומכת | **BL17** (הוי דם) | מניעה דם סטגנטי |
| מקומית | **REN4** | בטן תחתונה, רחם |
| מקומית | **ST29** | נקודה מקומית לרחם |
| איזון | **LI4** | "ארבעת השערים" עם LR3, מניע צ'י ודם |

**סה"כ**: 7 נקודות — מרשם ממוקד, מאוזן, ויעיל.

## 7. תרגילים

### תרגיל 1: בנה מרשם

לכל מצב, בנה מרשם טיפולי מנומק (5-8 נקודות):
1. כאב ראש צדדי חוזר (דפוס שאו יאנג)
2. עצירות כרונית מחסר צ'י
3. נדודי שינה עם חלומות רבים (חסר דם-לב)
4. כאבי ברכיים דו-צדדיים בקשישה

### תרגיל 2: ביקורת מרשם

מטפל הציע את המרשם הבא לכאבי גב תחתון: LI4, ST36, SP6, HT7, PC6, LR3, GB34, ST40, REN12.
- מהי הביקורת שלך?
- כיצד תשפר את המרשם?

### תרגיל 3: בחירת קטגוריה

לכל מצב, החלט באיזו קטגוריית נקודות תשתמש:
1. התקף אסטמה חריף
2. חולשת כליות כרונית
3. כאבי כתף מגידים דלקתיים
4. בחילות הריון
