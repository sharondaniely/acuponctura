# אבחון לשון (舌诊)

## Tongue Diagnosis - She Zhen

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לבצע בדיקת לשון שיטתית ומדויקת
2. להבחין בין צבע גוף הלשון, צורתה, תנועתה וציפויה
3. למפות אזורי הלשון לאיברים הפנימיים
4. לקשר ממצאי לשון לדפוסים פתולוגיים
5. להימנע מגורמים שמשבשים את הבדיקה

---

## 1. מדוע הלשון כה חשובה?

אבחון הלשון נחשב **לשיטת האבחון האובייקטיבית ביותר** ברפואה הסינית. בניגוד לדופק (שתלוי ברגישות המטפל) או לשאלות (שתלויות בדיווח המטופל), הלשון **נראית לעין** ומספקת מידע אמין ויציב.

### למה דווקא הלשון?

- הלשון היא **שריר גלוי** עם אספקת דם עשירה — משקפת את מצב הדם והצ'י
- כל ערוצי הזאנג-פו (臟腑) מתחברים ללשון ישירות או בעקיפין
- הלשון **פחות מושפעת מגורמים רגעיים** (בניגוד לדופק שמשתנה מרגע לרגע)
- שינויים בלשון מתפתחים לאט → משקפים מצבים כרוניים ויציבים

---

## 2. כיצד לבדוק את הלשון

### 2.1 תנאי בדיקה

- **תאורה**: אור טבעי עדיף, או אור לבן ניטרלי (לא צהוב, לא ניאון)
- **זמן**: עדיף בבוקר, לפני אכילה
- **הוראות למטופל**: "הוציאו את הלשון בחוצה בנינוחות, לא בכוח מרבי"
- **משך**: בדיקה קצרה (15-20 שניות), אחרת הלשון מתחילה לרעוד ומשנה צבע

### 2.2 גורמים שמשבשים בדיקה

| גורם | השפעה |
|---|---|
| מזון/משקה צבעוני (קפה, סלק, סוכריות) | משנה צבע ציפוי |
| צחצוח לשון | מסיר ציפוי |
| עישון | ציפוי צהוב מלאכותי |
| תרופות (אנטיביוטיקה) | ציפוי שחור מלאכותי |
| נשימת פה ממושכת | לשון יבשה מלאכותית |
| עונת השנה | ציפוי עבה מעט יותר בחורף |

---

## 3. גוף הלשון (舌体 She Ti)

### 3.1 צבע גוף הלשון (舌色 She Se)

צבע גוף הלשון משקף את מצב **הדם, הין והיאנג**:

| צבע | סינית | משמעות | דפוסים אופייניים |
|---|---|---|---|
| **ורוד-בהיר** | 淡红 (Dan Hong) | תקין | לשון בריאה |
| **חיוור** | 淡白 (Dan Bai) | חסר דם, חסר יאנג, קור | חסר דם (血虚), חסר יאנג טחול |
| **אדום** | 红 (Hong) | חום, חום מחסר ין | חום פנימי, חסר ין |
| **אדום כהה** | 绛 (Jiang) | חום חמור, חום בדם | חום ברמת הדם (血分热) |
| **סגול** | 紫 (Zi) | סטגנציית דם | סטגנציית דם (血瘀) |
| **סגול-אדום** | 紫红 | סטגנציית דם + חום | חום עם סטגנציית דם |
| **סגול-כחול** | 紫蓝 | סטגנציית דם + קור | קור עם סטגנציית דם |

### 3.2 צורת גוף הלשון (舌形 She Xing)

| צורה | סינית | משמעות |
|---|---|---|
| **רזה/דקה** | 瘦薄 (Shou Bo) | חסר דם, חסר ין |
| **נפוחה** | 胖大 (Pang Da) | חסר צ'י, לחות, פלגמה |
| **נפוחה עם סימני שיניים** | 齿痕 (Chi Hen) | חסר צ'י טחול, לחות |
| **קשיחה** | 强硬 (Qiang Ying) | חום גבוה, רוח פנימית, פלגמה חוסמת |
| **רפויה** | 痿软 (Wei Ruan) | חסר צ'י חמור, חסר ין חמור |
| **סדוקה** | 裂纹 (Lie Wen) | חסר ין, חום, חסר דם |
| **קוצנית** (פפילות בולטות) | 芒刺 (Mang Ci) | חום חמור, אש |
| **קצרה** | 短 (Duan) | חסר חמור, או פלגמה/קור חמורים |
| **ארוכה** | 长 (Chang) | חום בלב, פלגמה-אש |

### 3.3 תנועת הלשון (舌态 She Tai)

| תנועה | סינית | משמעות |
|---|---|---|
| **רועדת** | 颤动 (Chan Dong) | חסר צ'י, רוח פנימית מחסר |
| **סוטה לצד** | 歪斜 (Wai Xie) | רוח פנימית, שבץ (או סכנה לשבץ) |
| **מתכנסת** (לא מצליח להוציא) | 短缩 (Duan Suo) | קור חמור, או חום חמור עם רוח |
| **מתגלגלת/זזה** | 吐弄 (Tu Nong) | חום בלב, רוח פנימית |

## 4. ציפוי הלשון (舌苔 She Tai)

הציפוי משקף את מצב **הקיבה והטחול**, ואת נוכחות **גורמים פתוגניים** (חום, קור, לחות).

### 4.1 עובי הציפוי

- **דק** (薄 Bo): ניתן לראות את גוף הלשון דרכו → תקין, או מחלה חיצונית קלה
- **עבה** (厚 Hou): לא ניתן לראות את גוף הלשון → לחות, פלגמה, מזון מצטבר

### 4.2 צבע הציפוי

| צבע ציפוי | סינית | משמעות |
|---|---|---|
| **לבן דק** | 薄白 (Bo Bai) | תקין, או קור חיצוני |
| **לבן עבה** | 厚白 (Hou Bai) | קור-לחות |
| **צהוב דק** | 薄黄 (Bo Huang) | חום קל |
| **צהוב עבה** | 厚黄 (Hou Huang) | לחות-חום |
| **אפור** | 灰 (Hui) | לחות קרה פנימית, או חום כרוני |
| **שחור** | 黑 (Hei) | קור קיצוני או חום קיצוני |

### 4.3 לחות הציפוי

- **לח** (润 Run): נוזלי גוף תקינים
- **יבש** (燥 Zao): חסר נוזלים, חום, חסר ין
- **חלקלק** (滑 Hua): לחות יתרה, קור
- **דביק/שמנוני** (腻 Ni): פלגמה, לחות-חום

### 4.4 ציפוי מושרש לעומת חסר שורש

- **מושרש** (有根 You Gen): הציפוי "צומח" מגוף הלשון → צ'י קיבה תקין
- **חסר שורש** (无根 Wu Gen): הציפוי "מונח" על הלשון, ניתן לגרד → חסר צ'י קיבה חמור

### 4.5 ציפוי מקולף (剥苔 Bo Tai)

- **ציפוי מקולף חלקית** → חסר ין חלקי של קיבה
- **ציפוי מקולף לחלוטין** (לשון "מראה") → חסר ין חמור של קיבה/כליות
- **ציפוי מקולף במפה** (גיאוגרפי) → חסר ין כרוני, לחות-חום

## 5. אזורי הלשון (舌区 She Qu)

הלשון מחולקת לאזורים, כל אחד מתאים לאיבר:

```
         קצה
        ╱    ╲
       / לב/  \
      / ריאות  \
     /          \
    | כבד  כבד  |
    | שק   שק   |
    | מרה  מרה  |
    |            |
    | טחול/קיבה |
    |            |
     \          /
      \ כליות /
       \ מעיים/
        ╲    ╱
         שורש
```

| אזור | איברים |
|---|---|
| **קצה** (舌尖 She Jian) | לב (心), ריאות (肺) |
| **צדדים** (舌边 She Bian) | כבד (肝), שק מרה (胆) |
| **מרכז** (舌中 She Zhong) | טחול (脾), קיבה (胃) |
| **שורש** (舌根 She Gen) | כליות (肾), שלפוחית, מעיים |

### יישום קליני של האזורים

- **קצה אדום** → חום בלב (חרדה, נדודי שינה)
- **צדדים אדומים** → אש כבד (כעס, מתח)
- **ציפוי עבה במרכז** → לחות בטחול/קיבה (בעיות עיכול)
- **ציפוי עבה בשורש** → לחות-קור בכליות/מעי גס

## 6. דוגמאות קליניות

### דוגמה 1: חסר צ'י טחול
- **גוף**: חיוור, נפוח, סימני שיניים
- **ציפוי**: לבן, דק
- **אזורים**: מרכז קצת שקוע
- **פרשנות**: הטחול לא מתפקד, לחות מצטברת

### דוגמה 2: חסר ין כליות
- **גוף**: אדום, רזה, סדקים (במיוחד סדק אמצעי עמוק)
- **ציפוי**: ללא ציפוי, או ציפוי מקולף
- **אזורים**: שורש אדום במיוחד
- **פרשנות**: ין כליות מותש, חום מחסר

### דוגמה 3: סטגנציית צ'י כבד
- **גוף**: צדדים אדומים או סגלגלים
- **ציפוי**: דק, לבן או צהבהב
- **אזורים**: צדדים בולטים
- **פרשנות**: צ'י הכבד אינו זורם בחופשיות

### דוגמה 4: לחות-חום בטחול/קיבה
- **גוף**: אדום, אולי נפוח
- **ציפוי**: צהוב, עבה, שמנוני-דביק
- **אזורים**: מרכז הלשון עם ציפוי הכי עבה
- **פרשנות**: לחות וחום מצטברים במערכת העיכול

### דוגמה 5: סטגנציית דם
- **גוף**: סגול, אולי עם כתמים כהים
- **ציפוי**: יכול להיות כל ציפוי
- **אזורים**: כתמים מופיעים באזור הרלוונטי
- **פרשנות**: דם אינו זורם בחופשיות

---

## 7. עקרונות פרשנות

### 7.1 כללי עדיפות

1. **גוף הלשון חשוב מהציפוי** — גוף הלשון משקף מצבים עמוקים וכרוניים יותר
2. **צבע הגוף חשוב מהצורה** — צבע משקף ישירות את מצב הדם והצ'י
3. **כשיש סתירה בין לשון לדופק** — בדרך כלל הלשון אמינה יותר למצבים כרוניים, הדופק למצבים חריפים

### 7.2 שילוב ממצאים

אל תפרשו ממצא בודד בבידוד. תמיד שלבו:
- צבע גוף + צורה + ציפוי + אזורים = תמונה שלמה
- לשון + דופק + שאלות = אבחנה מבוססת

---

## 8. שאלות לחזרה

1. מדוע אבחון הלשון נחשב לאובייקטיבי ביותר מבין שיטות האבחון?
2. מה ההבדל בין לשון חיוורת ללשון אדומה — מה כל אחת מעידה?
3. מהם סימני שיניים בלשון ומה הם מעידים?
4. מה ההבדל בין ציפוי מושרש לציפוי חסר שורש?
5. מטופל עם לשון אדומה, רזה, סדוקה, ללא ציפוי — מה הדפוס הסביר?
6. מפו את אזורי הלשון וציינו את האיברים המתאימים לכל אזור.
