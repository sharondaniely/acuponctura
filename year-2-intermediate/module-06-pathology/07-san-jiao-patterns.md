# דפוסי שלוש המבערות (三焦辨证 San Jiao Bian Zheng)

## San Jiao Pattern Identification

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לתאר את מערכת דפוסי סאן ג'יאו ואת מפתחה, וו ג'ו טונג
2. לזהות דפוסים פתולוגיים בכל אחת משלוש המבערות
3. להבין את ההבדל בין מערכת זו לארבע הרמות של יה טיאן שי
4. ליישם את שתי המערכות יחד לאבחון מחלות חום

---

## מבוא היסטורי

מערכת **דפוסי סאן ג'יאו** פותחה על ידי **וו ג'ו טונג** (吴鞠通 Wu Ju Tong) בספרו **"דיון שיטתי על מחלות חום"** (温病条辨 Wen Bing Tiao Bian), שנכתב ב-1798. וו ג'ו טונג הרחיב את עבודתו של יה טיאן שי והציע מערכת סיווג לפי **מיקום** — עליון, אמצעי, תחתון.

### הקשר לארבע הרמות

שתי המערכות (ארבע הרמות ושלוש המבערות) עוסקות ב**מחלות חום** (温病 Wen Bing) ומשלימות זו את זו:

| מערכת | קריטריון | פיתוח |
|-------|---------|-------|
| ארבע הרמות (卫气营血) | **עומק** — רמת חדירה | יה טיאן שי |
| שלוש המבערות (三焦) | **מיקום** — איזור בגוף | וו ג'ו טונג |

ניתן ורצוי **לשלב** בין שתי המערכות: למשל, "חום ברמת צ'י במבערה העליונה" (חום בריאות).

---

## עקרון החדירה

מחלות חום-לחות נוטות לחדור **מלמעלה למטה** דרך שלוש המבערות:

```
מבערה עליונה (上焦 Shang Jiao) — ריאות, לב, פריקרד
          ↓
מבערה אמצעית (中焦 Zhong Jiao) — טחול, קיבה
          ↓
מבערה תחתונה (下焦 Xia Jiao) — כבד, כליות
```

> "温邪上受，首先犯肺" (Wen Xie Shang Shou, Shou Xian Fan Fei)
> "פתוגן חום חודר מלמעלה, ותוקף תחילה את הריאות"
> — וו ג'ו טונג

---

## 1. מבערה עליונה (上焦 Shang Jiao)

האיברים: **ריאות, לב, פריקרד**

### 1.1 חום-לחות תוקף ריאות (邪犯肺卫)

זהו **השלב הראשוני** — הפתוגן החם רק חדר.

| סימפטום | ביטוי |
|---------|-------|
| חום | חום עם צמרמורת קלה |
| שיעול | שיעול |
| כאב גרון | אדום, כואב |
| צמא | קל |
| **לשון** | קצה אדום, ציפוי דק לבן או צהוב דק |
| **דופק** | שטחי, מהיר |

**טיפול:** שחרר שטח בקירור — ין צ'יאו סאן (银翘散), סאנג ג'ו ין (桑菊饮)
**דיקור:** LU7, LI4, LI11, GV14, SJ5

### 1.2 חום בריאות (邪热壅肺)

המחלה חדרה עמוק יותר בריאות.

| סימפטום | ביטוי |
|---------|-------|
| חום | גבוה |
| שיעול | חמור, כיח צהוב עבה |
| נשימה | קוצר נשימה, אסתמה |
| צמא | חמור |
| **לשון** | אדומה, ציפוי צהוב |
| **דופק** | חלקלק, מהיר |

**טיפול:** נקה חום מריאות — מא שינג שי גאן טאנג (麻杏石甘汤)
**דיקור:** LU5, LU10, LI11, ST40, GV14

### 1.3 חום חודר לפריקרד (热入心包 Re Ru Xin Bao)

**מצב חירום** — החום חדר ללב/פריקרד ופוגע בשן (תודעה).

| סימפטום | ביטוי |
|---------|-------|
| תודעה | **דליריום, בלבול, אובדן הכרה** |
| חום | גבוה, לילי |
| דיבור | מלמול לא ברור, או חוסר דיבור |
| גפיים | קרות (חום חסום בפנים) |
| **לשון** | אדומה כהה (绛), קצה אדום בוהק |
| **דופק** | דק, מהיר, או לא סדיר |

**טיפול:** פתח אוריפיסים, קרר לב — אן גונג ניו הואנג וואן (安宫牛黄丸)
**דיקור:** PC7, PC9 ג'ונג צ'ונג (הקזה), HT7, GV26 שוי גו, KI1 יונג צ'ואן

## 2. מבערה אמצעית (中焦 Zhong Jiao)

האיברים: **טחול, קיבה**

### 2.1 חום יאנג מינג (阳明气分热盛)

חום חזק ברמת הקיבה — דומה לשלב יאנג מינג בשאנג האן לון.

| סימפטום | ביטוי |
|---------|-------|
| חום | **גבוה, ללא צמרמורת** |
| הזעה | מרובה |
| צמא | **חמור מאוד**, רוצה שתייה קרה |
| פנים | אדומות |
| **לשון** | אדומה, ציפוי צהוב יבש |
| **דופק** | מלא, מהיר, גדול |

**טיפול:** נקה חום מקיבה — באי הו טאנג (白虎汤)
**דיקור:** LI11, LI4, ST44, ST43 שיאן גו, GV14

### 2.2 לחות-חום במבערה האמצעית (湿热蕴脾 / 中焦湿热)

**הדפוס הייחודי ביותר** למערכת סאן ג'יאו — כאשר לחות וחום מצטברים בטחול-קיבה.

| סימפטום | ביטוי |
|---------|-------|
| חום | מתמשך, **לא יורד בהזעה** |
| גוף | כבדות, "כמו עוטפים שמיכה רטובה" |
| בטן | נפיחות, מלאות, כאב |
| בחילה | יש, אין תיאבון |
| צואה | דביקה, רכה, מסריחה |
| שתן | כהה |
| צמא | **ללא צמא** או צמא ללא רצון לשתות |
| **לשון** | אדומה, **ציפוי צהוב שמן** — סימן מפתח! |
| **דופק** | רפוי ומהיר (濡数 Ru Shuo) |

**מאפיין מיוחד:** חום מלווה בלחות — קשה לנקות. הלחות "לוכדת" את החום ומקשה על הפירוק.

**טיפול:** ייבש לחות + נקה חום — סאן רן טאנג (三仁汤)
**דיקור:** SP9, SP6, ST36, REN12, REN9, LI11

### 2.3 לחות-חום עם צהבת (湿热发黄)

| סימפטום | ביטוי |
|---------|-------|
| עור/עיניים | **צהובים** (צהבת) — צהוב בוהק (יאנג צהבת) |
| בטן | נפיחות, כאב |
| חום | מתמשך |
| שתן | כהה מאוד |
| **לשון** | ציפוי צהוב שמן עבה |

**טיפול:** נקה חום-לחות + הסר צהבת — ין צ'ן האו טאנג (茵陈蒿汤)
**דיקור:** GB34, LR14, SP9, DU9, REN12

## 3. מבערה תחתונה (下焦 Xia Jiao)

האיברים: **כבד, כליות, מעיים, שלפוחית**

### 3.1 חסר ין כליות (肾阴耗损)

בשלב המאוחר של מחלת חום, **ין הכליות נפגע** מהחום הממושך.

| סימפטום | ביטוי |
|---------|-------|
| חום | **חום נמוך לילי** שנעלם בבוקר |
| יובש | פה יבש, גרון יבש |
| שינה | נדודי שינה |
| כפות | **חום בחמש** (כפות ידיים, כפות רגליים, חזה) |
| שתן | מועט, כהה |
| **לשון** | אדומה, **ללא ציפוי**, יבשה |
| **דופק** | דק, מהיר (细数 Xi Shuo) |

**טיפול:** הזן ין כליות — ג'יא ג'יאן פו מאי טאנג (加减复脉汤)
**דיקור:** KI3, KI6, SP6, REN4, LU7+KI6

### 3.2 חסר ין כבד עם רוח פנימית (肝肾阴虚，虚风内动)

חום ממושך פוגע בין הכבד → **רוח פנימית** מתעוררת.

| סימפטום | ביטוי |
|---------|-------|
| רעד | **ידיים רועדות**, תנועות בלתי רצוניות |
| עוויתות | אפשריות — שבץ חום |
| סחרחורת | חמורה |
| שריר | מתיחות, קשיחות, עוויתות |
| **לשון** | אדומה כהה, יבשה, רועדת |
| **דופק** | מיתרי, דק, מהיר |

**טיפול:** הזן ין כבד, הרגע רוח — דא דינג פנג ג'ו (大定风珠)
**דיקור:** LR3, KI3, SP6, GB20, DU20, טאי צ'ונג

### 3.3 חום-לחות במבערה התחתונה

| סימפטום | ביטוי |
|---------|-------|
| שתן | תכוף, צורב, כהה, עכור |
| צואה | שלשול עם ריר ודם, טנזמוס |
| **לשון** | ציפוי צהוב שמן |
| **דופק** | רפוי, מהיר |

**טיפול:** נקה חום-לחות מתחתון — בא ג'נג סאן (八正散) לשתן; באי טואו וונג (白头翁汤) לדיזנטריה

## סיכום — טבלת שלוש המבערות

| מבערה | איברים | דפוסים עיקריים | סימפטום מנחה | חומרה |
|-------|--------|---------------|-------------|-------|
| עליונה (上焦) | ריאות, לב | רוח-חום בריאות, חום בפריקרד | חום + שיעול / דליריום | קלה → חמורה |
| אמצעית (中焦) | טחול, קיבה | חום יאנג מינג, לחות-חום | חום + נפיחות + ציפוי שמן | בינונית |
| תחתונה (下焦) | כבד, כליות | חסר ין, רוח פנימית | חום נמוך + יובש / רעד | מחלה מאוחרת |

---

## שילוב שתי המערכות — ארבע הרמות + שלוש המבערות

| מבערה | רמת ווי | רמת צ'י | רמת יינג | רמת שיואה |
|-------|---------|---------|----------|----------|
| עליונה | רוח-חום בריאות | חום בריאות | חום בפריקרד | — |
| אמצעית | — | חום קיבה, לחות-חום | — | דימום קיבה/מעי |
| תחתונה | — | — | חום בכבד | חסר ין + רוח |

**שילוב מעשי:** השתמש ב**שלוש מבערות** לזיהוי **היכן** המחלה, וב**ארבע הרמות** לזיהוי **כמה עמוק** חדרה.

---

## רלוונטיות קלינית מודרנית

- **לחות-חום במבערה אמצעית** — שכיח מאוד: גסטרואנטריטיס, הפטיטיס, IBS עם חום
- **חום בפריקרד** — אנצפליטיס, מנינגיטיס, דליריום זיהומי
- **חסר ין מבערה תחתונה** — שלב ההחלמה ממחלות חמורות

---

## נקודות חשובות לזכירה

1. **לחות-חום** הוא המושג המרכזי במערכת סאן ג'יאו — שילוב קשה לטיפול
2. **ציפוי צהוב שמן** = לחות-חום — סימן המפתח למבערה אמצעית
3. **חום שלא יורד בהזעה** = לחות "לוכדת" חום — צריך לייבש לחות קודם
4. **מבערה עליונה** = שלב מוקדם; **תחתונה** = שלב מאוחר (חסר ין)
5. **שלב את שתי המערכות** — סאן ג'יאו למיקום, ארבע הרמות לעומק
6. **וו ג'ו טונג** ויה טיאן שי הם שני עמודי התווך של אסכולת וון בינג
