# ארבע הרמות — וון בינג (温病)

## Four Levels — Wen Bing (Warm Disease Theory)

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לתאר את ההבדל בין שאנג האן (נזקי קור) לוון בינג (מחלות חום)
2. לזהות את ארבע רמות החדירה של מחלת חום
3. לתאר סימפטומים, לשון ודופק של כל רמה
4. להבין את עקרונות הטיפול ולהכיר מרשמים מרכזיים

---

## מבוא היסטורי

מערכת **ארבע הרמות** (卫气营血辨证 Wei Qi Ying Xue Bian Zheng) פותחה על ידי **יה טיאן שי** (叶天士 Ye Tian Shi) בספרו **"דיון על מחלות חום"** (温热论 Wen Re Lun) בתקופת שושלת צ'ינג (המאה ה-18).

### מדוע נדרשה מערכת חדשה?

שאנג האן לון של ג'אנג ג'ונג ג'ינג עסק בעיקר ב**מחלות קור** (伤寒). אולם בדרום סין, מחלות **חום-לחות** (温病 Wen Bing) היו נפוצות יותר — מלריה, טיפוס, מחלות זיהומיות חמות. מחלות אלו לא התאימו למערכת שש השלבים, ונדרשה גישה חדשה.

### הבדלים מהותיים

| | שאנג האן (伤寒) | וון בינג (温病) |
|--|---------------|---------------|
| גורם | קור חיצוני | חום חיצוני / חום-לחות |
| כניסה | דרך העור (טאי יאנג) | דרך האף והפה (ריאות) |
| כיוון חדירה | שטח → פנים (6 שלבים) | ווי → צ'י → יינג → שיואה |
| טבע | קור שהופך לחום | חום מההתחלה |
| טיפול ראשוני | שחרר שטח בחימום | שחרר שטח בקירור |

## ארבע הרמות — סקירה כללית

```
ווי (卫) → צ'י (气) → יינג (营) → שיואה (血)
  ↓          ↓          ↓           ↓
הגנה      פנים-צ'י   הזנה        דם
(שטחי)    (חום חזק)  (עמוק)     (מסכן חיים)
(קל)                             (חמור)
```

חדירת המחלה היא **מבחוץ פנימה, מקל לחמור**. כל רמה מייצגת עומק גדול יותר של חדירה ומסכנת את הגוף יותר.

---

## 1. רמת ווי — הרמה ההגנתית (卫分证 Wei Fen Zheng)

### 1.1 מהות

הפתוגן החם תוקף את **ווי צ'י** (צ'י ההגנה) — השכבה החיצונית ביותר. הריאות נפגעות ראשונות כי הן "האיבר העדין" (娇脏) הפתוח לחוץ.

### 1.2 סימפטומים

| סימפטום | ביטוי |
|---------|-------|
| חום | **חום דומיננטי**, צמרמורת קלה (בניגוד לשאנג האן שבו צמרמורת דומיננטית) |
| ראש | כאב ראש |
| גרון | כאב, אדמומיות, נפיחות |
| אף | גודש או נזלת |
| שיעול | קל |
| צמא | קל |
| **לשון** | קצה אדום, ציפוי דק לבן או צהוב דק |
| **דופק** | שטחי ומהיר (浮数 Fu Shuo) |

### 1.3 טיפול

- **עיקרון:** שחרר שטח בקירור, פזר רוח-חום (辛凉解表 Xin Liang Jie Biao)
- **מרשם:** ין צ'יאו סאן (银翘散) — המרשם המפורסם ביותר לרמה זו
- **מרשם נוסף:** סאנג ג'ו ין (桑菊饮) — כאשר שיעול דומיננטי
- **דיקור:** LU7, LI4, LI11, GV14, SJ5, GB20

### 1.4 מפתח לזיהוי

> **חום > צמרמורת + כאב גרון + דופק שטחי מהיר = רמת ווי**

---

## 2. רמת צ'י — רמת הצ'י (气分证 Qi Fen Zheng)

### 2.1 מהות

הפתוגן חדר **מעבר לשטח** אל רמת הצ'י הפנימית. **חום חזק** שורר, הגוף נלחם בעוצמה. זוהי הרמה הרחבה ביותר ויכולה לערב איברים שונים.

### 2.2 סימפטומים

| סימפטום | ביטוי |
|---------|-------|
| חום | **חום גבוה**, ללא צמרמורת |
| הזעה | הזעה מרובה |
| צמא | **צמא חמור** למשקאות קרים |
| פנים | אדומות |
| שתן | כהה |
| **לשון** | אדומה, ציפוי צהוב |
| **דופק** | מהיר, גדול ומלא (洪大 Hong Da) |

### 2.3 תתי-דפוסים

| מיקום | סימפטומים ייחודיים | מרשם |
|-------|-------------------|------|
| חום בריאות | חום + שיעול + כיח צהוב + צמא | מא שינג שי גאן טאנג (麻杏石甘汤) |
| חום בקיבה | 4 גדולים — חום, הזעה, צמא, דופק גדול | באי הו טאנג (白虎汤) |
| חום במעיים | עצירות + חום אחה"צ + בטן כואבת | דפוס יאנג מינג — טיהור |
| לחות-חום | חום + כבדות + בחילה + ציפוי שמן צהוב | לחות-חום בסאן ג'יאו |

### 2.4 דיקור

- **עיקרון:** נקה חום, ייצר נוזלים
- **נקודות:** LI11, LI4, GV14, ST44, LU5 צ'י צה

### 2.5 מפתח לזיהוי

> **חום גבוה + ללא צמרמורת + צמא + הזעה + לשון אדומה ציפוי צהוב = רמת צ'י**

---

## 3. רמת יינג — רמת ההזנה (营分证 Ying Fen Zheng)

### 3.1 מהות

החום חדר **עמוק** לרמת הדם המזין (营 Ying). זוהי רמה חמורה — הלב נפגע (יינג קשור ללב), והשן (תודעה) מתערער.

### 3.2 סימפטומים

| סימפטום | ביטוי |
|---------|-------|
| חום | **חום שמחמיר בלילה** (夜热甚) |
| תודעה | **אי-שקט חמור, בלבול, דליריום** |
| שינה | נדודי שינה חמורים |
| עור | פריחות דקות (斑疹), אדמומיות |
| פה | יובש, **אך ללא צמא** (או צמא ללא רצון לשתות) |
| **לשון** | **אדומה כהה (绛 Jiang)** — סימן מפתח! |
| **דופק** | דק ומהיר (细数 Xi Shuo) |

### 3.3 טיפול

- **עיקרון:** נקה יינג, קרר דם, פתח לב (清营泄热 Qing Ying Xie Re)
- **מרשם:** צ'ינג יינג טאנג (清营汤) — קרנות שור מים, קורן רינוצרוס (כיום: קרן תאו מים)
- **דיקור:** PC7 דא לינג, PC8 לאו גונג, HT7, KI6, SP6

### 3.4 מפתח לזיהוי

> **חום לילי + בלבול/דליריום + לשון אדומה כהה (绛) + ללא צמא = רמת יינג**

---

## 4. רמת שיואה — רמת הדם (血分证 Xue Fen Zheng)

### 4.1 מהות

הרמה **העמוקה והמסוכנת ביותר**. החום חודר לדם עצמו, גורם ל**דימומים** ולפגיעה חמורה בין. מצב מסכן חיים.

### 4.2 סימפטומים

| סימפטום | ביטוי |
|---------|-------|
| חום | חום גבוה |
| דימום | **דימום מכל מקום** — אף, עור (פורפורה), הקאת דם, דם בצואה, דם בשתן |
| תודעה | דליריום חמור, אובדן הכרה, תרדמת |
| עור | פטכיות, כתמי דם תת-עוריים |
| **לשון** | **אדומה כהה מאוד (深绛)**, יבשה, אפשר קוצים |
| **דופק** | דק, מהיר, או מפורר (散 San) |

### 4.3 שני היבטים עיקריים

1. **חום מזיז דם** (热迫血行) → דימום — הדם "נשפך" מהכלים
2. **חום מייבש ין** → רוח פנימית → עוויתות, שבץ

### 4.4 טיפול

- **עיקרון:** קרר דם, עצור דימום, פתח אוריפיסים (凉血散血 Liang Xue San Xue)
- **מרשם:** שי ג'יאו די הואנג טאנג (犀角地黄汤)
- **מרשמי חירום:** אן גונג ניו הואנג וואן (安宫牛黄丸) — לפתיחת תודעה
- **דיקור:** SP10, BL40 (הקזה), SP6, KI3, PC7, נקודות ג'ינג-ווי (井 Jing) — הקזה

### 4.5 מפתח לזיהוי

> **דימום + דליריום/תרדמת + לשון אדומה כהה מאוד + פטכיות = רמת שיואה**

## השוואה בין ארבע הרמות

| | ווי (卫) | צ'י (气) | יינג (营) | שיואה (血) |
|--|---------|---------|----------|-----------|
| עומק | שטחי | פנים-קל | עמוק | עמוק מאוד |
| חום | + צמרמורת | גבוה | לילי | גבוה + דימום |
| צמא | קל | חמור | ללא צמא | ללא צמא |
| תודעה | תקינה | תקינה | בלבול | דליריום/תרדמת |
| לשון | קצה אדום | אדומה | אדומה כהה (绛) | אדומה כהה מאוד |
| דופק | שטחי מהיר | מלא מהיר | דק מהיר | דק / מפורר |
| חומרה | קלה | בינונית | חמורה | מסכנת חיים |
| טיפול | שחרר בקירור | נקה חום | נקה יינג | קרר דם |

## מעבר בין הרמות

### חוקי התקדמות

1. **התקדמות רצופה:** ווי → צ'י → יינג → שיואה — הנפוצה ביותר
2. **דילוג על רמות:** במחלה חזקה — ישירות מווי ליינג
3. **חדירה ישירה לפריקרד:** מצב חירום — דליריום פתאומי (热入心包 Re Ru Xin Bao)

### עיקרון טיפולי מנחה

> **"לטפל ברמת ווי — חייבים להזיע; ברמת צ'י — חייבים לנקות;
> ברמת יינג — יש לחדור ליינג;
> ברמת שיואה — יש לפזר ולקרר דם"**
> — יה טיאן שי

```
卫之后方言气 — "מווי עוברים לצ'י"
气之后方言营 — "מצ'י עוברים ליינג"  
营之后方言血 — "מיינג עוברים לשיואה"
```

## רלוונטיות קלינית מודרנית

למרות שמערכת וון בינג פותחה למחלות זיהומיות, היא רלוונטית גם ל:
- **מחלות ויראליות חמות** — שפעת חמה, COVID-19
- **דלקות חריפות** — דלקת ריאות, מנינגיטיס
- **מצבי חירום** עם חום גבוה ודימום
- **מחלות אוטואימוניות** עם מרכיב חום

---

## נקודות חשובות לזכירה

1. **וון בינג = מחלות חום** — בניגוד לשאנג האן שעוסק בקור
2. **רמת ווי** = שטח; **שיואה** = הכי עמוק ומסוכן
3. **לשון אדומה כהה (绛)** = סימן המפתח שהמחלה ברמת יינג/שיואה
4. **דימום** = המחלה ברמת שיואה — מצב חירום
5. **"חום לילי ללא צמא"** = רמת יינג — מבדיל מרמת צ'י (שם יש צמא חמור)
6. **ין צ'יאו סאן** הוא המרשם היסודי לרמת ווי — חשוב להכיר
