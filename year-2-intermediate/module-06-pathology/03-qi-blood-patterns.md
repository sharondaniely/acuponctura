# דפוסי צ'י, דם, ין, יאנג ונוזלי גוף

## Patterns of Qi, Blood, Yin, Yang & Body Fluids

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לזהות ולהבדיל בין דפוסי הפתולוגיה של צ'י (חסר, קיפאון, שקיעה, מרידה)
2. לזהות דפוסי דם (חסר, קיפאון, חום, קור)
3. לתאר דפוסי חסר ין וחסר יאנג
4. להבין פתולוגיות נוזלי גוף (ליחה, לחות, בצקת)
5. לציין לשון, דופק ועיקרון טיפולי לכל דפוס

## חלק א׳ — דפוסי צ'י (气的病证)

### 1. חסר צ'י (气虚 Qi Xu)

צ'י לא מספיק לבצע את תפקודיו — חימום, הגנה, הנעה, החזקה, המרה.

| מאפיין | ביטוי |
|--------|-------|
| סימפטומים | עייפות, חולשה, קוצר נשימה (במיוחד במאמץ), קול חלש, חוסר תיאבון |
| הזעה | ספונטנית (自汗 Zi Han) — צ'י לא מחזיק את הנקבוביות |
| חיסון | חולני תכוף — ווי צ'י חלש |
| **לשון** | חיוורת, נפוחה, אפשר סימני שיניים |
| **דופק** | ריק, חלש (虚弱 Xu Ruo) |
| **עיקרון טיפולי** | חזק צ'י (补气 Bu Qi) |
| **נקודות מפתח** | ST36 צו סאן לי, REN6 צ'י האי, LU9 טאי יואן |

### 2. קיפאון צ'י (气滞 Qi Zhi)

צ'י קיים אך **לא זורם** — תקוע, חסום.

| מאפיין | ביטוי |
|--------|-------|
| סימפטומים | תחושת נפיחות ומתיחה (胀 Zhang), כאב שמשתנה במיקום, נודד |
| רגשות | עצבנות, תסכול, דיכאון, אנחות תכופות |
| עיכול | נפיחות, גזים, גיהוקים |
| מחזור | לא סדיר, כאבי מחזור, PMS |
| **לשון** | תקינה או עם צדדים מעט אדומים |
| **דופק** | מיתרי (弦 Xian) |
| **עיקרון טיפולי** | הזז צ'י, שחרר קיפאון (理气 Li Qi / 疏肝 Shu Gan) |
| **נקודות מפתח** | LR3 טאי צ'ונג, LR14 צ'י מן, PC6 ניי גואן, GB34 יאנג לינג צ'ואן |

### 3. שקיעת צ'י (气陷 Qi Xian)

צ'י (בעיקר של הטחול) לא מצליח **לעלות** — שוקע כלפי מטה.

| מאפיין | ביטוי |
|--------|-------|
| סימפטומים | תחושת כבדות וגרירה כלפי מטה, צניחת איברים (רחם, קיבה, רקטום) |
| שלשול | כרוני, עם מזון לא מעוכל |
| ספציפי | טחורים, בריחת שתן, צניחת רחם, דימום כרוני |
| **לשון** | חיוורת |
| **דופק** | חלש, במיוחד בעמדת צ'י (יד ימין, אמצע) |
| **עיקרון טיפולי** | העלה צ'י (升提 Sheng Ti) |
| **נקודות מפתח** | DU20 באי הוי, REN6, ST36 — מוקסה |

### 4. מרידת צ'י / צ'י הפוך (气逆 Qi Ni)

צ'י זורם ב**כיוון הפוך** — עולה במקום לרדת.

| מאפיין | ביטוי |
|--------|-------|
| צ'י קיבה הפוך | בחילה, הקאה, שיהוקים, גיהוקים |
| צ'י ריאות הפוך | שיעול, אסתמה, קוצר נשימה |
| צ'י כבד הפוך | כאב ראש, סחרחורת, הקאת דם, כעס פתאומי |
| **לשון** | תלוי באיבר המעורב |
| **דופק** | עולה, מיתרי |
| **עיקרון טיפולי** | הורד צ'י (降气 Jiang Qi) |
| **נקודות מפתח** | REN17 טאן ג'ונג, PC6, ST36, LR3 |

## חלק ב׳ — דפוסי דם (血的病证)

### 5. חסר דם (血虚 Xue Xu)

דם לא מספיק להזין את הרקמות, האיברים והנפש.

| מאפיין | ביטוי |
|--------|-------|
| סימפטומים | חיוורון (פנים, שפתיים, ציפורניים), סחרחורת, ראייה מטושטשת |
| שינה | נדודי שינה, חלומות מרובים |
| עור/שיער | עור יבש, שיער יבש ודליל |
| מחזור | מועט, חיוור, או נעדר |
| גפיים | חוסר תחושה, נימול (麻木 Ma Mu) |
| **לשון** | חיוורת ודקה |
| **דופק** | דק (细 Xi), מחוספס (涩 Se) |
| **עיקרון טיפולי** | הזן דם (补血 Bu Xue) |
| **נקודות מפתח** | SP6 סאן ין ג'יאו, ST36, BL17 גה שו, BL20 פי שו |

### 6. קיפאון דם (血瘀 Xue Yu)

דם **נתקע** — לא זורם כראוי, יוצר חסימות.

| מאפיין | ביטוי |
|--------|-------|
| כאב | **קבוע במיקום, דוקר, חד** (刺痛 Ci Tong) — מחמיר בלילה ובלחיצה |
| מראה | שפתיים כהות/סגולות, עור כהה, עיגולים שחורים מתחת לעיניים |
| גושים | גידולים, מיומות, ציסטות |
| דימום | דם כהה עם קרישים |
| מחזור | כואב, עם קרישים כהים |
| **לשון** | סגולה או עם כתמים סגולים, ורידים מתחת ללשון מורחבים |
| **דופק** | מחוספס (涩 Se) או קשור (结 Jie) |
| **עיקרון טיפולי** | הזז דם, פזר קיפאון (活血化瘀 Huo Xue Hua Yu) |
| **נקודות מפתח** | SP10 שיואה האי, BL17, LR3, SP6, LI4 |

### 7. חום דם (血热 Xue Re)

חום חודר לרמת הדם ו**מאיץ** את תנועתו.

| מאפיין | ביטוי |
|--------|-------|
| סימפטומים | עור אדום, פריחות, דימום (מאף, חניכיים, עור) |
| דימום | דם אדום בוהק |
| מחזור | מוקדם, מרובה, דם אדום |
| רגש | אי-שקט, עצבנות |
| **לשון** | אדומה בוהקת |
| **דופק** | מהיר (数 Shuo) |
| **עיקרון טיפולי** | קרר דם (凉血 Liang Xue) |
| **נקודות מפתח** | SP10, LI11 צ'ו צ'י, BL40 ווי ג'ונג, SP6 |

### 8. קור דם (血寒 Xue Han)

קור חודר לדם ו**מקפיא** את תנועתו.

| מאפיין | ביטוי |
|--------|-------|
| סימפטומים | כאב שמשתפר בחום, גפיים קרות וכחולות |
| מחזור | מאחר, מועט, כהה עם קרישים, כאבי מחזור שמשתפרים בחום |
| **לשון** | סגולה/כהה, ציפוי לבן |
| **דופק** | עמוק, איטי, מתוח (沉迟紧) |
| **עיקרון טיפולי** | חמם דם (温经散寒 Wen Jing San Han) |
| **נקודות מפתח** | ST36, SP6, REN4 גואן יואן — עם מוקסה |

## חלק ג׳ — דפוסי ין ויאנג

### 9. חסר ין (阴虚 Yin Xu)

ין — הפן המקררת, הלחה, המזינה — חסר. **חום מדומה** (虚热) מופיע כי אין מספיק ין לאזן יאנג.

| מאפיין | ביטוי |
|--------|-------|
| חום | **חום בחמש** (五心烦热) — כפות ידיים, כפות רגליים, חזה |
| הזעה | **הזעות לילה** (盗汗 Dao Han) |
| יובש | פה יבש, גרון יבש, עור יבש |
| שתן | מועט וכהה |
| **לשון** | אדומה, ללא ציפוי או ציפוי מועט, סדקים |
| **דופק** | דק ומהיר (细数 Xi Shuo) |
| **עיקרון טיפולי** | הזן ין (滋阴 Zi Yin) |
| **נקודות מפתח** | KI6 ג'או האי, SP6, KI3, LU7 + KI6 (צירוף קלאסי) |

### 10. חסר יאנג (阳虚 Yang Xu)

יאנג — הפן המחממת, המניעה, המגינה — חסר. **קור פנימי** (虚寒) מופיע.

| מאפיין | ביטוי |
|--------|-------|
| קור | קור כללי, גפיים קרות |
| פנים | חיוורות, בצקתיות |
| אנרגיה | עייפות חמורה, רצון לשכב |
| שתן | שקוף, מרובה, נוקטוריה |
| צואה | רפויה, מזון לא מעוכל |
| מיניות | ירידה בליבידו, אימפוטנציה |
| **לשון** | חיוורת, נפוחה, לחה, סימני שיניים |
| **דופק** | עמוק, איטי, חלש (沉迟弱) |
| **עיקרון טיפולי** | חמם יאנג (温阳 Wen Yang) |
| **נקודות מפתח** | REN4, DU4 מינג מן, ST36, KI3 — עם מוקסה |

## חלק ד׳ — דפוסי נוזלי גוף (津液病证)

### 11. חסר נוזלים (津液不足 Jin Ye Bu Zu)

| מאפיין | ביטוי |
|--------|-------|
| סימפטומים | יובש כללי — עור, שפתיים, אף, גרון; צמא |
| שתן | מועט וכהה |
| צואה | יבשה |
| **לשון** | יבשה, סדקים |
| **דופק** | דק (细 Xi) |
| **עיקרון טיפולי** | הזן נוזלים (生津 Sheng Jin) |

### 12. ליחה (痰 Tan)

ליחה היא **תוצר פתולוגי של חילוף חומרים לקוי של נוזלים**. זהו אחד הגורמים הפתוגניים החשובים ביותר.

**שני סוגים:**
- **ליחה מוחשית (有形之痰):** כיח, הפרשות — ניתן לראות
- **ליחה בלתי-מוחשית (无形之痰):** גורמת לסחרחורת, בלבול, גושים, שיגעון — לא ניתן לראות

| סוג | סימפטומים | לשון | דופק |
|-----|----------|------|------|
| ליחה-קור (寒痰) | כיח לבן שקוף, שיעול | חיוורת, ציפוי לבן שמן | חלקלק (滑 Hua) |
| ליחה-חום (热痰) | כיח צהוב עבה, חום | אדומה, ציפוי צהוב שמן | חלקלק ומהיר |
| ליחה-רוח (风痰) | סחרחורת, שבץ, אפילפסיה, שיגעון | ציפוי שמן | מיתרי וחלקלק |
| ליחה חוסמת לב (痰迷心窍) | בלבול, אובדן הכרה, שיגעון | ציפוי שמן עבה | חלקלק |

> **"百病多由痰作祟"** — "מאה מחלות נובעות מליחה"

**עיקרון טיפולי:** פזר ליחה (化痰 Hua Tan)
**נקודות מפתח:** ST40 פנג לונג (נקודת המפתח לפיזור ליחה), SP9, REN12

### 13. לחות (湿 Shi)

לחות היא **כבדה, דביקה, שוקעת כלפי מטה**, ונוטה להתעכב.

| מאפיין | ביטוי |
|--------|-------|
| סימפטומים | כבדות בגוף ובגפיים, "כאילו עוטפים מגבת רטובה" |
| ראש | כבדות, "ערפל מוחי", סחרחורת |
| עיכול | נפיחות, בחילה, חוסר תיאבון, טעם דביק בפה |
| הפרשות | דביקות, מרובות |
| עור | פריחה מלחלחת, אקזמה |
| **לשון** | ציפוי שמן (腻 Ni) — **סימן המפתח!** |
| **דופק** | חלקלק (滑 Hua) או רפוי (濡 Ru) |
| **עיקרון טיפולי** | ייבש לחות (燥湿 Zao Shi) / פזר לחות (利湿 Li Shi) |
| **נקודות מפתח** | SP9 ין לינג צ'ואן, SP6, ST36, REN9 שוי פן |

### 14. בצקת (水肿 Shui Zhong)

הצטברות מים בגוף — קשורה בעיקר לריאות, טחול וכליות.

| סוג | מיקום | סיבה | טיפול |
|-----|-------|------|-------|
| בצקת יאנג (阳水) | פנים, חלק עליון | ריאות לא מפזרות מים | פזר ריאות, הסר לחות |
| בצקת ין (阴水) | רגליים, חלק תחתון | טחול/כליות חלשים | חמם יאנג, חזק טחול |

## סיכום — טבלת השוואה מהירה

| דפוס | כאב | לשון | דופק | מפתח |
|------|-----|------|------|------|
| חסר צ'י | אין/קל | חיוורת נפוחה | חלש | עייפות + הזעה ספונטנית |
| קיפאון צ'י | נפיחות, נודד | תקינה | מיתרי | נפיחות + עצבנות |
| חסר דם | אין | חיוורת דקה | דק | חיוורון + סחרחורת |
| קיפאון דם | דוקר, קבוע | סגולה/כתמים | מחוספס | כאב קבוע + כהות |
| חום דם | שורף | אדומה בוהקת | מהיר | דימום + פריחה |
| חסר ין | אין | אדומה ללא ציפוי | דק מהיר | חום בחמש + יובש |
| חסר יאנג | אין/קל | חיוורת נפוחה | עמוק איטי | קור + עייפות |
| ליחה | כבדות | ציפוי שמן | חלקלק | כיח/גושים + כבדות |
| לחות | כבדות | ציפוי שמן | רפוי | כבדות + דביקות |

## נקודות חשובות לזכירה

1. **"צ'י הוא מפקד הדם, דם הוא אם הצ'י"** — הם תמיד קשורים; חסר באחד משפיע על השני
2. **קיפאון צ'י ממושך → קיפאון דם** — קיפאון צ'י הוא לעתים קרובות השלב הראשון
3. **חסר ין ≠ חום אמיתי** — חום בחסר ין הוא "חום דמוי" (虚热), לא חום אמיתי
4. **ליחה היא "גורם לכל מחלה"** — כשאין הסבר, חשוב על ליחה
5. **ציפוי שמן = לחות/ליחה** — סימן הלשון החשוב ביותר לזיהוי
6. **ST40** היא נקודת המפתח לפיזור ליחה; **SP9** ללחות; **SP6** לדם
