# שנה ב׳ — רמת ביניים

## תיאור קצר

שנה שנייה זו מעמיקה את ההבנה התיאורטית ומפתחת את כישורי האבחון והטיפול. הסטודנט ילמד את תורת זאנג-פו (Zang-Fu), פתולוגיה, זיהוי סינדרומים (ביאן ג'נג — 辨证), ארבע שיטות האבחון, חשיבה קלינית, קטגוריות נקודות מיוחדות, וטכניקות דיקור מתקדמות.

## מטרות למידה

- להבין את תורת זאנג-פו (脏腑) — תפקודי האיברים המלאים (Zang) והחלולים (Fu) ויחסיהם
- לזהות דפוסים פתולוגיים (ביאן ג'נג) באמצעות שמונת העקרונות (בא גאנג — 八纲) ושיטות נוספות
- ליישם את ארבע שיטות האבחון: התבוננות (וואנג — 望), האזנה והרחה (וון — 闻), שאילה (וון — 问), מישוש (צ'יאה — 切)
- לפתח חשיבה קלינית — ממסימפטומים לדפוס, מדפוס לעקרון טיפולי, ומעקרון לנקודות
- לשלוט בקטגוריות נקודות מיוחדות (חמש נקודות שו, יואן, לואו, שו גב, מו קדמי) ובטכניקות מתקדמות (בו-שיה, מוקסה, כוסות רוח, גואה שא)
