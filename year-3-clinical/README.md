# שנה ג׳ — יישום קליני

## תיאור קצר

שנה שלישית זו ממוקדת ביישום קליני מעשי. הסטודנט ילמד לטפל במגוון תחומים: רפואה פנימית, כאב ואורתופדיה, גינקולוגיה, מצבים נפשיים-רגשיים, דרמטולוגיה, ושילוב פורמולות צמחים (草药) עם הדיקור.

## מטרות למידה

- לטפל במצבים של רפואה פנימית: נשימה, עיכול, לב וכלי דם, שתן ורבייה, נוירולוגיה, חיסון ואוטואימוניות
- לאבחן ולטפל בתסמונות כאב ואורתופדיה — תסמונת בי (Bi — 痹), כאב גב, צוואר וכתף, ברך וירך, פציעות ספורט
- לטפל בהפרעות גינקולוגיות: מחזור, פריון, הריון, לידה, גיל המעבר, PCOS ואנדומטריוזיס
- לטפל בהפרעות שן (Shen — 神) ומצבים נפשיים-רגשיים: חרדה, דיכאון, נדודי שינה, PTSD
- לשלב פורמולות צמחים עם הדיקור ולטפל במצבים דרמטולוגיים

## מה תשיג בסיום השנה

בסיום שנה ג׳ תוכל לטפל באופן עצמאי במגוון רחב של מצבים קליניים, לבנות פרוטוקולי טיפול מותאמים, ולשלב דיקור עם צמחי מרפא. תהיה מוכן/ה לשנה ד׳ — רמת מתקדם.
