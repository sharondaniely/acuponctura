# אקנה ורוזציאה

## Acne and Rosacea in TCM

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לזהות ולאבחן את הדפוסים השונים של אקנה ורוזציאה לפי TCM
2. להבין את הקשר בין איברי הפנים ומיקום הפריחה בפנים
3. לבנות פרוטוקול דיקור מותאם הכולל נקודות מקומיות ורחוקות
4. לספק הנחיות תזונתיות מתאימות
5. לתכנן סדרת טיפולים מעשית

---

## 1. הבנה כללית — פנים ברפואה הסינית

### 1.1 מפת הפנים (面部脏腑分布)

לפי הרפואה הסינית, אזורים שונים בפנים מתאימים לאיברים שונים:

| אזור בפנים | איבר | משמעות פריחה באזור |
|---|---|---|
| **מצח** | לב / אש קטנה | חום בלב, מתח, נדודי שינה |
| **בין הגבות** | כבד | סטגנציית כבד, כעס, הורמונים |
| **אף** | טחול / קיבה | חום בקיבה, תזונה לקויה |
| **לחיים** | ריאות (ימין) / כבד (שמאל) | חום ריאות/כבד |
| **סנטר/לסת** | כליות / הורמונלי | חוסר איזון הורמונלי, מחזור |
| **סביב הפה** | קיבה / טחול | חום-לחות בקיבה |

### 1.2 גורמי אקנה ב-TCM

- **חום** (热) — הגורם המרכזי
- **לחות** (湿) — מחמיר ויוצר מוגלה
- **ריר** (痰) — יוצר ציסטות עמוקות
- **סטאזיס דם** (血瘀) — יוצר צלקות וכתמים
- **רעילות** (毒) — זיהום, מוגלה

## 2. אקנה (痤疮 Cuo Chuang)

### 2.1 דפוסים

#### דפוס 1: חום ריאות (肺热型)

| מאפיין | פרטים |
|---|---|
| **ביטוי** | קומדונים (ראשים שחורים/לבנים), פפולות אדומות קטנות |
| **מיקום** | מצח, אף, לחיים |
| **סימנים נלווים** | עור שמנוני, פנים אדומות, אף סתום, עצירות |
| **לשון** | קצה אדום, ציפוי צהוב דק |
| **דופק** | צף (浮), מהיר (数) |
| **עיקרון** | לנקות חום ריאות (清肺热) |
| **נקודות** | LI4, LI11, LU5, LU7, GV14, ST44 |
| **פורמולה** | פי פא צ'ינג פיי ין (枇杷清肺饮) |

#### דפוס 2: חום קיבה (胃热型)

| מאפיין | פרטים |
|---|---|
| **ביטוי** | פפולות ופוסטולות (מוגלה), אדומות וכואבות |
| **מיקום** | סביב האף והפה, לחיים |
| **סימנים נלווים** | תיאבון מוגבר, צמא, ריח פה, עצירות, צואה יבשה |
| **לשון** | אדומה, ציפוי צהוב עבה |
| **דופק** | חלקלק (滑), מהיר (数), חזק |
| **עיקרון** | לנקות חום קיבה (清胃热) |
| **נקודות** | ST44, ST36, LI4, LI11, CV12, SP10 |
| **פורמולה** | צ'ינג ווי סאן (清胃散) |

#### דפוס 3: לחות-חום (湿热蕴结型)

| מאפיין | פרטים |
|---|---|
| **ביטוי** | פפולות עם מוגלה, ציסטות, נודולות, עור שמנוני ומבריק |
| **מיקום** | לחיים, סנטר, גב עליון |
| **סימנים נלווים** | גוף כבד, עייפות, שתן כהה, הפרשות וגינליות |
| **לשון** | אדומה, ציפוי צהוב שמנוני |
| **דופק** | חלקלק (滑), מהיר (数) |
| **עיקרון** | לנקות חום ולייבש לחות (清热利湿) |
| **נקודות** | SP9, SP6, ST36, LI11, SP10, BL20 |
| **פורמולה** | ין צ'ן האו טאנג (茵陈蒿汤) + תוספות |

#### דפוס 4: סטאזיס דם עם ריר (痰瘀互结型)

| מאפיין | פרטים |
|---|---|
| **ביטוי** | ציסטות עמוקות, נודולות קשות, כתמים כהים, צלקות |
| **מיקום** | לסת, צדדי פנים |
| **סימנים נלווים** | מחזור כואב, כתמים כהים בעור, גושים |
| **לשון** | סגולה, כתמים |
| **דופק** | מיתרי (弦), מחוספס (涩) |
| **עיקרון** | להניע דם, לפזר ריר (活血化痰散结) |
| **נקודות** | SP10, BL17, LR3, SP6, ST40, נקודות מקומיות |
| **פורמולה** | טאו הונג סי וו טאנג (桃红四物汤) + חאי זאו, בייי ג'יא |

## 3. רוזציאה (酒渣鼻 Jiu Zha Bi)

### 3.1 סקירה כללית

רוזציאה נקראת ברפואה הסינית "ג'יו ג'א בי" (酒渣鼻 — "אף שמרי היין") בגלל האדמומיות האופיינית באף ובלחיים. קשורה בעיקר ל**חום בקיבה ובריאות** ול**סטאזיס דם**.

### 3.2 דפוסים

#### דפוס 1: חום ריאות-קיבה (肺胃热盛型)

| מאפיין | פרטים |
|---|---|
| **ביטוי** | אדמומיות באף ולחיים, הרגשת חום בפנים, פפולות קטנות |
| **גורמים מחמירים** | אלכוהול, מזון חריף, חום, שמש, מתח |
| **לשון** | אדומה, ציפוי צהוב |
| **דופק** | חלקלק (滑), מהיר (数) |
| **עיקרון** | לנקות חום ריאות וקיבה (清肺胃热) |
| **נקודות** | LI4, LI11, LI20, ST44, LU5, GV14, Yin Tang |
| **פורמולה** | פי פא צ'ינג פיי ין (枇杷清肺饮) + שנג די, מו דאן פי |

#### דפוס 2: סטאזיס דם (血瘀型)

| מאפיין | פרטים |
|---|---|
| **ביטוי** | אדמומיות כהה קבועה, טלנגיאקטזיות (נימי דם), אף מעובה (rhinophyma) |
| **לשון** | סגולה |
| **דופק** | מחוספס (涩) |
| **עיקרון** | להניע דם ולפזר סטאזיס (活血化瘀) |
| **נקודות** | SP10, BL17, LR3, LI4, נקודות מקומיות סביב האף |
| **פורמולה** | טאו הונג סי וו טאנג (桃红四物汤) |

## 4. נקודות פנים לדרמטולוגיה

### 4.1 נקודות מקומיות חשובות

| נקודה | שם | מיקום | שימוש |
|---|---|---|---|
| **LI20** | יינג שיאנג (迎香) | חריץ האף | רוזציאה, אקנה באף |
| **ST2** | סי באי (四白) | מתחת לעין | אקנה בלחיים |
| **ST4** | די צ'אנג (地仓) | פינת הפה | אקנה סביב הפה |
| **ST6** | ג'יא צ'ה (颊车) | שריר הלסת | אקנה בלסת |
| **ST7** | שיא גואן (下关) | מתחת לזיגומה | אקנה בלחיים |
| **Yin Tang** | ין טאנג | בין הגבות | אקנה במצח, הרגעה |
| **GV23** | שאנג שינג (上星) | מעל קו שיער | אקנה במצח |

### 4.2 עקרונות דיקור פנים

1. **מחטים דקות**: 0.20-0.25 מ"מ (גייג' 36-38)
2. **זווית**: שטחית, 15-30° ביחס לעור
3. **עומק**: 0.3-0.5 צון
4. **שימור**: 15-20 דקות
5. **לא לדקור ישירות** לתוך אקנה דלקתית פעילה — סביב הנגע בלבד
6. **חיטוי קפדני**: הפנים רגישים לזיהום

## 5. הנחיות תזונתיות

### 5.1 מזונות להימנע

| קטגוריה | דוגמאות | סיבה |
|---|---|---|
| **חריף** | פלפלים, שום, בצל | מייצרים חום |
| **שומני/מטוגן** | טיגון עמוק, פאסט פוד | מייצרים לחות-חום |
| **מתוק מרוכז** | סוכר, שוקולד, ממתקים | מייצרים לחות וריר |
| **אלכוהול** | במיוחד יין אדום ובירה | מייצר חום-לחות |
| **חלב** | חלב, גבינות שמנות | מייצר לחות וריר |

### 5.2 מזונות מומלצים

- **ירקות ירוקים**: מלפפון, סלרי, חסה (מקררים חום)
- **תה ירוק**: מנקה חום מהפנים
- **שעועית מאש** (绿豆 Lu Dou): מנקה חום-רעילות
- **דלעת מרה** (苦瓜 Ku Gua): מנקה חום, מרה
- **תפוח עץ, אגס**: מלחלחים, מקררים

## 6. פרוטוקול טיפולי מעשי

### 6.1 אקנה — מהלך טיפול

| שלב | תקופה | מטרה | גישה |
|---|---|---|---|
| **חריף** | 2-4 שבועות | הפחתת דלקת | 2-3 טיפולים/שבוע, נקודות מנקות חום |
| **ייצוב** | 4-8 שבועות | איזון דפוס | 1-2 טיפולים/שבוע, נקודות מקומיות + סיסטמיות |
| **תחזוקה** | חודשי | מניעת חזרה | 1 טיפול/שבוע-שבועיים, חיזוק + תזונה |

### 6.2 רוזציאה — מהלך טיפול

| שלב | תקופה | מטרה | גישה |
|---|---|---|---|
| **התפרצות** | 2-4 שבועות | ניקוי חום | 2 טיפולים/שבוע, ניקוי חום ריאות-קיבה |
| **ייצוב** | 2-3 חודשים | הפחתת אדמומיות | שבועי, הנעת דם + ניקוי חום |
| **ארוך טווח** | מתמשך | בקרה | חודשי + ניהול תזונה |

### 6.3 דוגמה לטיפול — אקנה מלחות-חום

**נקודות:**
- **מרוחקות**: LI11, LI4, SP9, SP10, ST36, SP6
- **מקומיות**: ST2, ST6, Yin Tang, LI20
- **אוזן**: Shen Men, Lung, Endocrine, Cheek

**טכניקה:**
- נקודות מרוחקות: דיקור רגיל, עומק 0.5-1 צון
- נקודות פנים: מחטים דקות, שטחי, 0.3 צון
- שימור: 20 דקות

## 7. סיכום

> אקנה ורוזציאה הן מחלות עור שכיחות עם ביטוי משמעותי בפנים, ולכן ההשפעה הפסיכולוגית על המטופל גדולה. הרפואה הסינית מציעה גישה מקיפה: טיפול בשורש (ניקוי חום, ייבוש לחות, הנעת דם) יחד עם טיפול מקומי (נקודות פנים). מפת הפנים מסייעת באבחנה, ושילוב עם הנחיות תזונתיות מעצים את הטיפול. סבלנות חשובה — שיפור משמעותי בדרך כלל לאחר 8-12 טיפולים.
