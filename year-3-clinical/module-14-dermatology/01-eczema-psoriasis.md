
# אקזמה ופסוריאזיס

## Eczema and Psoriasis in TCM

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לזהות ולאבחן את הדפוסים השונים של אקזמה ופסוריאזיס לפי TCM
2. להבחין בין שלבים חריפים לכרוניים ולהתאים טיפול
3. לבנות פרוטוקול דיקור מותאם לכל דפוס
4. להכיר את טכניקת "הקפת הדרקון" (围刺法)
5. להכיר פורמולות צמחיות מרכזיות לדרמטולוגיה

---

## 1. עקרונות כלליים — עור ברפואה הסינית

### 1.1 העור ואיברי הפנים

| איבר | קשר לעור |
|---|---|
| **ריאות** (肺) | שולטות בעור ובשיער הגוף (主皮毛); מפיצות ווי צ'י לשטח |
| **טחול** (脾) | מייצר דם שמזין את העור; לחות מצטברת → בעיות עור רטובות |
| **כבד** (肝) | מאחסן דם; סטגנציה → חום/רעילות → בעיות עור |
| **דם** (血) | מזין את העור; חוסר דם → יובש, גרד; חום בדם → אדמומיות |
| **רוח** (风) | גורם פתוגני מרכזי בדרמטולוגיה → גרד, שינויים מהירים |

### 1.2 גורמים פתוגניים עיקריים בדרמטולוגיה

| גורם | ביטוי בעור |
|---|---|
| **רוח** (风 Feng) | גרד, שינויים מהירים, נדידה, הופעה פתאומית |
| **חום** (热 Re) | אדמומיות, דלקת, כאב, נפיחות |
| **לחות** (湿 Shi) | הפרשות, שלפוחיות, נפיחות, נוזלים |
| **יובש** (燥 Zao) | סדקים, קילוף, עור מחוספס |
| **רעילות** (毒 Du) | זיהום, מוגלה, פצעים חמורים |
| **סטאזיס דם** (血瘀) | כתמים כהים, עור מעובה, צבע סגלגל |

## 2. אקזמה (湿疹 Shi Zhen)

### 2.1 סקירה כללית

אקזמה ברפואה הסינית מכונה "שי ג'ן" (湿疹) — "פריחת לחות". הגורם המרכזי הוא תמיד **לחות** (湿), בשילוב עם גורמים נוספים.

### 2.2 דפוסים

#### דפוס 1: רוח-חום (风热型)

| מאפיין | פרטים |
|---|---|
| **שלב** | חריף |
| **ביטוי** | אדמומיות, גרד חד, פריחה שמתפשטת מהר, יכולה לנדוד |
| **מיקום** | פלג גוף עליון, פנים, ידיים |
| **לשון** | אדומה, ציפוי צהוב דק |
| **דופק** | צף (浮), מהיר (数) |
| **עיקרון** | לגרש רוח, לנקות חום (疏风清热) |
| **נקודות** | LI11, LI4, SP10, BL17, GV14, GB31 |
| **פורמולה** | שיאו פנג סאן (消风散) |

#### דפוס 2: לחות-חום (湿热型)

| מאפיין | פרטים |
|---|---|
| **שלב** | תת-חריף |
| **ביטוי** | שלפוחיות, הפרשות נוזליות, גרד עז, עור מודלק ורטוב |
| **מיקום** | אזורי קפלים, מפשעות, אחורי ברכיים |
| **לשון** | אדומה, ציפוי צהוב שמנוני |
| **דופק** | חלקלק (滑), מהיר (数) |
| **עיקרון** | לנקות חום ולייבש לחות (清热利湿) |
| **נקודות** | SP9, SP6, ST36, LI11, SP10, BL20 |
| **פורמולה** | לונג דאן שיה גאן טאנג (龙胆泻肝汤) |

#### דפוס 3: חוסר דם עם רוח (血虚风燥型)

| מאפיין | פרטים |
|---|---|
| **שלב** | כרוני |
| **ביטוי** | עור יבש, מחוספס, קילוף, גרד שמחמיר בלילה, עור מעובה |
| **מיקום** | מפרקים, כפות ידיים, אזורים כרוניים |
| **לשון** | חיוורת, יבשה |
| **דופק** | דק (细), מחוספס (涩) |
| **עיקרון** | להזין דם, ללחלח רוח (养血润燥祛风) |
| **נקודות** | SP10, BL17, SP6, ST36, LI11, LR8 |
| **פורמולה** | סי וו טאנג (四物汤) + פאנג פנג, צ'אן טוי |

#### דפוס 4: חולשת טחול עם לחות (脾虚湿蕴型)

| מאפיין | פרטים |
|---|---|
| **שלב** | כרוני-חוזר |
| **ביטוי** | אקזמה חוזרת, הפרשות בהירות, עייפות, נפיחות, עור חיוור-רטוב |
| **מיקום** | רגליים, אזורי קפלים |
| **לשון** | חיוורת, נפוחה, סימני שיניים, ציפוי לבן שמנוני |
| **דופק** | רך (濡), חלש |
| **עיקרון** | לחזק טחול ולייבש לחות (健脾利湿) |
| **נקודות** | ST36, SP9, SP6, BL20, CV12, LI11 |
| **פורמולה** | שן לינג באי ג'ו סאן (参苓白术散) |

## 3. פסוריאזיס (银屑病 Yin Xie Bing / 牛皮癣 Niu Pi Xuan)

### 3.1 סקירה כללית

פסוריאזיס ברפואה הסינית נקרא "ין שיה בינג" (银屑病 — "מחלת קשקשי הכסף") או "ניו פי שואן" (牛皮癣 — "גרד עור השור"). הדגש הוא על **דם** כגורם מרכזי.

### 3.2 דפוסים

#### דפוס 1: חום בדם (血热型)

| מאפיין | פרטים |
|---|---|
| **שלב** | פעיל / מתקדם (进行期) |
| **ביטוי** | נגעים אדומים חדשים, קשקשים כסופים, גרד, נגעים מתרחבים, תופעת קבנר חיובית |
| **לשון** | אדומה, ציפוי צהוב |
| **דופק** | מהיר (数), חלקלק (滑) |
| **עיקרון** | לקרר דם ולנקות חום (凉血清热) |
| **נקודות** | SP10, BL17, LI11, GV14, LI4, SP6 |
| **פורמולה** | שי הוי דאו צ'י ין (犀角地黄汤 — כיום עם שווי ניו ג'יאו) |

#### דפוס 2: סטאזיס דם (血瘀型)

| מאפיין | פרטים |
|---|---|
| **שלב** | יציב / כרוני (稳定期) |
| **ביטוי** | נגעים קבועים, כהים, סגלגלים, עור מעובה, קשקשים עבים |
| **לשון** | סגולה, כתמים סגולים |
| **דופק** | מחוספס (涩) |
| **עיקרון** | להניע דם ולפזר סטאזיס (活血化瘀) |
| **נקודות** | SP10, BL17, LR3, SP6, LI11, נקודות מקומיות |
| **פורמולה** | טאו הונג סי וו טאנג (桃红四物汤) |

#### דפוס 3: יובש דם (血燥型)

| מאפיין | פרטים |
|---|---|
| **שלב** | רגרסיה / כרוני (退行期) |
| **ביטוי** | נגעים חיוורים, קשקשים דקים, עור יבש, גרד קל |
| **לשון** | חיוורת, יבשה |
| **דופק** | דק (细), מחוספס (涩) |
| **עיקרון** | להזין דם וללחלח יובש (养血润燥) |
| **נקודות** | SP10, BL17, SP6, ST36, LR8, KI6 |
| **פורמולה** | דאנג גווי ין זי (当归饮子) |

## 4. נקודות מפתח בדרמטולוגיה

### 4.1 נקודות בסיס

| נקודה | שם | פעולה דרמטולוגית |
|---|---|---|
| **LI11** | צ'ו צ'י (曲池) | הנקודה החשובה ביותר — מנקה חום, מגרשת רוח, מטפלת בגרד |
| **SP10** | שואה האי (血海) | "ים הדם" — מקררת דם, מטפלת בכל בעיה דמית בעור |
| **BL17** | גה שו (膈俞) | נקודת השפעה של דם — מזינה/מקררת/מניעה דם |
| **LI4** | חה גו (合谷) | מגרשת רוח, מנקה חום, פעולה על פנים ופלג גוף עליון |
| **SP6** | סאן ין ג'יאו (三阴交) | מזינה דם וין, מטפלת ביובש עור |
| **GV14** | דא ג'וי (大椎) | מנקה חום מכל הגוף, נקודה חשובה לפסוריאזיס |
| **GB31** | פנג שי (风市) | "שוק הרוח" — מגרשת רוח, מטפלת בגרד |

### 4.2 נקודות מקומיות — עקרונות

- **סביב הנגע**: דיקור 0.5-1 צון מגבול הנגע — "הקפת הדרקון" (ראה סעיף 5)
- **אזור הנגע**: בנגעים כרוניים, ניתן לדקור ישירות לתוך הנגע
- **נקודות א-שי**: נקודות כאב/גרד מקומיות

## 5. טכניקת "הקפת הדרקון" (围刺法 Wei Ci Fa / 围龙刺 Wei Long Ci)

### 5.1 תיאור הטכניקה

טכניקה שבה מחטי דיקור מוחדרות **מסביב לנגע העורי** בצורת טבעת, עם הכוונים כלפי מרכז הנגע.

### 5.2 אופן ביצוע

1. **מיקום**: 0.5-1 צון מחוץ לגבול הנגע
2. **כיוון**: המחטים מכוונות אלכסונית **לכיוון מרכז הנגע** (15-30°)
3. **מרווח**: כל 1-2 ס"מ לאורך הגבול
4. **עומק**: 0.5-1 צון (תלוי במיקום ובעובי העור)
5. **זמן שימור**: 20-30 דקות

### 5.3 מנגנון פעולה

- מגביר זרימת דם מקומית
- מפזר סטגנציה מקומית
- מפעיל תגובה חיסונית מקומית
- מאיץ ריפוי רקמות

### 5.4 התוויות

- פסוריאזיס — פלאקים יציבים
- אקזמה כרונית — אזורים מעובים
- ויטיליגו
- צלקות

## 6. פורמולות צמחיות מרכזיות

### 6.1 לאקזמה

| פורמולה | סינית | שימוש |
|---|---|---|
| שיאו פנג סאן | 消风散 | רוח-חום/לחות — חריף |
| לונג דאן שיה גאן טאנג | 龙胆泻肝汤 | לחות-חום חריפה |
| שן לינג באי ג'ו סאן | 参苓白术散 | חולשת טחול כרונית |
| סי וו טאנג + תוספות | 四物汤加味 | חוסר דם כרוני |

### 6.2 לפסוריאזיס

| פורמולה | סינית | שימוש |
|---|---|---|
| שי הוי דאו צ'י ין (מותאם) | 犀角地黄汤 | חום בדם — שלב פעיל |
| טאו הונג סי וו טאנג | 桃红四物汤 | סטאזיס דם — שלב יציב |
| דאנג גווי ין זי | 当归饮子 | יובש דם — שלב רגרסיה |

### 6.3 צמחים בודדים חשובים בדרמטולוגיה

| צמח | סינית | פעולה |
|---|---|---|
| שנג די הואנג | 生地黄 | מקרר דם, מזין ין |
| מו דאן פי | 牡丹皮 | מקרר דם, מפזר סטאזיס |
| צ'י שאו | 赤芍 | מקרר דם, מפחית דלקת |
| באי שיאן פי | 白鲜皮 | מנקה חום-לחות מהעור |
| די פו זי | 地肤子 | מגרש רוח, מייבש לחות, מפחית גרד |
| פאנג פנג | 防风 | מגרש רוח — מפחית גרד |

## 7. הנחיות תזונתיות

### 7.1 מזונות להימנע (בזמן התפרצות)

- מזונות חריפים וחמים: שום, בצל, פלפל, ג'ינג'ר מרובה
- אלכוהול
- רכיכות ופירות ים (מחמירים לחות-חום)
- מזונות מעובדים, סוכר
- חלב ומוצרי חלב (מייצרים לחות)

### 7.2 מזונות מומלצים

- ירקות ירוקים (מקררים חום)
- שעועית מאש, שעועית אדום (ייבוש לחות)
- דגני בוקר מלאים (חיזוק טחול)
- דגים (מזינים ין ודם)
- כורכום (מניע דם, מפחית דלקת)

## 8. סיכום

> דרמטולוגיה ברפואה הסינית מתמקדת בזיהוי הגורם הפתוגני (רוח, חום, לחות, יובש, סטאזיס דם) ובהתאמת הטיפול לשלב המחלה (חריף/כרוני). אקזמה קשורה בעיקר ללחות, פסוריאזיס קשור בעיקר לדם. נקודות הבסיס — LI11, SP10, BL17 — רלוונטיות כמעט לכל מצב דרמטולוגי. שילוב דיקור מקומי (הקפת דרקון), דיקור סיסטמי וצמחי מרפא מעניק את התוצאות הטובות ביותר.
