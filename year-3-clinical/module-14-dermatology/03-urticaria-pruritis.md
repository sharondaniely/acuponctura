# אורטיקריה וגרד

## Urticaria and Pruritus in TCM

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לזהות ולאבחן את הדפוסים השונים של אורטיקריה (סרפדת) וגרד
2. להבחין בין מצבים חריפים לכרוניים ולהתאים גישה טיפולית
3. לבנות פרוטוקול דיקור לכל דפוס
4. להכיר פורמולות צמחיות מרכזיות ונקודות מפתח
5. לשלב הנחיות מניעה וניהול אורח חיים

## 1. אורטיקריה / סרפדת (荨麻疹 Qian Ma Zhen)

### 1.1 סקירה כללית

אורטיקריה ברפואה הסינית נקראת "צ'יאן מא ג'ן" (荨麻疹) או בשם הקלאסי **"פנג ג'ן קואי" (风疹块 — "גושי פריחת רוח")**. השם מעיד על הגורם המרכזי — **רוח (风)**.

**מאפיינים:**
- פפולות (wheals) שמופיעות פתאום ונעלמות פתאום — אופייני לרוח
- גרד עז — רוח גורמת לגרד
- שינוי מיקום — רוח נודדת
- מוחמרת ע"י רוח, שינויי מזג אוויר

### 1.2 חריפה לעומת כרונית

| מאפיין | חריפה (< 6 שבועות) | כרונית (> 6 שבועות) |
|---|---|---|
| **אופי** | עודף (实) | חסרון (虚) עם רוח |
| **גורם** | רוח חיצונית + חום/קור | רוח פנימית מחוסר דם |
| **התחלה** | פתאומית | חוזרת ונשנית |
| **טיפול** | גירוש רוח, ניקוי חום | הזנת דם, גירוש רוח |

### 1.3 דפוסים

#### דפוס 1: רוח-קור (风寒型)

| מאפיין | פרטים |
|---|---|
| **ביטוי** | פפולות חיוורות/ורודות, גרד, מוחמר בקור ורוח |
| **סימנים נלווים** | מועדף חום, גפיים קרות, אין צמא |
| **לשון** | חיוורת, ציפוי לבן דק |
| **דופק** | צף (浮), הדוק (紧) |
| **עיקרון** | לגרש רוח ולפזר קור (疏风散寒) |
| **נקודות** | LI4, LU7, GB20, BL12, GV14 + מוקסה |
| **פורמולה** | מא הואנג טאנג (麻黄汤) או גווי ג'י טאנג (桂枝汤) + פאנג פנג |

#### דפוס 2: רוח-חום (风热型)

| מאפיין | פרטים |
|---|---|
| **ביטוי** | פפולות אדומות, גרד בוער, מוחמר בחום |
| **סימנים נלווים** | צמא, פנים אדומות, עצבנות, עצירות |
| **לשון** | אדומה, ציפוי צהוב |
| **דופק** | צף (浮), מהיר (数) |
| **עיקרון** | לגרש רוח ולנקות חום (疏风清热) |
| **נקודות** | LI11, LI4, SP10, GV14, GB31, BL17 |
| **פורמולה** | שיאו פנג סאן (消风散) או ין צ'יאו סאן (银翘散) + תוספות |

#### דפוס 3: חוסר דם עם רוח (血虚生风型) — **כרונית**

| מאפיין | פרטים |
|---|---|
| **ביטוי** | פפולות חיוורות/ורודות חוזרות, גרד שמחמיר בלילה ובעייפות |
| **סימנים נלווים** | חיוורון, עייפות, סחרחורת, שיער יבש, ציפורניים שבירות |
| **לשון** | חיוורת, דקה |
| **דופק** | דק (细), מחוספס (涩) |
| **עיקרון** | להזין דם ולגרש רוח (养血祛风) |
| **נקודות** | SP10, BL17, SP6, ST36, LI11, GB31, LR8 |
| **פורמולה** | דאנג גווי ין זי (当归饮子) |

## 2. גרד / פרוריטוס (皮肤瘙痒 Pi Fu Sao Yang)

### 2.1 סקירה כללית

גרד ללא פריחה ראשונית (pruritus sine materia) הוא אתגר שכיח. ברפואה הסינית, הגורם המרכזי לגרד הוא תמיד **רוח (风)**:

> **"无风不作痒" (Wu Feng Bu Zuo Yang)**
> **"ללא רוח אין גרד"**

אך הרוח יכולה להיות חיצונית או פנימית, והגורם לרוח הפנימית קובע את הטיפול.

### 2.2 דפוסים

#### דפוס 1: חוסר דם מייצר רוח (血虚生风)

| מאפיין | פרטים |
|---|---|
| **אוכלוסייה** | נשים אחרי לידה, קשישים, אנמיים |
| **ביטוי** | גרד כללי, מחמיר בלילה, עור יבש |
| **לשון** | חיוורת, יבשה |
| **דופק** | דק (细), חלש |
| **עיקרון** | להזין דם ולרכך רוח (养血润燥熄风) |
| **נקודות** | SP10, BL17, SP6, ST36, LR8, LI11, GB31 |
| **פורמולה** | סי וו טאנג (四物汤) + פאנג פנג, צ'אן טוי, חה שו וו |

#### דפוס 2: רוח כבד (肝风)

| מאפיין | פרטים |
|---|---|
| **אוכלוסייה** | אנשים עם לחץ דם גבוה, מתוחים, כועסים |
| **ביטוי** | גרד שמחמיר עם מתח ועצבנות, עשוי לנדוד |
| **לשון** | צדדים אדומים |
| **דופק** | מיתרי (弦) |
| **עיקרון** | להרגיע כבד ולרכך רוח (平肝熄风) |
| **נקודות** | LR3, GB20, GB34, LI11, SP10, GV20 |
| **פורמולה** | טיאן מא גו טנג ין (天麻钩藤饮) + מרכיבים נגד גרד |

#### דפוס 3: לחות-חום (湿热)

| מאפיין | פרטים |
|---|---|
| **אוכלוסייה** | אזורי קפלים, מפשעות, בית השחי |
| **ביטוי** | גרד עם לחות, הפרשות, הרגשת "חום ורטיבות" |
| **לשון** | ציפוי צהוב שמנוני |
| **דופק** | חלקלק (滑), מהיר (数) |
| **עיקרון** | לנקות חום ולייבש לחות (清热利湿) |
| **נקודות** | SP9, SP6, LI11, ST36, SP10, BL20 |
| **פורמולה** | לונג דאן שיה גאן טאנג (龙胆泻肝汤) |

## 3. נקודות מפתח לגרד ואורטיקריה

### 3.1 נקודות בסיס — "ארבע נקודות לגרד"

| נקודה | שם | פעולה ספציפית |
|---|---|---|
| **LI11** | צ'ו צ'י (曲池) | הנקודה מס' 1 לגרד — מנקה חום, מגרשת רוח |
| **SP10** | שואה האי (血海) | מקררת דם, מפזרת רוח בדם |
| **BL17** | גה שו (膈俞) | נקודת השפעה של דם — מכסה את כל מצבי הדם |
| **GB31** | פנג שי (风市) | "שוק הרוח" — מגרשת רוח מהעור |

### 3.2 נקודות נוספות לפי מצב

| מצב | נקודות נוספות |
|---|---|
| גרד בפלג גוף עליון | LI4, LU7, GB20 |
| גרד בפלג גוף תחתון | SP6, SP9, ST36 |
| גרד בראש ובפנים | GV20, GB20, LI4, ST44 |
| גרד באזור המפשעות | LR5, SP6, SP9 |
| גרד שמחמיר בלילה | HT7, SP6, KI6 (חוסר ין/דם) |
| גרד שמחמיר בקור | מוקסה על GV14, ST36, LI4 |

### 3.3 טכניקות מיוחדות

#### הקזת דם (刺血 Ci Xue)
- **נקודות**: LI11, BL40, GV14 (גב האוזן)
- **טכניקה**: דקירה מהירה עם מחט שלושה קצוות + כוסות רוח
- **שימוש**: חום חריף, אורטיקריה חמורה
- **יעילות**: לעיתים מביא הקלה מיידית

#### דיקור אוזן
- **נקודות**: Shen Men, Lung, Allergy, Adrenal, Endocrine
- **שימוש**: טיפול משלים, ניתן להשאיר זרעים (ear seeds)

## 4. פורמולות צמחיות

### 4.1 לאורטיקריה

| פורמולה | סינית | דפוס |
|---|---|---|
| שיאו פנג סאן | 消风散 | רוח-חום/לחות — בסיסית |
| גווי ג'י טאנג + פאנג פנג | 桂枝汤加防风 | רוח-קור |
| דאנג גווי ין זי | 当归饮子 | חוסר דם כרוני |
| יו פינג פנג סאן | 玉屏风散 | מניעה — חיזוק ווי צ'י |

### 4.2 לגרד

| פורמולה | סינית | דפוס |
|---|---|---|
| סי וו טאנג + פאנג פנג | 四物汤加防风 | חוסר דם |
| שיאו יאו סאן | 逍遥散 | סטגנציית כבד + גרד |
| לונג דאן שיה גאן טאנג | 龙胆泻肝汤 | לחות-חום |
| ליו ווי די הואנג וואן | 六味地黄丸 | חוסר ין עם יובש |

### 4.3 צמחים בודדים חשובים לגרד

| צמח | סינית | פעולה |
|---|---|---|
| פאנג פנג | 防风 | מגרש רוח — הצמח הבסיסי ביותר לגרד |
| צ'אן טוי | 蝉蜕 | קליפת ציקדה — מגרש רוח, מפחית גרד |
| באי ג'י לי | 白蒺藜 | מגרש רוח, מפזר סטגנציית כבד |
| די פו זי | 地肤子 | מייבש לחות, מפחית גרד |
| באי שיאן פי | 白鲜皮 | מנקה חום-לחות מהעור |
| חה שו וו | 何首乌 | מזין דם — לגרד מחוסר דם |

## 6. אבחנה מבדלת — מתי להפנות?

| מצב | סימנים מחשידים | פעולה |
|---|---|---|
| **אנגיואדמה** | נפיחות בשפתיים/לשון/גרון, קושי בנשימה | חירום רפואי — הפנה מיידית |
| **אנפילקסיס** | ירידת לחץ דם, קושי בנשימה, סרפדת נרחבת | חירום — אדרנלין + הפנייה |
| **גרד עם ירידה במשקל** | גרד כללי + ירידה לא מוסברת במשקל | שלול ממאירות |
| **גרד עם צהבת** | גרד + עיניים/עור צהובים | בדיקת כבד |
| **אורטיקריה + חום** | פפולות + חום + כאב מפרקים | שלול מחלה אוטואימונית |

## 7. סיכום

> אורטיקריה וגרד הם ביטויים של **רוח** (风) — חיצונית או פנימית. במצבים חריפים, הגישה היא גירוש רוח וניקוי הגורם הפתוגני (חום/קור/לחות). במצבים כרוניים, המפתח הוא **הזנת דם** — כי דם חזק "מעגן" את הרוח ומונע ממנה לנדוד בעור. שילוב נקודות הבסיס (LI11, SP10, BL17, GB31) עם נקודות מותאמות לדפוס, וצמחי מרפא, נותן תוצאות טובות במיוחד.
