בסיום שיעור זה, הסטודנט יוכל:

## 1. חרדה (焦虑 Jiao Lv)

### 1.1 הבנה כללית

חרדה ברפואה הסינית קשורה לשיבוש בשן (神) ולחוסר יציבות של צ'י הלב. המושג הקלאסי הקרוב ביותר הוא **"心悸" (Xin Ji) — דפיקות לב** ו-**"怔忡" (Zheng Chong) — פחד ובלבול"**.

חרדה יכולה לנבוע מ:
- **חסרון** (虚): חוסר בדם, ין, או צ'י שמעגנים את השן
- **עודף** (实): חום, ריר, או סטגנציה שמפריעים לשן

### 1.2 דפוסי חרדה

#### דפוס 1: חוסר צ'י/ין בלב (心气/阴虚)

| מאפיין | פרטים |
|---|---|
| **סימנים** | דפיקות לב, חרדה שמחמירה בלילה, הזעת לילה, חום בכפות ידיים/רגליים |
| **לשון** | אדומה, ללא ציפוי, סדקים |
| **דופק** | דק (细), מהיר (数) |
| **נקודות** | HT7, HT6, PC6, KI6, SP6, BL15 |
| **פורמולה** | טיאן וואנג בו שין דאן (天王补心丹) |

#### דפוס 2: חוסר כליות עם פחד (肾虚恐惧)

| מאפיין | פרטים |
|---|---|
| **סימנים** | פחד כרוני, בהלות, כאב גב תחתון, הרטבת לילה, ברכיים חלשות |
| **לשון** | חיוורת או אדומה (תלוי בין/יאנג) |
| **דופק** | עמוק (沉), חלש (弱) |
| **נקודות** | KI3, KI7, BL23, GV4, HT7, Yin Tang |
| **פורמולה** | ליו ווי די הואנג וואן (六味地黄丸) + מרכיבים מרגיעים |

#### דפוס 3: סטגנציית צ'י כבד (肝气郁结)

| מאפיין | פרטים |
|---|---|
| **סימנים** | חרדה עם עצבנות, תחושת גוש בגרון, לחץ בחזה, אנחות, PMS |
| **לשון** | צדדים אדומים/סגולים |
| **דופק** | מיתרי (弦) |
| **נקודות** | LR3, LR14, PC6, GB34, CV17, Yin Tang |
| **פורמולה** | שיאו יאו סאן (逍遥散) |

#### דפוס 4: ריר מפריע ללב (痰扰心神)

| מאפיין | פרטים |
|---|---|
| **סימנים** | חרדה עם בלבול, תחושת כובד בראש, בחילות, ערפול מנטלי |
| **לשון** | ציפוי שמן, דביק |
| **דופק** | חלקלק (滑) |
| **נקודות** | ST40, PC5, HT7, GV20, CV12, SP9 |
| **פורמולה** | וון דאן טאנג (温胆汤) |

## 2. דיכאון (抑郁 Yi Yu)

### 2.1 הבנה כללית

דיכאון ברפואה הסינית מתואר לעיתים קרובות תחת הקטגוריה של **"郁证" (Yu Zheng) — תסמונות סטגנציה"**. ג'ו דאן שי (朱丹溪) כתב: **"气血冲和，万病不生，一有怫郁，诸病生焉"** — "כאשר צ'י ודם זורמים בהרמוניה, אין מחלה; ברגע שיש סטגנציה, מחלות נוצרות".

### 2.2 דפוסי דיכאון

#### דפוס 1: סטגנציית צ'י כבד (肝气郁结) — **הנפוץ ביותר**

| מאפיין | פרטים |
|---|---|
| **סימנים** | מצב רוח ירוד, עצבנות, אנחות, לחץ בחזה/צדדים, גוש בגרון (梅核气 Mei He Qi) |
| **לשון** | רגילה או צדדים אדומים |
| **דופק** | מיתרי (弦) |
| **נקודות** | LR3, LR14, PC6, GB34, CV17, GV20, Yin Tang |
| **פורמולה** | שיאו יאו סאן (逍遥散), צ'אי הו שו גאן סאן (柴胡疏肝散) |

#### דפוס 2: חוסר דם לב (心血虚)

| מאפיין | פרטים |
|---|---|
| **סימנים** | עצב, שכחנות, נדודי שינה, עייפות, דפיקות, חיוורון, סחרחורת |
| **לשון** | חיוורת, דקה |
| **דופק** | דק (细), חלש (弱) |
| **נקודות** | HT7, BL15, BL17, SP6, ST36, GV20 |
| **פורמולה** | גווי פי טאנג (归脾汤) |

#### דפוס 3: חוסר צ'י טחול (脾气虚)

| מאפיין | פרטים |
|---|---|
| **סימנים** | חשיבת יתר, דאגנות, עייפות, תיאבון ירוד, שלשול, נפיחות |
| **לשון** | חיוורת, נפוחה, סימני שיניים |
| **דופק** | חלש (弱), רך (濡) |
| **נקודות** | ST36, SP3, SP6, CV12, BL20, GV20 |
| **פורמולה** | סי ג'ון זי טאנג (四君子汤) + תוספות |

#### דפוס 4: חוסר יאנג כליות (肾阳虚)

| מאפיין | פרטים |
|---|---|
| **סימנים** | אדישות עמוקה, חוסר מוטיבציה, קור, עייפות חמורה, ליבידו נמוך |
| **לשון** | חיוורת, נפוחה, ציפוי לבן רטוב |
| **דופק** | עמוק (沉), איטי (迟), חלש |
| **נקודות** | GV4, KI3, KI7, BL23, CV4, ST36, מוקסה |
| **פורמולה** | ג'ין גווי שן צ'י וואן (金匮肾气丸) |

## 3. נקודות מפתח לחרדה ודיכאון

### 3.1 נקודות בסיסיות (משותפות לרוב הדפוסים)

| נקודה | שם | פעולה |
|---|---|---|
| **HT7** | שן מן (神门) | שער הרוח — מרגיעה שן, מטפלת בנדודי שינה, חרדה |
| **PC6** | ניי גואן (内关) | שער פנימי — פותחת חזה, מרגיעה לב, מונעת בחילה |
| **GV20** | באי הוי (百会) | מאה מפגשים — מרימה רוח, מבהירה תודעה |
| **Yin Tang** | ין טאנג (印堂) | אולם החותם — מרגיעה רוח, מפחיתה חרדה |
| **LR3** | טאי צ'ונג (太冲) | מפזרת סטגנציית כבד, מווסתת רגשות |
| **SP6** | סאן ין ג'יאו (三阴交) | מזינה דם וין, מרגיעה רוח |

### 3.2 נקודות נוספות לפי הצורך

- **ST36** (זו סאן לי) — לחיזוק כללי, עייפות
- **CV17** (שאן ג'ונג) — לפתיחת החזה, לחץ רגשי
- **GV24** (שן טינג) — להרגעה עמוקה, מאניה
- **An Mian** (אן מיאן) — נקודה חוץ-ערוצית לשינה
- **KI1** (יונג צ'ואן) — להורדת עודף מהראש

## 4. פורמולות צמחיות מרכזיות

### 4.1 לחרדה

| פורמולה | סינית | שימוש עיקרי |
|---|---|---|
| טיאן וואנג בו שין דאן | 天王补心丹 | חוסר ין לב/כליות עם חרדה ונדודי שינה |
| סואן זאו רן טאנג | 酸枣仁汤 | חוסר דם כבד עם נדודי שינה |
| גאן מאי דא זאו טאנג | 甘麦大枣汤 | היסטריה, בכי פתאומי (臟躁 Zang Zao) |

| פורמולה | סינית | שימוש עיקרי |
|---|---|---|
| שיאו יאו סאן | 逍遥散 | סטגנציית כבד + חולשת טחול (הבסיסית ביותר) |

| יואה ג'ו וואן | 越鞠丸 | שש סטגנציות (צ'י, דם, ריר, אש, לחות, מזון) |

## 5. בסיס מחקרי

### 5.1 מחקרים מרכזיים

- **מטא-אנליזה (2019):** דיקור נמצא יעיל יותר מטיפול מדומה (sham) לחרדה, עם גודל אפקט בינוני
- **סקירת Cochrane:** עדויות מעודדות לדיקור כטיפול משלים לדיכאון, במיוחד בשילוב עם SSRI
- **מנגנון מוצע:** דיקור ב-HT7, PC6 ו-GV20 משפיע על ציר HPA, רמות סרוטונין ו-GABA
- **מחקרי fMRI:** דיקור ב-LR3 מפחית פעילות באמיגדלה

### 5.2 שילוב עם טיפול קונבנציונלי

- דיקור כטיפול **משלים** — לא כתחליף לטיפול פסיכיאטרי/פסיכולוגי
- יש לשים לב לאינטראקציות בין צמחי מרפא ותרופות (במיוחד SSRI + St. John's Wort)
- תקשורת עם הצוות המטפל חיונית
- מטופלים עם מחשבות אובדניות — להפנות מיידית לגורם מקצועי

## 6. פרוטוקול טיפולי מעשי

### 6.1 חרדה — פרוטוקול בסיסי

| שלב | פעולה |
|---|---|
| 1 | הערכה: זיהוי דפוס (חסרון/עודף, איזה איבר) |
| 2 | נקודות בסיס: HT7, PC6, GV20, Yin Tang |
| 3 | נקודות לפי דפוס: הוסף 3-4 נקודות ספציפיות |
| 4 | טכניקה: דיקור עדין, שימור 20-30 דקות, אווירה רגועה |
| 5 | תדירות: פעמיים בשבוע ל-4-6 שבועות, ואז הפחתה |
| 6 | שילוב: הדרכה לנשימות, תזונה, שינה |

### 6.2 דיכאון — פרוטוקול בסיסי

| שלב | פעולה |
|---|---|
| 1 | הערכה: חומרה, סיכון אובדני, דפוס TCM |
| 2 | נקודות בסיס: LR3, GV20, Yin Tang, HT7 |
| 3 | נקודות לפי דפוס: הוסף 3-4 נקודות ספציפיות |
| 4 | טכניקה: גירוי חזק יותר (לסטגנציה), עדין (לחסרון) |
| 5 | תדירות: פעמיים בשבוע, סדרה של 8-12 טיפולים |
| 6 | שילוב: תרגול גופני, טיפול רגשי, צמחי מרפא |

## 7. סיכום

> חרדה ודיכאון הם מהתלונות הנפוצות ביותר בקליניקה. הרפואה הסינית מציעה מסגרת אבחנתית מדויקת המאפשרת טיפול מותאם אישית. המפתח הוא אבחנה מדויקת של הדפוס — האם מדובר בחסרון שצריך חיזוק, או עודף שצריך פיזור — ולהתאים את הנקודות, הצמחים וגישת הטיפול בהתאם.
