# דפוסי נדודי שינה (失眠)

## Insomnia Patterns (Shi Mian)

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. להבחין בין סוגי נדודי שינה שונים ולקשר כל סוג לדפוס TCM
2. לזהות ולאבחן את ששת הדפוסים העיקריים של נדודי שינה
3. לבנות פרוטוקול טיפולי מותאם לכל דפוס
4. להכיר נקודות מפתח ופורמולות צמחיות לכל דפוס
5. לשלב הנחיות היגיינת שינה בטיפול

## 1. הבנה כללית — שינה ברפואה הסינית

### 1.1 מנגנון השינה

שינה ברפואה הסינית מוסברת דרך תנועת ווי צ'י (卫气 Wei Qi):

- **ביום**: ווי צ'י נעה **בשטח** (יאנג) → ערות
- **בלילה**: ווי צ'י נכנסת **פנימה** לאיברי הין → שינה

כמו כן, שינה תלויה ב:
- **שן (神) יציב** — השן חייב להיות מעוגן בלב כדי שהאדם יירדם
- **דם הלב מספיק** — דם הוא ה"עוגן" של השן
- **הון (魂) שוכן בכבד** — ההון שב לכבד בלילה; אם דם הכבד חלש, ההון "נודד" → חלומות מרובים

> **"阳入于阴则寐，阳出于阴则寤"**
> **"כאשר היאנג נכנס לין — שינה; כאשר היאנג יוצא מהין — ערות"**

### 1.2 סיווג סוגי נדודי שינה

| סוג | תיאור | דפוסים נפוצים |
|---|---|---|
| **קושי להירדם** | שוכב ער זמן רב | סטגנציית צ'י כבד, חוסר דם לב-טחול |
| **התעוררויות באמצע הלילה** | מתעורר 1-3 בלילה | חום בכבד, חוסר ין |
| **התעוררות מוקדמת** | מתעורר 3-5 בלילה ואינו נרדם | חוסר ין ריאות/כליות |
| **שינה עם חלומות מרובים** | חלומות מטרידים, שינה לא מרעננת | חוסר דם כבד, אש בכבד |
| **שינה קלה/שטחית** | מתעורר מכל רעש | חוסר לב-כיס מרה (心胆虚 Xin Dan Xu) |
| **שינה לא מרעננת** | ישן אך מתעורר עייף | ריר-לחות, סטגנציית מזון |

## 2. דפוסים עיקריים של נדודי שינה

### 2.1 חוסר דם לב-טחול (心脾两虚)

**הסוג הנפוץ ביותר בקליניקה**

| מאפיין | פרטים |
|---|---|
| **מנגנון** | טחול חלש → לא מייצר מספיק דם → דם לב חסר → שן לא מעוגן |
| **סוג נדודי שינה** | קושי להירדם, שינה קלה, חלומות |
| **סימנים נלווים** | עייפות, חיוורון, שכחנות, דפיקות, תיאבון ירוד, צואה רכה |
| **לשון** | חיוורת, דקה |
| **דופק** | דק (细), חלש (弱) |

**טיפול:**

| נקודות | פעולה |
|---|---|
| HT7 (שן מן) | מעגנת את השן |
| SP6 (סאן ין ג'יאו) | מזינה דם, מרגיעה |
| BL15 (שין שו) | נקודת גב לב — מזינה דם לב |
| BL20 (פי שו) | נקודת גב טחול — מחזקת ייצור דם |
| ST36 (זו סאן לי) | מחזקת טחול וצ'י |
| An Mian (אן מיאן) | נקודה חוץ-ערוצית לשינה |
| GV20 (באי הוי) | מרימה צ'י, מבהירה תודעה |

**פורמולה:** גווי פי טאנג (归脾汤)

### 2.2 חוסר ין לב עם חום ריק (心阴虚 / 虚火)

| מאפיין | פרטים |
|---|---|
| **מנגנון** | חוסר ין → חום ריק עולה → מפריע לשן |
| **סוג נדודי שינה** | מתעורר באמצע הלילה, שינה שטחית, חלומות |
| **סימנים נלווים** | דפיקות, הזעת לילה, חום בכפות ידיים/רגליים, פה יבש, אי-שקט |
| **לשון** | אדומה, ללא ציפוי, סדקים בקצה |
| **דופק** | דק (细), מהיר (数) |

**טיפול:**

| נקודות | פעולה |
|---|---|
| HT7 (שן מן) | מעגנת שן, מנקה חום ריק |
| HT6 (ין שי) | נקודת שי — להזעת לילה וחום ריק |
| KI6 (ג'או האי) | מזינה ין כליות |
| SP6 (סאן ין ג'יאו) | מזינה ין ודם |
| PC7 (דא לינג) | מנקה חום לב |
| Yin Tang | מרגיעה רוח |

**פורמולה:** טיאן וואנג בו שין דאן (天王补心丹) או הואנג ליאן א ג'יאו טאנג (黄连阿胶汤)

### 2.3 אש בכבד (肝火上炎)

| מאפיין | פרטים |
|---|---|
| **מנגנון** | כעס/תסכול כרוני → סטגנציית כבד הופכת לאש → עולה ומפריעה לשן |
| **סוג נדודי שינה** | שינה עם סיוטים, חלומות אלימים, התעוררות בכעס |
| **סימנים נלווים** | עצבנות, כאב ראש, עיניים אדומות, טעם מר, עצירות |
| **לשון** | אדומה, צדדים אדומים במיוחד, ציפוי צהוב |
| **דופק** | מיתרי (弦), מהיר (数) |

**טיפול:**

| נקודות | פעולה |
|---|---|
| LR2 (שינג ג'יאן) | נקודת ינג-אש — מנקה אש כבד |
| LR3 (טאי צ'ונג) | מפזרת סטגנציה, מורידה יאנג |
| GB20 (פנג צ'י) | מורידה יאנג מהראש |
| GV20 (באי הוי) | מבהירה ראש |
| HT7 (שן מן) | מרגיעה שן |
| An Mian | לשינה |

**פורמולה:** לונג דאן שיה גאן טאנג (龙胆泻肝汤)

### 2.4 סטגנציית מזון (食积)

| מאפיין | פרטים |
|---|---|
| **מנגנון** | אכילת יתר / אכילה מאוחרת → מזון לא מעוכל → חום וריר → מפריעים לשן |
| **סוג נדודי שינה** | שינה לא מרעננת, כובד, התעוררות עם בחילה |
| **סימנים נלווים** | נפיחות, גיהוקים, ריפלוקס, טעם רע בפה, צואה מסריחה |
| **לשון** | ציפוי עבה, שמנוני |
| **דופק** | חלקלק (滑) |

**טיפול:**

| נקודות | פעולה |
|---|---|
| CV12 (ג'ונג וואן) | מווסתת קיבה |
| ST36 (זו סאן לי) | מחזקת עיכול |
| ST44 (ניי טינג) | מנקה חום קיבה |
| SP4 (גונג סון) | מווסתת טחול-קיבה |
| PC6 (ניי גואן) | מורידה צ'י הפוך, מרגיעה |

**פורמולה:** באו הה וואן (保和丸)

### 2.5 ניתוק לב-כליות (心肾不交)

| מאפיין | פרטים |
|---|---|
| **מנגנון** | אש הלב לא יורדת לחמם כליות; מים הכליות לא עולים לקרר לב → חום למעלה, קור למטה |
| **סוג נדודי שינה** | נדודי שינה חמורים, אי-שקט בלילה, שינה מועטה |
| **סימנים נלווים** | דפיקות, אי-שקט, כאב גב, טינטון, סחרחורת, רגליים קרות |
| **לשון** | אדומה בקצה, חיוורת בשורש |
| **דופק** | מהיר בגואן, חלש בצ'י |

**טיפול:**

| נקודות | פעולה |
|---|---|
| HT7 (שן מן) | מנקה אש לב, מרגיעה שן |
| KI6 (ג'או האי) | מזינה ין כליות |
| KI3 (טאי שי) | מחזקת כליות |
| PC7 (דא לינג) | מנקה חום לב |

**פורמולה:** ג'יאו טאי וואן (交泰丸) — הואנג ליאן + רו גווי

### 2.6 ריר-חום (痰热)

| מאפיין | פרטים |
|---|---|
| **מנגנון** | ריר מצטבר + חום → ריר-חום עולה ומפריע לשן |

| **סימנים נלווים** | תחושת כובד בראש וגוף, סחרחורת, בחילה, כיח |
| **לשון** | ציפוי צהוב, שמנוני |
| **דופק** | חלקלק (滑), מהיר (数) |

**טיפול:**

| נקודות | פעולה |

| ST40 (פנג לונג) | מפזרת ריר |
| PC5 (ג'יאן שי) | מנקה ריר מהלב |

| CV12 (ג'ונג וואן) | מווסתת ג'יאו אמצעי |

**פורמולה:** וון דאן טאנג (温胆汤) + הואנג ליאן

## 5. סיכום

> נדודי שינה הם סימפטום שכיח אך מורכב. המפתח לטיפול יעיל הוא אבחנה מדויקת של הדפוס: מהו **סוג** נדודי השינה (קושי להירדם, התעוררויות, שינה לא מרעננת), מהם הסימנים הנלווים, ומהו מצב הלשון והדופק. כל דפוס דורש גישה טיפולית שונה — מחיזוק דם וין ועד ניקוי חום וריר.
