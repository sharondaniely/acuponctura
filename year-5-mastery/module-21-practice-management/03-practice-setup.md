# הקמת קליניקה

## Practice Setup (诊所建设)

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לתכנן ולעצב חלל קליניקה מתאים לטיפולי אקופונקטורה
2. להכיר את הדרישות הרגולטוריות והביטוחיות
3. לבנות נוכחות דיגיטלית ומערכת שיווק
4. לנהל את ההיבטים הפיננסיים של קליניקה פרטית
5. לבנות רשת הפניות מקצועית

---

## 1. עיצוב ותכנון הקליניקה

### 1.1 בחירת מיקום

| שיקול | פירוט |
|---|---|
| **נגישות** | תחבורה ציבורית, חנייה, גישה לנכים |
| **אוכלוסיית יעד** | קרבה לקהל המטרה (שכונת מגורים / מרכז עסקים) |
| **שכנות** | קרבה לקליניקות משלימות (פיזיותרפיה, נטורופתיה) |
| **תקציב** | שכר דירה סביר — אל תתחילו "גדול מדי" |
| **אווירה** | שכונה שקטה, תחושת ביטחון |

### 1.2 חללי הקליניקה

| חלל | דרישות מינימום | הערות |
|---|---|---|
| **חדר המתנה** | ישיבה נוחה, שקט, חומרי קריאה | ראשון שנראה — רושם ראשוני |
| **חדר טיפול** | מיטת טיפול, תאורה מתכווננת, פרטיות | מינימום 9 מ"ר |
| **אזור שטיפה** | כיור, סבון, מגבונים, פח מחטים | חובה — היגיינה |
| **אחסון** | ארון נעול לתיקים, ציוד | שמירה על פרטיות ובטיחות |
| **שירותים** | נגישים למטופלים | נקיים תמיד |

### 1.3 אווירה טיפולית

- **צבעים**: גוונים רכים — בז', ירוק זית, תכלת
- **תאורה**: מעומעמת, טבעית עדיפה
- **קול**: מוזיקה שקטה או שקט מוחלט (שאלו את המטופל)
- **ריח**: נקי, ניטרלי (לא בשמים חזקים)
- **טמפרטורה**: חמה — מטופלים שוכבים ללא תנועה

## 2. ציוד וחומרים מתכלים

### 2.1 ציוד בסיסי

| פריט | כמות התחלתית | הערות |
|---|---|---|
| **מיטת טיפול** | 1-2 | איכותית, מתכווננת, עם חור לפנים |
| **כיסא מטפל** | 1-2 | מתגלגל, מתכוונן |
| **כריות/גלילים** | 3-4 | לתמיכת ברכיים, צוואר |
| **שמיכות** | 4-6 | תמיד נקיות, חמות |
| **סדיני נייר/בד** | מלאי | חד-פעמי או כביסה |
| **תאורת נקודה** | 1 | לאיתור נקודות |
| **דגם אנטומי** | 1 | להסבר למטופלים |

### 2.2 ציוד טיפולי

| פריט | פירוט |
|---|---|
| **מחטים** | מגוון גדלים: 0.20×13, 0.25×25, 0.25×40, 0.30×50 |
| **מוקסה** | מקלות, קונוסים, מוקסה בקופסה |
| **כוסות רוח** | סט כוסות זכוכית/פלסטיק + משאבה |
| **גואה שא** | כלי חרסינה או קרן שור |
| **זרעי אוזן** | זרעי Vaccaria + פלסטר |
| **אלקטרו** | מכשיר אלקטרו-אקופונקטורה |
| **כדורי כותנה, אלכוהול** | לחיטוי |
| **פח מחטים** | Sharps Container — חובה חוקית |

## 3. דרישות רגולטוריות

### 3.1 רישוי

| דרישה | פירוט |
|---|---|
| **רישום מטפל** | רישום במשרד הבריאות כמטפל ברפואה משלימה |
| **תעודת הסמכה** | הצגת תעודה ממוסד מוכר |
| **רישיון עסק** | מהרשות המקומית |
| **ביטוח אחריות מקצועית** | חובה — מכסה תביעות רשלנות |
| **תקני בטיחות** | עמידה בתקנות כיבוי אש, נגישות |
| **פינוי פסולת רפואית** | חוזה עם חברת פינוי מורשית |

### 3.2 תיעוד חובה

- **טופס הסכמה מדעת** — חתום לפני טיפול ראשון
- **כרטיס מטופל** — אבחנה, טיפול, הערות בכל ביקור
- **רישום מלאי מחטים** — מעקב אחר ציוד חד-פעמי
- **תיעוד תקריות** — אם אירעה תקרית (עילפון, דימום חריג)

## 5. שיווק ונוכחות דיגיטלית

### 5.1 אתר אינטרנט

| מרכיב | חשיבות |
|---|---|
| **עמוד "אודות"** | סיפור אישי, הכשרה, גישה טיפולית |
| **שירותים** | מה מציעים, באילו מצבים מטפלים |
| **מחירון** | שקיפות מחירים |
| **המלצות** | עדויות מטופלים (באישור) |
| **בלוג/מאמרים** | תוכן חינוכי — בונה סמכות |
| **יצירת קשר** | טלפון, אימייל, כתובת, מפה |
| **הזמנת תור אונליין** | נוחות למטופל |

### 5.2 רשתות חברתיות

- **אינסטגרם/פייסבוק** — תוכן חינוכי, טיפים, מאחורי הקלעים
- **גוגל ביזנס** — קריטי להופעה בחיפוש מקומי
- **ביקורות** — בקשו ממטופלים מרוצים לכתוב ביקורת

### 5.3 שיווק אורגני

- **הרצאות** בקהילה — ספריות, מועדונים, חברות
- **סדנאות** — "מבוא לאקופונקטורה", "לחיצה עצמית"
- **שיתופי פעולה** — עם מטפלים משלימים, חדרי כושר, סטודיו יוגה
- **תוכן** — מאמרים, וידאו, פודקאסטים

## 6. ניהול פיננסי

### 6.1 תמחור

| שיקול | הנחיה |
|---|---|
| **מחיר שוק** | בדקו מה מקובל באזור שלכם |
| **עלויות** | שכ"ד + ציוד + ביטוח + מסים + שיווק |
| **ערך** | מחיר משקף ערך — אל תתמחרו נמוך מדי |
| **חבילות** | הציעו חבילות (5/10 טיפולים) — מחייב ומשתלם |
| **מדיניות ביטולים** | הגדירו מראש — 24 שעות ביטול |

### 6.2 ניהול כספי

- **הנהלת חשבונות** — רואה חשבון מההתחלה
- **חשבוניות** — לכל מטופל, בכל טיפול
- **ניהול תזרים** — מעקב חודשי אחר הכנסות והוצאות
- **חיסכון** — 25-30% מההכנסה למסים וביטוח

- אל תסתמכו על זיכרון — תעדו הכל דיגיטלית

## 8. בניית רשת הפניות

### 8.1 מקורות הפניה

| מקור | כיצד לבנות |
|---|---|
| **רופאים** | פגישות היכרות, מכתבים מקצועיים, שיתוף תוצאות |
| **מטפלים משלימים** | רשת הפניות הדדית (נטורופת, אוסטאופת, פסיכולוג) |
| **מטופלים** | מטופלים מרוצים = שיווק הכי טוב |
| **קהילה** | הרצאות, סדנאות, נוכחות מקומית |

### 8.2 מכתב הפניה מקצועי

כשמפנים מטופל לרופא, כתבו מכתב מקצועי:
- **פרטי מטופל** + **סיבת ההפניה**
- **ממצאים רלוונטיים** (בשפה רפואית)
- **בקשה ספציפית** (בדיקה, חוות דעת)
- **פרטי התקשרות** שלכם
