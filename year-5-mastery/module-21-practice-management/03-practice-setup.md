# הקמת קליניקה

## Practice Setup (诊所建设)

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לתכנן ולעצב קליניקה מתאימה
2. לרכוש ציוד נדרש
3. להבין דרישות רגולטוריות וביטוח
4. לבנות נוכחות דיגיטלית ושיווק
5. לנהל את הצד הכספי של הקליניקה
6. לפתח רשת הפניות מקצועית

---

## 1. תכנון הקליניקה

### 1.1 מיקום

| שיקול | פירוט |
|---|---|
| **נגישות** | קל להגיע, חנייה, תחבורה ציבורית |
| **קהל יעד** | קרבה לאוכלוסיית היעד |
| **שכנות** | רצוי ליד מרפאות/ג'ימים/מרכזי בריאות |
| **שכירות** | עלות סבירה ביחס להכנסה צפויה |
| **רגולציה** | בדוק שימוש מותר לקליניקה |

### 1.2 עיצוב חלל הטיפול

**חדר טיפול**:
- מיטת טיפול מתכוננת, 70 ס"מ רוחב לפחות
- תאורה ניתנת לעמעום
- טמפרטורה נוחה (22-25°C)
- פרטיות: דלת נסגרת, מחיצות בין מיטות
- שקט: בידוד אקוסטי מהרחוב

**חדר המתנה**:
- ריהוט נוח ומזמין
- חומרי קריאה על TCM
- מים/תה
- פרטיות: אין שיחות על מטופלים

**שירותים**:
- נגישים ונקיים
- סבון, מגבות נייר
- פח סניטרי

### 1.3 אווירה

- **צבעים**: גוונים טבעיים, חמים — לא פלורסנט קר
- **ריח**: עדין — אפשר קטורת או שמן אתרי קל
- **מוזיקה**: מוזיקה שקטה, אופציונלית (שאל את המטופל)
- **ניקיון**: חובה מוחלטת — רושם מקצועי

## 2. ציוד

### 2.1 ציוד בסיסי

| פריט | פירוט |
|---|---|
| **מיטת טיפול** | מתכוונת גובה, ריפוד איכותי, כרית פנים |
| **מחטים** | מידות שונות (0.20×25, 0.25×25, 0.25×40, 0.30×50) |
| **מחטי אוזן** | ASP, זרעי ואקריה, פלסטרים |
| **מוקסה** | רולים, קונוסים, מוקסה ללא עשן |
| **גביעים** | סט גביעי זכוכית או סיליקון, 4-6 מידות |
| **אלקטרו** | מכשיר אלקטרו-דיקור (ES-160 או דומה) |
| **חיטוי** | כפפות, אלכוהול 70%, מגשים חד-פעמיים |
| **מגבות/סדינים** | ספי נייר חד-פעמי + מגבות כותנה |

### 2.2 ציוד עזר

| פריט | שימוש |
|---|---|
| **מנורת TDP** | מנורת אינפרא-אדום — חימום אזורי |
| **כרית חימום** | חימום מטופלים קרים |
| **גואה שא כלים** | כפות/לוחות גואה שא |
| **סטטוסקופ + מד לחץ דם** | בדיקות בסיסיות |
| **דגם אנטומי** | עזר הסבר למטופלים |
| **מפות ערוצים** | פוסטרים ללימוד ולהסבר |

### 2.3 ציוד משרדי

- מחשב + תוכנת ניהול מטופלים
- מדפסת (טפסים, קבלות)
- טלפון מקצועי (נפרד מאישי)
- ארון תיקים נעול (GDPR/פרטיות)

## 3. דרישות רגולטוריות

### 3.1 רישוי

| דרישה | פירוט |
|---|---|
| **רישיון מטפל** | תעודת הסמכה מוכרת, לפי חוק מקומי |
| **רישיון עסק** | רישיון עירוני לקליניקה |
| **רישום מקצועי** | רישום בגוף המפקח (משרד הבריאות / לשכה) |
| **אישור כיבוי אש** | בטיחות (במיוחד עם מוקסה) |
| **נגישות** | עמידה בדרישות נגישות לנכים |

### 3.2 ביטוח

| סוג ביטוח | מה מכסה |
|---|---|
| **ביטוח אחריות מקצועית** | תביעות רשלנות (חובה!) |
| **ביטוח מבנה/תכולה** | נזק לקליניקה, גניבה |
| **ביטוח צד שלישי** | פגיעה במטופל בקליניקה (מעידה וכו') |
| **ביטוח הפסקת עבודה** | אם לא יכול לעבוד (מחלה, פציעה) |

### 3.3 בטיחות ותברואה

- **סילוק מחטים**: מכל חד (Sharps Container) — סילוק מוסדר
- **חיטוי**: פרוטוקול חיטוי בין מטופלים
- **כביסה**: סדינים/מגבות — כביסה ב-60°C לפחות
- **ערכת עזרה ראשונה**: כולל אדרנלין לתגובה אלרגית
- **תיק חירום**: מספרי טלפון חירום, פרוטוקול עילפון

## 4. שיווק ונוכחות דיגיטלית

### 4.1 אתר אינטרנט

| רכיב | תוכן |
|---|---|
| **דף בית** | מי אתה, מה אתה מטפל, איך ליצור קשר |
| **אודות** | הכשרה, ניסיון, גישה טיפולית |
| **שירותים** | סוגי טיפול, מצבים מטופלים |
| **מחירון** | שקיפות — מחיר טיפול, חבילות |
| **יצירת קשר** | טלפון, מייל, כתובת, מפה |
| **בלוג** | מאמרים על TCM — SEO + ערך למטופלים |

### 4.2 רשתות חברתיות

- **נוכחות קבועה**: פרסום 2-3 פוסטים בשבוע
- **תוכן**: טיפים בריאותיים, מאמרים, מקרים (עם הסכמה)
- **אינטראקציה**: תגובות, שאלות ותשובות
- **מקצועיות**: שמור על רמה — לא "רפואת פייסבוק"

### 4.3 שיווק מקומי

- **רשת הפניות**: רופאים, פיזיותרפיסטים, פסיכולוגים
- **הרצאות**: ספריות, מרכזים קהילתיים, חדרי כושר
- **שיתופי פעולה**: סטודיו ליוגה, מכון כושר
- **Google My Business**: רישום + ביקורות

## 5. ניהול כספי

### 5.1 תמחור

| שיקול | פירוט |
|---|---|
| **מחקר שוק** | מה המחירים המקובלים באזורך? |
| **עלויות** | שכירות, ציוד, מחטים, ביטוח, חשמל |
| **יעד הכנסה** | כמה טיפולים בשבוע צריך למחייה? |
| **חבילות** | הנחה על סדרת טיפולים — מעודד התמדה |
| **מחיר ראשון** | טיפול ראשון (ארוך יותר) → מחיר גבוה יותר |

### 5.2 ניהול כספי בסיסי

- **הנהלת חשבונות**: תוכנה או רואה חשבון — מהיום הראשון
- **קבלות**: על כל תשלום — חוקי וחשוב
- **הוצאות מוכרות**: ציוד, שכירות, ביטוח, השתלמויות
- **חיסכון**: הפרש 20-30% לפנסיה/חירום
- **מחזור אנושי**: חודשים חזקים (ינואר, ספטמבר) וחלשים (אוגוסט)

### 5.3 מערכות תשלום

- מזומן, אשראי, העברה בנקאית
- אפליקציית תשלום (Bit, PayBox)
- קופות חולים / ביטוח משלים (אם רלוונטי)
- קבלה + חשבונית

---

## 6. מערכת ניהול מטופלים

### 6.1 תוכנה

| צורך | פתרון |
|---|---|
| **ניהול תיקים** | תוכנת EMR (Electronic Medical Record) |
| **קביעת תורים** | מערכת תורים מקוונת (Cliniko, Jane App) |
| **תזכורות** | SMS/WhatsApp אוטומטי יום לפני |
| **חיוב** | ניהול חשבוניות וקבלות |
| **סטטיסטיקות** | מעקב אחרי מספר מטופלים, הכנסות |

### 6.2 רשומות דיגיטליות

- **גיבוי**: ענן + מקומי
- **אבטחה**: סיסמה, הצפנה, גישה מוגבלת
- **פרטיות**: עמידה בחוקי הגנת מידע
- **נגישות**: גישה מהקליניקה ומהבית (מאובטח)

## 7. רשת הפניות מקצועית

### 7.1 בניית רשת

| מקצוע | ערך הדדי |
|---|---|
| **רופא משפחה** | הפניות הדדיות, מידע רפואי |
| **אורתופד** | כאבי שלד-שריר — הפניות |
| **פסיכולוג** | מטופלים עם מרכיב רגשי |
| **פיזיותרפיסט** | שילוב טיפולי |
| **דיאטנית** | תזונה טיפולית |
| **נטורופתים** | רפואה משלימה |
| **מיילדת** | הריון ולידה |

### 7.2 כיצד לבנות קשרים

- **פגישת היכרות**: הצג את עצמך ואת עבודתך
- **מכתב הפניה מקצועי**: כשמפנה מטופל — מכתב ברור
- **עדכונים**: שלח סיכום טיפולי (עם הסכמת המטופל)
- **כנסים**: השתתף באירועי רשת מקצועיים

---

## 8. סיכום

הקמת קליניקה מצליחה דורשת תכנון בכל הממדים — מעיצוב החלל ועד הכספים, מרגולציה ועד שיווק. ההצלחה נבנית על מקצועיות, תקשורת טובה ובניית אמון — עם מטופלים ועם קולגות.

---

## נקודות מפתח לזכור

- ביטוח אחריות מקצועית — חובה מיום הראשון
- מכל חד (Sharps) — סילוק מחטים לפי תקנות
- אתר + Google My Business = הנוכחות הדיגיטלית הבסיסית
- רשת הפניות: רופאים, פיזיו, פסיכולוגים — השקע ביחסים
- הנהלת חשבונות מסודרת מהיום הראשון — לא "אחר כך"
