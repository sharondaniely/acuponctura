
# שנה ה׳ — מומחיות

## תיאור קצר

שנה חמישית ואחרונה זו משלימה את מסע ההכשרה ומכינה את הסטודנט לפרקטיקה עצמאית מלאה. המודולים מתמקדים באינטגרציה מתקדמת של כל השיטות שנלמדו, ניהול קליניקה, אתיקה, תקשורת עם מטופלים, ומתודולוגיית הוראה — להכשרת הדור הבא של מטפלים.

## מטרות למידה

- לבנות פרוטוקולים אינטגרטיביים המשלבים את כל השיטות שנלמדו
- להבין טיפול חוקתי (Constitutional Treatment) ורפואה מונעת (יאנג שנג — 养生)
- לנהל קליניקה — הקמה, אתיקה קלינית, תקשורת עם מטופלים
- ללמד אקופונקטורה ולהעביר השתלמויות מקצועיות
- להמשיך ולהתפתח כמטפל מומחה לאורך הקריירה
