# פרוטוקולים אינטגרטיביים

## Integrative Protocols (综合治疗方案)

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לשלב מודאליות טיפול שונות בפרוטוקול אחד
2. להשתמש במסגרת החלטה לבחירת כלים טיפוליים
3. להתאים טיפול לפי עונה, חוקה ודפוס
4. לתכנן פרוטוקולים אינטגרטיביים מורכבים
5. לנתח 5 מקרי בוחן מתקדמים

---

## 1. מסגרת אינטגרטיבית

### 1.1 ארסנל הכלים

לאחר ארבע שנות לימוד, למטפל ארסנל מגוון:

| כלי | חוזק עיקרי | מתי להשתמש |
|---|---|---|
| **דיקור גוף** (体针) | טיפול שורש + ענף מקיף | כמעט תמיד — בסיס הטיפול |
| **דיקור אוזן** (耳针) | רגשות, התמכרויות, כאב | תוספת לכל טיפול, טיפול ביתי |
| **דיקור קרקפת** (头针) | מצבים נוירולוגיים | שבץ, שיתוק, MS, כאב כרוני |
| **דיקור בטני** (腹针) | כאב, מטופלים רגישים | פוביית מחטים, ילדים, קשישים |
| **שורש כף יד/קרסול** (腕踝针) | כאב מיידי | כאב חריף, מצבי חירום |
| **מוקסה** (灸法) | חיזוק יאנג, חימום | חסר, קור, חולשה כרונית |
| **כוסות רוח** (拔罐) | שחרור שטח, דם עומד | כאבי שרירים, הצטננות, סטגנציה |
| **גואה שא** (刮痧) | שחרור חום ושטח | חום, כאבי שרירים, הצטננות |
| **אלקטרו** (电针) | כאב, נוירולוגיה | כאב כרוני, שיתוק, ניוון שרירים |
| **צמחי מרפא** (中药) | טיפול פנימי עמוק | מצבים כרוניים, חוקתיים |
| **טואי נא** (推拿) | שרירים, מפרקים, ילדים | כאבי שרירים, ילדים |

### 1.2 מסגרת החלטה — "5 השאלות"

לפני כל טיפול, שאל:
1. **מה הדפוס?** → קובע את בסיס בחירת הנקודות
2. **חריף או כרוני?** → קובע עוצמת גירוי ותדירות
3. **חסר או עודף?** → קובע טוניפיקציה/סדציה, מוקסה/כוסות
4. **מקומי או מערכתי?** → קובע שימוש במיקרו-סיסטם
5. **רגשי או פיזי?** → קובע הוספת אוזן/נקודות רוח

## 2. שילוב מודאליות — עקרונות

### 2.1 כללי שילוב

| שילוב | מתי | דוגמה |
|---|---|---|
| **גוף + אוזן** | כמעט תמיד אפשרי | LV3 + אוזן Liver + Shen Men |
| **גוף + קרקפת** | מצבים נוירולוגיים | נקודות גוף + אזור מוטורי |
| **גוף + מוקסה** | חסר, קור | ST36 + מוקסה על CV4 |
| **גוף + כוסות** | סטגנציה, כאב שרירים | דיקור + כוסות על הגב |
| **גוף + אלקטרו** | כאב כרוני חזק | אלקטרו 2-4 Hz על נקודות מפתח |
| **בטני + אוזן** | מטופלים רגישים | דיקור בטני שטוח + זרעי אוזן |
| **שורש כף/קרסול + גוף** | כאב חריף | WAA להקלה מיידית → גוף לשורש |

### 2.2 מה לא לשלב

- **לא יותר מדי** — 3 מודאליות בטיפול אחד זה מקסימום
- **לא סותרים** — אל תחמם (מוקסה) ותקרר (דיקור חזק) באותו אזור
- **לא מעמיסים** — מטופל חלש לא צריך גירוי חזק מכל הכיוונים

## 3. טיפול עונתי (四季治疗)

### 3.1 עקרונות

הניי ג'ינג מלמד: "החכם מתאים את עצמו לעונות" (智者之养生也，必顺四时). כל עונה דורשת התאמה:

| עונה | אלמנט | איבר | מיקוד טיפולי | מוקסה |
|---|---|---|---|---|
| **אביב** (春) | עץ | כבד | הזזת צ'י, שחרור כבד | לא מומלצת |
| **קיץ** (夏) | אש | לב | הרגעת לב, קירור | מצומצמת |
| **סוף קיץ** (长夏) | אדמה | טחול | חיזוק עיכול, ייבוש לחות | מותרת |
| **סתיו** (秋) | מתכת | ריאות | הזנת ין, לחות ריאות | מותרת |
| **חורף** (冬) | מים | כליות | חיזוק כליות, שימור יאנג | מומלצת מאוד |

### 3.2 נקודות עונתיות

- **אביב**: LV3, GB34, LV14 — שחרור כבד
- **קיץ**: HT7, PC6, CV17 — הרגעת לב
- **סוף קיץ**: ST36, SP6, CV12 — חיזוק טחול
- **סתיו**: LU7, LU9, KI6 — הזנת ריאות
- **חורף**: KI3, BL23, GV4 + מוקסה — חיזוק כליות

**פרוטוקול אינטגרטיבי:**

- **אוזן**: Shen Men, Sympathetic, Kidney, Liver — זרעים להשארה

- **כוסות**: גב עליון BL12-BL17 (שחרור שרירים)
- **תדירות**: 2×/שבוע חודש ראשון → 1×/שבוע

## 9. עקרונות מתקדמים

### 9.1 "מעט הוא הרבה"

עם הניסיון, המטפל לומד שלא צריך להשתמש בכל הכלים בכל טיפול:
- **מטפל מתחיל** — נוטה להשתמש ב-15-20 מחטים + מוקסה + כוסות
- **מטפל מנוסה** — בוחר 5-8 מחטים בדיוק הנכון
- **מאסטר** — לעתים מחט אחת במקום הנכון מספיקה

### 9.2 רגישות למטופל

- **שאלו** — "איך היה הטיפול האחרון? מה עזר?" — התאימו
- **הקשיבו לגוף** — הדופק, הלשון, הפנים מספרים מה צריך היום
- **גמישות** — אל תהיו כבולים לפרוטוקול; כל טיפול הוא ייחודי
- **אינטואיציה** — עם הניסיון, תדעו מה נכון עוד לפני שתנתחו

## 10. שאלות לחזרה

1. תאר את "5 השאלות" לקבלת החלטה על שילוב מודאליות.
2. אילו שילובים נפוצים ויעילים? אילו שילובים יש להימנע מהם?
3. כיצד מתאימים טיפול לעונה? תן דוגמה לחורף ולקיץ.
4. בחר אחד ממקרי הבוחן וצור פרוטוקול אינטגרטיבי חלופי.
5. כיצד מחליטים מתי להוסיף מוקסה ומתי כוסות רוח?

---

## 11. תרשים החלטה מהיר

### 11.1 עץ החלטות

```
מטופל חדש
  └── חריף? ← כן → שורש כף/קרסול (WAA) + נקודות מקומיות
  └── כרוני? ← כן → אבחון מלא
        └── חסר? → מוקסה + דיקור טוניפיקציה
        └── עודף? → כוסות רוח + דיקור סדציה
        └── סטגנציה? → הזזת צ'י/דם + כוסות
        └── נוירולוגי? → קרקפת + גוף
        └── רגשי? → אוזן + נקודות רוח
```
