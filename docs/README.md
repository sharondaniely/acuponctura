## מה כולל הקורס

- **תיאוריה מקיפה** — פילוסופיה, פתולוגיה, אבחון, וסינדרומים
- **~380 איורי נקודות דיקור** — תקריב אנטומי עם הוראות מיקום לכל נקודה
- **14 מפות מרידיאנים** — סקירת מסלול מלא לכל ערוץ
- **29 תיאורי מקרה** — ב-6 תחומים קליניים
- **כלי אבחון אינטראקטיבי** — מהסימפטום לנקודת הדיקור
- **פורמולות צמחים** — שילוב דיקור וצמחי מרפא
- **תרגילים ודפי עבודה** — לתרגול ולהעמקה
