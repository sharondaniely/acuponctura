
# שמונת העקרונות — בא גאנג

## Eight Principles — Ba Gang

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לסווג כל מצב קליני לפי ארבעת זוגות הניגודים
2. לתאר את הסימפטומים, הלשון והדופק של כל עיקרון
3. לזהות דפוסים מורכבים (כגון "קור אמיתי-חום מדומה")
4. להבין כיצד שמונת העקרונות מהווים בסיס לכל מערכות האבחון

---

## מבוא

**שמונת העקרונות** (Ba Gang) הם **הבסיס של כל אבחון ברפואה הסינית**. כל מצב פתולוגי ניתן לסיווג לפי ארבעה זוגות של ניגודים:

1. **פנים / חוץ** (/ Li / Biao) — היכן המחלה?
2. **קור / חום** (/ Han / Re) — מה טבע המחלה?
3. **חסר / עודף** (/ Xu / Shi) — מה מצב הגוף?
4. **ין / יאנג** (/ Yin / Yang) — הסיווג הכולל

> (Shan Zhen Zhe, Cha Se An Mai, Xian Bie Yin Yang)
> "המאבחן המיומן בוחן צבע ודופק, וקודם כל מבדיל בין ין ליאנג"
> — הואנג די ניי ג'ינג

---

## 1. פנים וחוץ (/ Li / Biao)

### 1.1 דפוס חוץ (Biao Zheng)

מחלה ברמה **שטחית** — בעור, בשרירים, בערוצים. בדרך כלל **חריפה** ונגרמת על ידי גורם חיצוני (רוח, קור, חום, לחות).

| סימפטום | ביטוי |
|---------|-------|
| תחילה | פתאומית, חריפה |
| צמרמורת/חום | צמרמורת עם/בלי חום |
| כאבים | כאבי גוף, כאב ראש |
| אף/גרון | גודש, נזלת, כאב גרון |
| הזעה | יש/אין (תלוי בדפוס) |
| **לשון** | ציפוי דק לבן |
| **דופק** | שטחי/צף (Fu) |

**עיקרון טיפולי:** שחרר את השטח (Jie Biao) — הוצא את הפתוגן החוצה

### 1.2 דפוס פנים (Li Zheng)

מחלה ב**איברים הפנימיים**. יכולה להיות כרונית או חריפה שחדרה פנימה.

| סימפטום | ביטוי |
|---------|-------|
| תחילה | הדרגתית, או חדירה משטח |
| אין סימני שטח | אין צמרמורת/גודש |
| סימנים פנימיים | בעיות עיכול, שינה, שתן, מחזור |
| **לשון** | שינויים בגוף הלשון עצמו |
| **דופק** | עמוק (Chen) |

**עיקרון טיפולי:** תלוי בטבע (קור/חום) ובמצב (חסר/עודף)

### 1.3 חצי-חוץ חצי-פנים (Ban Biao Ban Li)

מצב ביניים — הפתוגן לא בשטח ולא בעומק. זהו **שלב שאו יאנג**:
- חום וצמרמורת **לסירוגין**
- טעם מר בפה, חוסר תיאבון
- כאב בצדדי החזה
- דופק: דמוי מיתר (Xian)
- **טיפול:** הרמוניזציה (He Jie) — לא לשחרר ולא לנקות

## 2. קור וחום (/ Han / Re)

### 2.1 דפוס קור (Han Zheng)

| סימפטום | ביטוי |
|---------|-------|
| תחושה | קור, העדפת חום |
| פנים | חיוורות |
| שתייה | העדפה למשקאות חמים |
| שתן | שקוף ומרובה |
| צואה | רפויה או שלשול |
| הפרשות | שקופות, מימיות |
| **לשון** | חיוורת, ציפוי לבן לח |
| **דופק** | איטי (Chi) או מתוח (Jin) |

**עיקרון טיפולי:** חמם (Wen) — הפג קור

### 2.2 דפוס חום (Re Zheng)

| סימפטום | ביטוי |
|---------|-------|
| תחושה | חום, העדפת קור |
| פנים | אדומות |
| שתייה | צמא למשקאות קרים |
| שתן | כהה ומועט |
| צואה | יבשה, עצירות |
| הפרשות | צהובות, עבות |
| **לשון** | אדומה, ציפוי צהוב |
| **דופק** | מהיר (Shuo) |

**עיקרון טיפולי:** קרר (Qing) — נקה חום

---

## 3. חסר ועודף (/ Xu / Shi)

### 3.1 דפוס חסר (Xu Zheng)

**צ'י הגוף חלש** — חסר בצ'י, דם, ין או יאנג.

| סימפטום | ביטוי |
|---------|-------|
| כללי | עייפות כרונית, חולשה |
| התפתחות | הדרגתית, ממושכת |
| קול | חלש, שקט |
| נשימה | קוצר נשימה, במיוחד במאמץ |
| בטן | כאב שמשתפר בלחיצה |
| הזעה | ספונטנית (חסר צ'י) או לילית (חסר ין) |
| **לשון** | חיוורת (חסר צ'י/יאנג) או אדומה ללא ציפוי (חסר ין) |
| **דופק** | חלש (Ruo) או ריק (Xu) |

**עיקרון טיפולי:** חזק/הזן (Bu) — בנה את מה שחסר

### 3.2 דפוס עודף (Shi Zheng)

**גורם פתוגני חזק** — הגוף עדיין חזק אך הפתוגן נוכח.

| סימפטום | ביטוי |
|---------|-------|
| כללי | כוח שמור, תגובה חזקה |
| התפתחות | חריפה, פתאומית |
| קול | חזק, רם |
| נשימה | גסה, שיעול חזק |
| בטן | כאב שמחמיר בלחיצה |
| הפרשות | מרובות, עבות |
| **לשון** | ציפוי עבה |
| **דופק** | מלא ועוצמתי (Shi) |

**עיקרון טיפולי:** נקה/פזר (Xie) — הסר את העודף

### 3.3 חסר-עודף מעורב

מצב נפוץ מאוד בפרקטיקה:
- **שורש חסר + ענף עודף:** למשל, חסר טחול (שורש) → הצטברות ליחה (עודף). טפל בשניהם
- **עיקרון:** זהה מה עיקרי ומה משני, וטפל בהתאם

## 4. ין ויאנג (/ Yin / Yang)

ין ויאנג הם **הסיווג הכללי ביותר** — כל שאר העקרונות נכללים בהם:

| ין | יאנג |
|---------|----------|
| פנים | חוץ |
| קור | חום |
| חסר | עודף |

### 4.1 דפוסי ין ויאנג כלליים

**קריסת יאנג (Wang Yang):**
- מצב חירום — יאנג קורס
- הזעה קרה, גפיים קפואות, פנים חיוורות, דופק כמעט נעלם
- טיפול: חמם יאנג בדחיפות — מוקסה על REN4, REN8; צמחים: שן פו טאנג

**קריסת ין (Wang Yin):**
- מצב חירום — ין קורס
- הזעה חמה ודביקה, צמא חמור, לשון אדומה ויבשה, דופק מהיר ודק
- טיפול: הזן ין בדחיפות

---

## 5. דפוסים מורכבים

### 5.1 קור אמיתי — חום מדומה

**המציאות:** הגוף קר מבפנים (חסר יאנג חמור)
**המראה:** סומק בפנים, אי-שקט, צמא — **אך** הפנים סמוקות רק בלחיים, הצמא ללא רצון לשתות, רוצה לשכב מכוסה

**מפתח לאבחנה:**
- דופק: למראית עין מהיר, אך **ריק** (חסר כוח)
- לשון: חיוורת למרות הסומק בפנים
- גפיים: קרות

### 5.2 חום אמיתי — קור מדומה

**המציאות:** חום חזק בפנים (עודף חום)
**המראה:** גפיים קרות — **אך** חום בחזה ובבטן, צמא למשקאות קרים, שתן כהה, עצירות

**מפתח לאבחנה:**
- דופק: מהיר ועוצמתי בעומק
- לשון: אדומה עם ציפוי צהוב
- **"ארבעת הגפיים קרות אך הגוף חם"** — חום שחוסם את הצ'י מלהגיע לגפיים

### 5.3 עיקרון האבחנה בדפוסים מורכבים

> **"כאשר יש סתירה — הלשון והדופק מספרים את האמת"**

- **הלשון** משקפת את המצב ה**כרוני** / האמיתי
- **הדופק** משקף את המצב ה**נוכחי**
- כאשר סימפטומים סותרים → בדוק לשון ודופק לעומק

## 6. טבלת סיכום — שמונת העקרונות

| עיקרון | לשון | דופק | סימפטום מנחה |
|--------|------|------|-------------|
| חוץ | ציפוי דק | שטחי | צמרמורת, כאבי גוף |
| פנים | שינוי בגוף הלשון | עמוק | סימנים פנימיים |
| קור | חיוורת, ציפוי לבן | איטי | קור, העדפת חום |
| חום | אדומה, ציפוי צהוב | מהיר | חום, העדפת קור |
| חסר | חיוורת/אדומה ללא ציפוי | חלש | עייפות, חולשה |
| עודף | ציפוי עבה | מלא | כוח שמור, סימנים חזקים |
| ין | חיוורת/רטובה | עמוק, איטי | פנים+קור+חסר |
| יאנג | אדומה/יבשה | שטחי, מהיר | חוץ+חום+עודף |

## 7. זרימת אבחון מעשית

```
שלב 1: פנים או חוץ?
  ├── חוץ → האם קור-חוץ או חום-חוץ?
  │ → שחרר שטח (בשיטה המתאימה)
  │
  └── פנים → המשך לשלב 2

שלב 2: קור או חום?
  ├── קור → חמם
  └── חום → קרר

שלב 3: חסר או עודף?
  ├── חסר → חזק (מה חסר? צ'י? דם? ין? יאנג?)
  └── עודף → נקה/פזר (מה עודף? ליחה? קיפאון? חום?)

שלב 4: ין או יאנג?
  → סיווג כולל → מנחה את כיוון הטיפול הכללי
```

---

## נקודות חשובות לזכירה

1. **שמונת העקרונות הם תמיד הצעד הראשון** בכל אבחון ברפואה סינית
2. **לשון ודופק** הם "עדי המלך" — כאשר יש סתירה, הם קובעים
3. **דפוסים מורכבים שכיחים** — "קור אמיתי חום מדומה" הוא מלכודת נפוצה
4. **חסר-עודף מעורב** נפוץ מאוד — חפש תמיד את שני הצדדים
5. **הסיווג מנחה את הטיפול** — קור = חמם, חום = קרר, חסר = חזק, עודף = נקה
6. **ין/יאנג** מסכמים הכל — פנים+קור+חסר = ין; חוץ+חום+עודף = יאנג
