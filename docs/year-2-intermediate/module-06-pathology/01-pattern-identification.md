
# זיהוי דפוסים — ביאן ג'נג

## Pattern Identification — Bian Zheng

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. להסביר מהו "דפוס" (Zheng) ובמה הוא שונה מ"מחלה" ברפואה המערבית
2. למנות את מערכות זיהוי הדפוסים השונות ברפואה הסינית
3. להבין מתי להשתמש בכל מערכת ומתי לשלב ביניהן
4. לתאר את הזרימה מסימפטום → דפוס → עיקרון טיפולי → טיפול

---

## מבוא — מהו זיהוי דפוסים?

**ביאן ג'נג לון ג'י** (Bian Zheng Lun Zhi) — "זיהוי דפוס וקביעת טיפול" — זהו **הליבה של הרפואה הסינית**. בניגוד לרפואה המערבית שמאבחנת "מחלה" (disease), הרפואה הסינית מזהה "דפוס" (pattern).

### ההבדל בין מחלה לדפוס

| | מחלה (Bing) | דפוס (Zheng) |
|--|---------------|----------------|
| מהות | ישות פתולוגית מוגדרת | תמונה קלינית רגעית |
| דוגמה | כאב ראש, אסתמה, יתר לחץ דם | עליית יאנג כבד, חסר ין כליות |
| יציבות | שם המחלה נשאר קבוע | הדפוס משתנה לאורך הזמן |
| טיפול | תרופה למחלה | טיפול מותאם לדפוס |

> **"אותה מחלה — דפוסים שונים; דפוסים שונים — טיפולים שונים"**
> (Tong Bing Yi Zhi)

**דוגמה:** עשרה מטופלים עם כאב ראש — ייתכנו עשרה דפוסים שונים, ולכן עשרה טיפולים שונים.

> **"מחלות שונות — דפוס אחד; דפוס אחד — טיפול אחד"**
> (Yi Bing Tong Zhi)

**דוגמה:** צניחת רחם, טחורים ושלשול כרוני — כולם יכולים לנבוע מ"שקיעת צ'י הטחול" ולקבל טיפול דומה.

## מערכות זיהוי דפוסים

הרפואה הסינית פיתחה מספר מערכות לזיהוי דפוסים. כל מערכת מתאימה למצבים קליניים שונים:

### סקירה כללית

| מערכת | שימוש עיקרי | פרק |
|-------|-------------|------|
| שמונת העקרונות | בסיס לכל אבחון — מסגרת כללית | 02 |
| דפוסי צ'י-דם-ין-יאנג | חסר/עודף של חומרים חיוניים | 03 |
| דפוסי זאנג-פו | מחלות פנימיות כרוניות | 04 |
| שש השלבים | מחלות חום חיצוניות (קור) | 05 |
| ארבע הרמות | מחלות חום-לחות (חום) | 06 |
| דפוסי סאן ג'יאו | מחלות חום-לחות (גישה נוספת) | 07 |
| דפוסי ערוצים | כאב, בעיות מקומיות, דיקור | 08 |

## 1. שמונת העקרונות

**הבסיס של כל אבחון.** ארבעה זוגות של ניגודים שמסווגים כל מצב:
- **ין / יאנג** — הסיווג הכללי ביותר
- **פנים / חוץ** (/ Li/Biao) — מיקום המחלה
- **קור / חום** (/ Han/Re) — טבע המחלה
- **חסר / עודף** (/ Xu/Shi) — מצב הגוף מול הפתוגן

→ **פירוט מלא בפרק 02**

---

## 2. דפוסי צ'י, דם, ין, יאנג ונוזלי גוף

מתמקדת ב**חומרים חיוניים** — האם יש חסר, עודף, קיפאון או שינוי כיוון:
- צ'י: חסר, קיפאון, שקיעה, מרידה
- דם: חסר, קיפאון, חום, קור
- ין: חסר → חום דמוי
- יאנג: חסר → קור דמוי
- נוזלים: חסר, ליחה, לחות, בצקת

→ **פירוט מלא בפרק 03**

---

## 3. דפוסי זאנג-פו

**המערכת המשומשת ביותר** במרפאה לטיפול במחלות פנימיות כרוניות. מזהה איזה איבר (או שילוב איברים) מעורב ומהו הדפוס הספציפי.

**דוגמאות:**
- חסר צ'י הטחול
- עליית יאנג הכבד
- אי-תקשורת לב-כליות

→ **סקירה כללית בפרק 04; פירוט מלא במודול 06ב — קומפנדיום סינדרומים**

---

## 4. שש השלבים

מערכת של **ג'אנג ג'ונג ג'ינג** מתוך ה**שאנג האן לון**. מתארת שישה שלבים של חדירת מחלת קור חיצונית:

```
טאי יאנג → יאנג מינג → שאו יאנג → טאי ין → שאו ין → ג'ואה ין
(שטחי) (עמוק)
(יאנג) (ין)
```

→ **פירוט מלא בפרק 05**

---

## 5. ארבע הרמות

מערכת של **יה טיאן שי** למחלות **חום-לחות** (Wen Bing). מתארת ארבע רמות חדירה:

```
ווי → צ'י → יינג → שיואה
(שטחי) (עמוק)
(קל) (חמור)
```

→ **פירוט מלא בפרק 06**

---

## 6. דפוסי סאן ג'יאו

מערכת של **וו ג'ו טונג** — גישה נוספת למחלות חום, מחלקת לפי **מיקום** (עליון, אמצעי, תחתון):

```
מבערה עליונה → מבערה אמצעית → מבערה תחתונה
   לב, ריאות טחול, קיבה כבד, כליות
```

→ **פירוט מלא בפרק 07**

---

## 7. דפוסי ערוצים

מתמקדת בזיהוי **באיזה ערוץ (מרידיאן)** נמצאת הבעיה, לפי:
- מיקום הכאב או התסמינים
- נקודות רגישות לאורך הערוץ
- קשר לאיבר הזאנג-פו של הערוץ

→ **פירוט מלא בפרק 08**

## כיצד לשלב את המערכות?

### תהליך האבחון המעשי

```
1. התחל בשמונת העקרונות
   → סווג: פנים/חוץ, קור/חום, חסר/עודף, ין/יאנג

2. קבע את סוג הבעיה:
   ├── מחלה חיצונית חריפה?
   │ ├── מחלת קור → שש השלבים
   │ └── מחלת חום → ארבע הרמות או סאן ג'יאו
   │
   ├── מחלה פנימית כרונית?
   │ ├── זהה את האיבר → דפוסי זאנג-פו
   │ └── זהה חומר חיוני פגוע → דפוסי צ'י-דם
   │
   └── כאב/בעיה מקומית?
       └── זהה ערוץ → דפוסי ערוצים

3. שלב בין המערכות לתמונה מלאה
   → דוגמה: חסר ין כליות (זאנג-פו) + חסר ין (צ'י-דם) + פנים-חום-חסר (שמונה עקרונות)
```

### דוגמה קלינית משולבת

**מטופלת, 45, מתלוננת על:** כאבי ראש, סחרחורת, טנטון, עצבנות, כאבי גב, הזעות לילה

| מערכת | אבחנה |
|-------|-------|
| שמונת העקרונות | פנים, חום (דמוי), חסר, ין |
| צ'י-דם | חסר ין |
| זאנג-פו | חסר ין כליות + עליית יאנג כבד |
| חמשת היסודות | מים לא מזינים עץ |

**עיקרון טיפולי:** הזן ין כליות, הורד יאנג כבד

## עקרונות יסוד בזיהוי דפוסים

### 1. עיקרון ה"שורש והענף" (Biao Ben)

- **שורש (Ben):** הסיבה העמוקה — האיבר הפגוע, החסר המקורי
- **ענף (Biao):** הביטוי השטחי — הסימפטומים
- **כלל:** טפל בשורש במחלה כרונית; טפל בענף במצב חריף; טפל בשניהם כשאפשר

### 2. עיקרון ה"ראשי והמשני" (Zhu Ci)

- **זהה את הדפוס העיקרי** (Zhu Zheng) ואת הדפוסים המשניים
- **טפל בעיקרי קודם**, אלא אם המשני דחוף יותר

### 3. מחלה משתנה — דפוס משתנה

- דפוסים **אינם קבועים** — הם משתנים לאורך המחלה
- **חובה לבדוק מחדש** בכל ביקור — הדפוס שהיה בביקור הקודם עשוי להשתנות
- מחלה יכולה להתחיל כעודף-חוץ ולהפוך לחסר-פנים

## נקודות חשובות לזכירה

1. **ביאן ג'נג הוא הלב של הרפואה הסינית** — ללא זיהוי דפוס נכון, הטיפול לא ממוקד
2. **שמונת העקרונות הם תמיד הצעד הראשון** — גם כשמשתמשים במערכת אחרת
3. **אותה מחלה = דפוסים שונים** — לכן אין "מתכון" אחד לכאב ראש או לנדודי שינה
4. **דפוסים משתנים** — האבחון חייב להתעדכן בכל ביקור
5. **שילוב מערכות** הוא המפתח לאבחון מקיף ומדויק
