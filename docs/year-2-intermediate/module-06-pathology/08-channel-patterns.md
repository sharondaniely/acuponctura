## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לתאר את ההבדל בין פתולוגיית ערוץ לפתולוגיית איבר (זאנג-פו)
2. לזהות דפוסים פתולוגיים לאורך כל אחד מ-12 הערוצים העיקריים
3. להבדיל בין דפוסי עודף וחסר בערוצים
4. להבין מתי להשתמש באבחון ערוצים ומתי באבחון זאנג-פו

## מבוא

**אבחון ערוצים** מתמקד בזיהוי הערוץ (מרידיאן) שבו נמצאת הבעיה, להבדיל מזיהוי האיבר הפנימי. זוהי מערכת חשובה במיוחד ל**דיקור** — כי בחירת הנקודות מבוססת רבות על הערוץ הפגוע.

### מתי משתמשים באבחון ערוצים?

| מצב | מערכת מתאימה |
|-----|-------------|
| כאב לאורך מסלול ספציפי | **ערוצים** |
| בעיה מקומית (כאב כתף, ברך, גב) | **ערוצים** |
| בעיות שרירים, גידים, מפרקים | **ערוצים** |
| מחלה פנימית כרונית (עיכול, שינה) | **זאנג-פו** |
| הפרעות רגשיות | **זאנג-פו** |
| כאב + סימנים פנימיים | **שילוב שניהם** |

### ערוץ מול איבר

- **פתולוגיית ערוץ:** כאב, נפיחות, חוסר תחושה, חולשה **לאורך מסלול הערוץ**
- **פתולוגיית איבר:** הפרעות בתפקוד האיבר הפנימי (עיכול, נשימה, שינה, רגשות)
- **שניהם יכולים להופיע יחד:** למשל, כאב בצד הראש (ערוץ כיס מרה) + טעם מר בפה (כיס מרה כאיבר)

## עקרונות כלליים — עודף וחסר בערוצים

### עודף בערוץ

- כאב, נפיחות, אדמומיות, חום מקומי
- תחושת מלאות או לחץ
- כאב שמחמיר בלחיצה

### חסר בערוץ

- חולשה, כבדות, חוסר תחושה לאורך הערוץ
- קור מקומי
- כאב עמום שמשתפר בלחיצה

## 12 הערוצים העיקריים — פתולוגיות

### 1. ערוץ הריאות — טאי ין של היד

**מסלול:** מהחזה → זרוע פנימית (צד אגודל) → אגודל

| עודף | חסר |
|------|-----|
| שיעול חזק, כיח מרובה | שיעול חלש, קוצר נשימה |
| כאב ומלאות בחזה | קול חלש, נשימה שטחית |
| כאב בכתף הקדמית ובזרוע (צד רדיאלי) | חולשה בזרוע |
| חום, צמא | עור יבש, חיסון חלש |

**נקודות מפתח:** LU1 ג'ונג פו, LU5 צ'י צה, LU7 ליה צ'ואה, LU9 טאי יואן

### 2. ערוץ המעי הגס — יאנג מינג של היד

**מסלול:** אצבע מורה → זרוע חיצונית → כתף → פנים → אף

| עודף | חסר |
|------|-----|
| כאב שיניים, כאב גרון | שיניים רגישות, חניכיים רכות |
| דימום אף, אף סתום | נזלת שקופה, ריח אף |
| כאב בכתף ובזרוע (צד רדיאלי-חיצוני) | חולשה בזרוע |
| עצירות, חום | שלשול, קור בבטן |

**נקודות מפתח:** LI4 חה גו, LI11 צ'ו צ'י, LI15 ג'יאן יו, LI20 יינג שיאנג

### 3. ערוץ הקיבה — יאנג מינג של הרגל

**מסלול:** מתחת לעין → פנים → צוואר → חזה קדמי → בטן → רגל קדמית → אצבע 2

| עודף | חסר |
|------|-----|
| כאבי פנים, שיניים, חניכיים | תיאבון ירוד, נפיחות |
| רעב מוגזם, ריח פה | עייפות, שרירים חלשים |
| כאב בחזית הברך, בשוקה | קור בבטן, ברכיים חלשות |
| חום, הזעה, פנים אדומות | חיוורון, רזון |

**נקודות מפתח:** ST36 צו סאן לי, ST25 טיאן שו, ST44 ניי טינג, ST40 פנג לונג

### 4. ערוץ הטחול — טאי ין של הרגל

**מסלול:** בוהן → רגל פנימית → מפשעה → בטן → חזה צדדי

| עודף | חסר |
|------|-----|
| נפיחות בטנית, כאב בטן | תיאבון ירוד, שלשול |
| כבדות בגפיים | חולשה בשרירים, עייפות |
| הפרשות מרובות, לחות | שרירים רפויים, צניחה |
| בחילה, הקאה | חבורות, דימום |

**נקודות מפתח:** SP3 טאי באי, SP6 סאן ין ג'יאו, SP9 ין לינג צ'ואן, SP10 שיואה האי

### 5. ערוץ הלב — שאו ין של היד

**מסלול:** לב → בית השחי → זרוע פנימית (צד אולנרי) → זרת

| עודף | חסר |
|------|-----|
| כאב בחזה, דפיקות | דפיקות קלות, חולשה |
| כאב לאורך זרוע פנימית | חוסר תחושה בזרת וזרוע |
| פנים אדומות, צמא | חיוורון, תת-לחץ דם |
| אי-שקט, שיגעון | חרדה, נדודי שינה |

**נקודות מפתח:** HT3 שאו האי, HT5 טונג לי, HT7 שן מן, HT9 שאו צ'ונג

### 6. ערוץ המעי הדק — טאי יאנג של היד

**מסלול:** זרת → זרוע אחורית-חיצונית → כתף אחורית → צוואר → פנים → אוזן

| עודף | חסר |
|------|-----|
| כאב גרון, לסת, אוזן | חוסר שמיעה, אוזן חסומה |
| כאב בכתף אחורית | חולשה בכתף ובצוואר |
| שתן כהה וצורב | שתן תכוף ושקוף |

**נקודות מפתח:** SI3 הואו שי, SI5 יאנג גו, SI9 ג'יאן ג'ן, SI19 טינג גונג

### 7. ערוץ שלפוחית השתן — טאי יאנג של הרגל

**מסלול:** עין פנימית → ראש → עורף → **כל הגב** → ירך אחורית → שוקה → אצבע קטנה ברגל

| עודף | חסר |
|------|-----|
| כאב ראש אחורי, כאב עורף | כבדות בראש, סחרחורת |
| כאבי גב לאורך עמוד השדרה | חולשה בגב, כאבי גב עמומים |
| כאב ישיאטי, כאב ירך אחורית | חולשה ברגליים |
| צמרמורת, קשיחות בעורף | שתן תכוף |

**הערוץ הארוך ביותר בגוף** — חשוב במיוחד לכאבי גב ולכאבי ראש אחוריים.

**נקודות מפתח:** BL2 צואן ג'ו, BL10 טיאן ג'ו, BL23 שן שו, BL40 ווי ג'ונג, BL60 קון לון

### 8. ערוץ הכליות — שאו ין של הרגל

**מסלול:** כף רגל → קרסול פנימי → רגל פנימית → בטן → חזה

| עודף | חסר |
|------|-----|
| רגל חמה ויבשה | כפות רגליים קרות |
| כאב בגרון, יובש בפה | כאבי גב תחתון, חולשה בברכיים |
| שתן כהה | שתן תכוף, הרטבה |
| — | טנטון, ירידה בשמיעה, שיבה |

**נקודות מפתח:** KI1 יונג צ'ואן, KI3 טאי שי, KI6 ג'או האי, KI7 פו ליו

### 9. ערוץ הפריקרד — ג'ואה ין של היד

**מסלול:** חזה → בית שחי → זרוע פנימית (מרכז) → כף יד → אצבע אמצעית

| עודף | חסר |
|------|-----|
| כאב ולחץ בחזה | דפיקות חלשות |
| כף יד חמה | כפות ידיים קרות |
| אי-שקט, צחוק ללא סיבה | חרדה, ביישנות |

**נקודות מפתח:** PC3 צ'ו צה, PC6 ניי גואן, PC7 דא לינג, PC8 לאו גונג

### 10. ערוץ סאן ג'יאו — שאו יאנג של היד

**מסלול:** אצבע טבעת → זרוע חיצונית → כתף → צד צוואר → מאחורי אוזן → גבה

| עודף | חסר |
|------|-----|
| כאב אוזן, טנטון | ירידה בשמיעה |
| כאב בצד הראש ובזרוע חיצונית | חולשה בזרוע |
| נפיחות בצוואר | עייפות כללית |

**נקודות מפתח:** SJ3 ג'ונג ג'ו, SJ5 וואי גואן, SJ6 ג'י גו, SJ17 יי פנג

### 11. ערוץ כיס המרה — שאו יאנג של הרגל

**מסלול:** עין חיצונית → צד הראש (זיגזג) → כתף → צד הגוף → ירך צדדית → שוקה → אצבע 4

| עודף | חסר |
|------|-----|
| **כאב ראש צדדי / מיגרנה** | סחרחורת קלה |
| כאב בצדדי החזה | חוסר החלטיות |
| טעם מר בפה | ראייה מטושטשת |
| כאב בירך צדדית, ברך צדדית | חולשה בצד הגוף |

**ערוץ חשוב מאוד לכאבי ראש צדדיים (מיגרנה) ולכאבי ירך.**

**נקודות מפתח:** GB20 פנג צ'י, GB21 ג'יאן ג'ינג, GB34 יאנג לינג צ'ואן, GB41 צו לין צ'י

### 12. ערוץ הכבד — ג'ואה ין של הרגל

**מסלול:** בוהן → רגל פנימית → מפשעה → גניטליה → צדדי בטן → צלעות

| עודף | חסר |
|------|-----|
| כאב בצדדי הבטן, צלעות | ראייה מטושטשת |
| כאב במפשעה, גניטליה | סחרחורת |
| עצבנות, כעס | דיכאון, אנחות |
| עיניים אדומות | עיניים יבשות |

**נקודות מפתח:** LR2 שינג ג'יאן, LR3 טאי צ'ונג, LR8 צ'ו צ'ואן, LR14 צ'י מן

## כיצד לזהות את הערוץ הפגוע?

### שלוש שיטות עיקריות

1. **לפי מיקום הכאב/תסמין:**
   - כאב ראש אחורי → שלפוחית (טאי יאנג)
   - כאב ראש צדדי → כיס מרה (שאו יאנג)
   - כאב ראש מצחי → קיבה (יאנג מינג)
   - כאב ראש קודקודי → כבד (ג'ואה ין)

2. **לפי מישוש — נקודות רגישות:**
   - אה-שי — נקודות כואבות לאורך ערוץ ספציפי
   - נקודות שו-גב — רגישות בגב מצביעה על האיבר/ערוץ

3. **לפי סימנים נלווים:**
   - כאב כתף + שיעול → ערוץ ריאות
   - כאב כתף + בעיות עיכול → ערוץ מעי גס
   - כאב כתף + בעיית צוואר/אוזן → ערוץ מעי דק

## נקודות חשובות לזכירה

1. **אבחון ערוצים** הוא חיוני לדיקור — הוא מנחה את בחירת הנקודות
2. **מיקום הכאב** הוא הרמז העיקרי — עקוב אחרי מסלול הערוץ
3. **ערוץ ≠ איבר** — כאב לאורך ערוץ הקיבה לא בהכרח אומר בעיה בקיבה
4. **שלב ערוצים + זאנג-פו** לאבחון מלא — למשל, מיגרנה (ערוץ כיס מרה) + טעם מר (כיס מרה כאיבר) = לחות-חום בכבד-כיס מרה
5. **שש שכבות** של ערוצים (טאי יאנג, שאו יאנג, יאנג מינג, טאי ין, שאו ין, ג'ואה ין) חוזרות גם בשאנג האן לון — אותם שמות, הקשר שונה
6. **כאב ראש** הוא הדוגמה הקלאסית לאבחון ערוצים — מיקום הכאב קובע את הערוץ
