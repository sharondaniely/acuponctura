## סדר לימוד מומלץ

1. התחל ב**שמונת העקרונות** (02) — הבסיס לכל זיהוי דפוס
2. עבור ל**דפוסי צ'י ודם** (03) — ברמת החומרים החיוניים
3. למד **דפוסי זאנג-פו** (04) — ברמת האיברים
4. הרחב עם **שש השלבים** (05), **ארבע הרמות** (06), ו**שלוש המבערות** (07)
5. סיים ב**פתולוגיות ערוצים** (08)
