
# איברי זאנג — האיברים המלאים

## Zang Organs — The Solid (Yin) Organs

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לתאר את חמשת איברי הזאנג ותפקודיהם ברפואה הסינית
2. להבין את הרקמות, איברי החישה, הרגשות והנוזלים המשויכים לכל איבר
3. לזהות פתולוגיות נפוצות של כל איבר זאנג
4. להסביר את ההבדל המהותי בין איברי זאנג לאיברי פו

---

## מבוא

חמשת איברי הזאנג (Wu Zang) הם הלב, הכבד, הטחול, הריאות והכליות. בניגוד לאנטומיה המערבית, ברפואה הסינית כל "איבר" מייצג **מערכת פונקציונלית שלמה** הכוללת תפקודים פיזיים, רגשיים ורוחניים.

איברי הזאנג הם **איברים ין** — מלאים, מאחסנים את החומרים החיוניים (צ'י, דם, ג'ינג, שן), ואינם מפרישים. הם עומדים במרכז הפיזיולוגיה הסינית.

> (Wu Zang Zhe, Cang Jing Qi Er Bu Xie Ye)
> "חמשת איברי הזאנג מאחסנים תמציות וצ'י ואינם מפרישים"
> — הואנג די ניי ג'ינג

## 1. הלב (Xin)

הלב נחשב ל**"קיסר של כל האיברים"** (Jun Zhu Zhi Guan). הוא מכוון את כל שאר האיברים ומהווה את מרכז התודעה.

### 1.1 תפקודים עיקריים

| תפקוד | הסבר |
|--------|-------|
| שולט בדם ובכלי הדם (Zhu Xue Mai) | הלב מניע את הדם בכלי הדם; תנועה, קצב וחיוּת הדם תלויים בצ'י הלב |
| מאכלס את השן (Cang Shen) | השן — התודעה, המחשבה, הזיכרון והשינה — כולם שוכנים בלב |
| נפתח ללשון (Kai Qiao Yu She) | מצב הלב משתקף בלשון — צבעה, צורתה ויכולת הדיבור |
| מתבטא במראה הפנים (Qi Hua Zai Mian) | פנים ורודות ובוהקות = לב בריא; חיוורון או כחלון = בעיה בלב |

### 1.2 שיוכים

- **רגש:** שמחה (Xi) — שמחה מוגזמת מפזרת את צ'י הלב
- **נוזל:** זיעה (Han) — "הזיעה היא נוזל הלב"
- **רקמה:** כלי הדם (Mai)
- **יסוד:** אש (Huo)
- **עונה:** קיץ
- **צבע:** אדום
- **טעם:** מר

### 1.3 פתולוגיות נפוצות

- **חסר צ'י הלב:** דפיקות, קוצר נשימה, עייפות, הזעה ספונטנית
- **חסר דם הלב:** חרדה, נדודי שינה, חלומות מרובים, חיוורון, לשון חיוורת
- **חסר ין הלב:** חום בחמש, הזעות לילה, אי-שקט, לשון אדומה ללא ציפוי
- **אש הלב בוערת:** כיבים בפה, אי-שקט חמור, שתן כהה, לשון אדומה עם קצה אדום בוהק
- **קיפאון דם הלב:** כאבי חזה דוקרים, שפתיים סגולות, לשון כהה עם כתמים

## 2. הכבד (Gan)

הכבד נקרא **"הגנרל"** (Jiang Jun Zhi Guan) — אחראי על תכנון אסטרטגי ועל זרימה חופשית.

### 2.1 תפקודים עיקריים

| תפקוד | הסבר |
|--------|-------|
| מבטיח זרימה חלקה של צ'י (Shu Xie) | התפקוד החשוב ביותר — כאשר הזרימה חלקה, הרגשות מאוזנים, העיכול תקין והמחזור סדיר |
| מאחסן דם (Cang Xue) | מווסת את כמות הדם לפי הצורך — בפעילות: דם לשרירים; במנוחה: דם חוזר לכבד |
| שולט בגידים (Zhu Jin) | הגמישות, התנועתיות וכוח השרירים תלויים בהזנת הגידים ע"י דם הכבד |
| נפתח לעיניים (Kai Qiao Yu Mu) | הראייה תלויה בהזנת העיניים ע"י דם הכבד |
| מאכלס את ההון (Cang Hun) | ההון — הנשמה האתרית, הדמיון, החזון והחלומות |

### 2.2 שיוכים

- **רגש:** כעס (Nu) — כולל תסכול, מרירות, דיכאון מודחק
- **נוזל:** דמעות (Lei)
- **רקמה:** גידים ורצועות (Jin)
- **יסוד:** עץ (Mu)
- **עונה:** אביב
- **צבע:** ירוק/תכלת
- **טעם:** חמוץ

### 2.3 פתולוגיות נפוצות

- **קיפאון צ'י הכבד:** הדפוס הנפוץ ביותר — תחושת גוש בגרון, נפיחות בצדדים, עצבנות, מחזור לא סדיר
- **עליית יאנג הכבד:** כאבי ראש, סחרחורת, טנטון, פנים אדומות, כעס פתאומי
- **רוח הכבד:** רעד, עוויתות, שבץ, סחרחורת חמורה
- **חסר דם הכבד:** ראייה מטושטשת, עיניים יבשות, ציפורניים שבירות, מחזור מועט, חיוורון

## 3. הטחול (Pi)

הטחול הוא **"השר האחראי על התחבורה"** (Cang Lin Zhi Guan) — מרכז העיכול וייצור הצ'י.

### 3.1 תפקודים עיקריים

| תפקוד | הסבר |
|--------|-------|
| שולט בהמרה והובלה (Yun Hua) | מעכל את המזון, מפיק ממנו את תמציות המזון (Gu Qi) ומעביר אותן לכל הגוף |
| מעלה את הצ'י הטהור (Sheng Qing) | מעלה את התמציות הטהורות כלפי מעלה לריאות וללב |
| שולט בדם (Tong Xue) | שומר על הדם בתוך כלי הדם ומונע דימומים |
| שולט בשרירים ובגפיים | שרירים חזקים וגפיים חמות = טחול בריא |
| נפתח לפה (Kai Qiao Yu Kou) | חוש הטעם וצורת השפתיים משקפים את מצב הטחול |
| מאכלס את הי (Cang Yi) | הי — הכוונה, הריכוז, הלמידה והזיכרון |

### 3.2 שיוכים

- **רגש:** דאגה/הרהור יתר (Si)
- **נוזל:** רוק (Xian)
- **רקמה:** שרירים (Ji Rou)
- **יסוד:** אדמה (Tu)
- **עונה:** סוף קיץ / תקופות מעבר
- **צבע:** צהוב
- **טעם:** מתוק

### 3.3 פתולוגיות נפוצות

- **חסר צ'י הטחול:** עייפות, תיאבון ירוד, נפיחות, שלשול, שרירים חלשים
- **חסר יאנג הטחול:** כנ"ל + קור בבטן, גפיים קרות, שלשול עם מזון לא מעוכל
- **הטחול לא שולט בדם:** דימום כרוני, וסת מרובה, חבורות, דם בצואה
- **לחות טחול:** כבדות בגוף, ציפוי לשון שמן, בחילה, הפרשות מרובות

## 4. הריאות (Fei)

הריאות נקראות **"שר ממשלה ראשי"** (Xiang Fu Zhi Guan) — אחראיות על קצב וסדר.

### 4.1 תפקודים עיקריים

| תפקוד | הסבר |
|--------|-------|
| שולטות בצ'י ובנשימה | קולטות את צ'י הטבע ומשלבות אותו עם צ'י המזון ליצירת צ'י אמיתי |
| שולטות בפיזור (Xuan Fa) | מפזרות ווי צ'י (צ'י מגן) ונוזלים לפני השטח של הגוף |
| שולטות בהורדה (Su Jiang) | מורידות צ'י ונוזלים כלפי מטה — לכליות ולשלפוחית |
| מווסתות מעברי מים (Tong Tiao Shui Dao) | "המקור העליון של המים" — מפזרות ומורידות נוזלים |
| שולטות בעור ובשערות גוף (Zhu Pi Mao) | עור בריא ולח = ריאות חזקות |
| נפתחות לאף (Kai Qiao Yu Bi) | חוש הריח ונשימה תקינה דרך האף |
| מאכלסות את הפּוֹ (Cang Po) | הפּוֹ — הנשמה הגופנית, האינסטינקטים, תחושות הגוף |

### 4.2 שיוכים

- **רגש:** עצב/אבל (Bei / You)
- **נוזל:** ריר אף (Ti)
- **רקמה:** עור ושערות גוף (Pi Mao)
- **יסוד:** מתכת (Jin)
- **עונה:** סתיו
- **צבע:** לבן
- **טעם:** חריף

### 4.3 פתולוגיות נפוצות

- **חסר צ'י הריאות:** קוצר נשימה, קול חלש, שיעול חלש, הזעה ספונטנית, חיסון חלש
- **חסר ין הריאות:** שיעול יבש, גרון יבש, קול צרוד, חום בחמש, הזעות לילה
- **רוח-קור תוקפת את הריאות:** שיעול, ריר שקוף, גודש באף, צמרמורת
- **רוח-חום תוקפת את הריאות:** שיעול, ריר צהוב, כאב גרון, חום
- **ליחה חוסמת את הריאות:** שיעול עם כיח רב, לחות בחזה, קוצר נשימה

## 5. הכליות (Shen)

הכליות נקראות **"שורש החיים"** (Xian Tian Zhi Ben) — שורש ין ויאנג של כל הגוף.

### 5.1 תפקודים עיקריים

| תפקוד | הסבר |
|--------|-------|
| מאחסנות ג'ינג (Cang Jing) | מאחסנות את התמצית — הן של "לפני השמיים" (תורשתית) והן של "אחרי השמיים" (נרכשת) |
| שולטות במים (Zhu Shui) | מווסתות את חילוף החומרים של נוזלי הגוף, מפרידות טהור מעכור |
| קולטות צ'י (Na Qi) | "אוחזות" בצ'י שהריאות שולחות כלפי מטה — חיוניות לנשימה עמוקה |
| שולטות בעצמות ומייצרות מוח עצם | עצמות חזקות ומוח תקין תלויים בג'ינג הכליות |
| נפתחות לאוזניים (Kai Qiao Yu Er) | שמיעה טובה = כליות חזקות; טנטון = חסר כליות |
| מתבטאות בשיער הראש (Qi Hua Zai Fa) | שיער עבה ובוהק = ג'ינג מלא; שיער דליל ושיבה מוקדמת = חסר ג'ינג |
| מאכלסות את הג'י (Cang Zhi) | הג'י — כוח הרצון, ההתמדה והמוטיבציה |
| שולטות בשערי החיים (Ming Men) | שער החיים — מקור אש הכליות המחממת את כל הגוף |

### 5.2 שיוכים

- **רגש:** פחד (Kong) ובהלה (Jing)
- **נוזל:** רוק עבה (Tuo)
- **רקמה:** עצמות (Gu)
- **יסוד:** מים (Shui)
- **עונה:** חורף
- **צבע:** שחור / כהה
- **טעם:** מלוח

### 5.3 פתולוגיות נפוצות

- **חסר ין הכליות:** חום בחמש, הזעות לילה, סחרחורת, טנטון, כאבי גב, לשון אדומה
- **חסר יאנג הכליות:** קור בגפיים, כאבי גב, תכיפות שתן, אימפוטנציה, לשון חיוורת נפוחה
- **חסר ג'ינג הכליות:** התפתחות איטית, אינפרטיליות, הזדקנות מוקדמת, ירידה בזיכרון
- **הכליות לא אוחזות בצ'י:** קוצר נשימה חמור (קשה לשאוף), אסתמה כרונית

## נקודות חשובות לזכירה

1. **כל איבר זאנג מאחסן** ואינו מפריש — זו תכונת הין שלו
2. **כל איבר מזווג** עם איבר פו (ראו שיעור הבא) ועם יסוד מחמשת היסודות
3. **הקשר רגש-איבר הוא דו-כיווני**: רגש מוגזם פוגע באיבר, ופגיעה באיבר גורמת לחוסר איזון רגשי
4. **ברפואה הסינית "איבר" אינו רק איבר אנטומי** — הוא מערכת תפקודית רחבה
5. **הכליות הן השורש** של כל האיברים, כי הן מאחסנות את הג'ינג שממנו נובעים ין ויאנג של כל הגוף
