
# חמש נקודות שו — וו שו שיואה

## The Five Transporting (Shu) Points

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:

1. להסביר את מערכת חמש נקודות שו ואת האנלוגיה של זרימת המים
2. לזהות את חמש נקודות שו בכל 12 הערוצים העיקריים
3. להבין את הקשר בין נקודות שו לחמשת האלמנטים בערוצי ין ויאנג
4. ליישם אסטרטגיות טיפוליות כולל טוניפיקציה וסדציה לפי אם-בן
5. לבחור נקודות שו בהתאם לעונה ולמצב הקליני

---

## 1. מהן חמש נקודות שו?

### 1.1 הרעיון המרכזי

חמש נקודות שו (Wu Shu Xue) הן חמש נקודות מיוחדות הממוקמות בכל ערוץ עיקרי, **מקצות האצבעות/בהונות ועד למרפק/ברך**. הן מתארות את הדרך שבה הצ'י זורם בערוצים — כמו מים שמתחילים כמעיין קטן ומתרחבים לנהר ולים.

מערכת זו מתועדה לראשונה ב**לינג שו** (Ling Shu, פרק 2) וב**נאן ג'ינג** (Nan Jing, קשיים 64-68).

### 1.2 אנלוגיית המים — חמש השלבות


| פיניין | שם עברי | אנלוגיה | תיאור |
| ------ | ------------------- | ------------------ | --------------------------- |
| Jing | באר (ג'ינג-וול) | מעיין מתחת לאדמה | הצ'י מבצבץ כלפי חוץ |
| Ying | מעיין (יינג-ספרינג) | מים שמתחילים לזרום | הצ'י מתחיל לנוע, עדיין רדוד |
| Shu | נחל (שו-סטרים) | זרם שמתעמק | הצ'י "יורד ומשתפך" |
| Jing | נהר (ג'ינג-ריבר) | נהר רחב וחזק | הצ'י זורם בעוצמה |
| He | ים (חה-סי) | מים שמגיעים לים | הצ'י נכנס לעומק הגוף |


>
> "מקום שממנו [הצ'י] יוצא — באר; מקום שבו גולש — מעיין; מקום שאליו נשפך — נחל; מקום שבו הולך — נהר; מקום שאליו נכנס — ים"
> — לינג שו, פרק 2

## 2. קשר לחמשת האלמנטים

### 2.1 ערוצי ין (Zang — איברים מלאים)

בערוצי ין, סדר האלמנטים מתחיל מ**עץ** ונע לפי מחזור היצירה:


| נקודה | אלמנט |
| --------------- | ------------ |
| ג'ינג-וול | **עץ** |
| יינג-ספרינג | **אש** |
| שו-סטרים | **אדמה** |
| ג'ינג-ריבר | **מתכת** |
| חה-סי | **מים** |


### 2.2 ערוצי יאנג (Fu — איברים חלולים)

בערוצי יאנג, סדר האלמנטים מתחיל מ**מתכת**:


| נקודה | אלמנט |
| --------------- | ------------ |
| ג'ינג-וול | **מתכת** |
| יינג-ספרינג | **מים** |
| שו-סטרים | **עץ** |
| ג'ינג-ריבר | **אש** |
| חה-סי | **אדמה** |

## 4. שימושים קליניים של כל סוג נקודה

### 4.1 ג'ינג-וול — נקודות באר

- **מיקום**: קצוות האצבעות והבהונות
- **אלמנט ין**: עץ — מניע סטגנציה, מפזר רוח
- **אלמנט יאנג**: מתכת — מנקה חום
- **שימוש עיקרי**: מצבי חירום, אובדן הכרה, חום גבוה, שינויים נפשיים חדים
- **טכניקה**: לרוב דקירה להוצאת טיפת דם (Fang Xue)
- **דוגמה**: דקירת כל 10 ג'ינג-וול בידיים (Shi Xuan) להורדת חום או להחזרת הכרה

### 4.2 יינג-ספרינג — נקודות מעיין

- **מיקום**: קרוב לבסיס האצבעות/בהונות
- **שימוש עיקרי**: מנקות **חום** (Re) בערוץ ובאיבר
- **דוגמה**: LR2 (יינג-ספרינג של כבד) — מנקה חום-כבד, כאבי ראש, עיניים אדומות

### 4.3 שו-סטרים — נקודות נחל

- **מיקום**: אזור שורש כף היד/כף הרגל
- **שימוש עיקרי**: כבדות בגוף, כאבי פרקים, **רטיבות** (Shi)
- **בערוצי ין**: נקודת שו-סטרים היא גם **נקודת יואן** (Yuan Xue)
- **דוגמה**: LR3 (שו-סטרים וגם יואן של כבד) — משמשת לכמעט כל דפוס של כבד

### 4.4 ג'ינג-ריבר — נקודות נהר

- **מיקום**: אזור שורש כף היד/כף הרגל עד מרפק/ברך
- **שימוש עיקרי**: שיעול, קוצר נשימה, בעיות גרון, **קור ורוח** שפולשים
- **דוגמה**: LU8 (ג'ינג-ריבר של ריאות) — שיעול, אסטמה

### 4.5 חה-סי — נקודות ים

- **מיקום**: אזור מרפק/ברך
- **שימוש עיקרי**: מחלות **פו** (איברים חלולים — קיבה, מעיים), שלשולים, הקאות
- **כלל**: (He Zhi Nei Fu) — "נקודות ים מטפלות באיברים הפנימיים"
- **דוגמה**: ST36 (חה-סי של קיבה) — כל בעיות העיכול

## 6. שיטת אם-בן לטוניפיקציה וסדציה

### 6.1 העיקרון

שיטה זו מבוססת על מחזור היצירה של חמשת האלמנטים (Sheng):

- **לחזק (Bu)**: בחר את נקודת ה**אם** (האלמנט שמוליד את אלמנט הערוץ)
- **לנקז (Xie)**: בחר את נקודת ה**בן** (האלמנט שנולד מאלמנט הערוץ)

### 6.2 דוגמאות

**ערוץ הריאות** — אלמנט: מתכת

- אם של מתכת = **אדמה** → נקודת שו-סטרים = **LU9** (טאי יואן) → **לחיזוק**
- בן של מתכת = **מים** → נקודת חה-סי = **LU5** (צ'י דזה) → **לניקוז**

**ערוץ הכבד** — אלמנט: עץ

- אם של עץ = **מים** → בערוץ ין: חה-סי = **LR8** (צ'ו צ'ואן) → **לחיזוק**
- בן של עץ = **אש** → בערוץ ין: יינג-ספרינג = **LR2** (שינג ג'יאן) → **לניקוז**

**ערוץ הקיבה** — אלמנט: אדמה

- אם של אדמה = **אש** → בערוץ יאנג: ג'ינג-ריבר = **ST41** (ג'יה שי) → **לחיזוק**
- בן של אדמה = **מתכת** → בערוץ יאנג: ג'ינג-וול = **ST45** (לי דואי) → **לניקוז**

### 6.3 טבלה מסכמת — טוניפיקציה וסדציה בערוצי ין


| ערוץ | אלמנט ערוץ | נקודת חיזוק (אם) | נקודת ניקוז (בן) |
| ------ | ---------- | ---------------- | ---------------- |
| ריאות | מתכת | LU9 (אדמה) | LU5 (מים) |
| טחול | אדמה | SP2 (אש) | SP5 (מתכת) |
| לב | אש | HT9 (עץ) | HT7 (אדמה) |
| כליות | מים | KI7 (מתכת) | KI1 (עץ) |
| כבד | עץ | LR8 (מים) | LR2 (אש) |
| פריקרד | אש | PC9 (עץ) | PC7 (אדמה) |

## 7. שאלות לחזרה

1. הסבר את אנלוגיית המים של חמש נקודות שו — מדוע הצ'י דומה לזרימת מים?
2. מהו ההבדל בסדר האלמנטים בין ערוצי ין לערוצי יאנג?
3. באיזה סוג נקודות שו תשתמש לטיפול במצב חירום עם אובדן הכרה?
4. מטופל סובל מחום-כבד עם עיניים אדומות. איזו נקודת שו תבחר ומדוע?
5. הסבר את שיטת אם-בן: כיצד תחזק את ערוץ הכליות? כיצד תנקז?
6. לפי הטיפול העונתי, באיזה סוג נקודה מומלץ להשתמש בחורף?
