
# מתודולוגיית אבחון - גישה שיטתית

## Diagnostic Methodology - Systematic Approach

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לגשת לכל מטופל בצורה שיטתית ומאורגנת
2. לנהל תהליך אבחוני מהתחלה ועד אבחנה סופית
3. לזהות דגלים אדומים ולדעת מתי להפנות
4. לתעדף מידע ולהבחין בין עיקר לטפל

---

## 1. מבוא — מדוע שיטתיות חשובה?

מטפל מתחיל נוטה ל"קפוץ" מהסימפטום הראשון ישר לאבחנה. מטפל מנוסה **אוסף קודם, מנתח אחר כך**.

> (Bian Zheng Lun Zhi)
> "לזהות דפוסים ולטפל בהתאם"

זהו עיקרון היסוד של הרפואה הסינית הקלינית. אין "מתכון" קבוע לכל מחלה — הטיפול מותאם ל**דפוס** הייחודי של המטופל.

---

## 2. תרשים זרימה אבחוני

```
מטופל נכנס
    │
    ▼
[1] התבוננות ראשונית ──→ שן? צבע? מבנה? תנועה?
    │
    ▼
[2] שאילה (עשר השאלות) ──→ איסוף מידע מקיף
    │
    ▼
[3] בדיקת לשון ──→ גוף + ציפוי + אזורים
    │
    ▼
[4] בדיקת דופק ──→ עמדות + עומקים + איכויות
    │
    ▼
[5] מישוש (בטן, גב, ערוצים) ──→ רגישות, מתח, טמפרטורה
    │
    ▼
[6] בדיקת דגלים אדומים ──→ צריך הפניה?
    │ │
    │ לא │ כן → הפנו לרפואה מערבית
    ▼
[7] ניתוח ממצאים
    │
    ├── שמונת העקרונות
    ├── זיהוי איבר/ערוץ
    └── דפוס ספציפי
    │
    ▼
[8] עקרון טיפולי
    │
    ▼
[9] בחירת נקודות
    │
    ▼
[10] ביצוע טיפול ──→ מעקב ──→ התאמה
```

---

## 3. שלב 1: התבוננות ראשונית (Wang)

**ברגע שהמטופל נכנס**, שימו לב ל:

| מה לבדוק | מה לחפש |
|---|---|
| שן (רוח) | עיניים בהירות? מבט עמום? חיוניות? |
| הליכה | יציבה? צליעה? איטית? |
| צבע פנים | צבע דומיננטי? שינויים מקומיים? |
| קול | חזק? חלש? צרוד? |
| מצב רוח | מתוח? רגוע? עצוב? |

**תזכורת**: אל תשתפו את המטופל בהתרשמויות שלכם בשלב זה. פשוט **רשמו לעצמכם**.

## 4. שלב 2: שאילה שיטתית (Wen)

### 4.1 סדר מומלץ

1. **תלונה עיקרית** — "מה הביא אותך אלי היום?"
2. **היסטוריית התלונה** — "מתי התחיל? מה קדם? מה משתנה?"
3. **עשר השאלות** — סריקה שיטתית של כל מערכות הגוף
4. **היסטוריה רפואית** — מחלות, ניתוחים, תרופות
5. **אורח חיים** — תזונה, שינה, מתח, ספורט

### 4.2 שאלות פתוחות ואז ממוקדות

- **פתוחה**: "ספרו לי על השינה שלכם" → מאפשרת למטופל לחשוף מה שחשוב לו
- **ממוקדת**: "האם אתם מתעוררים בשעה מסוימת?" → מכוונת למידע ספציפי
- **כלל**: התחילו פתוח, צמצמו בהדרגה

## 5. שלב 3-5: בדיקה פיזית

### 5.1 לשון (ראו מודול 07-02)

בדקו: צבע גוף, צורה, תנועה, ציפוי (צבע, עובי, לחות), אזורים.

### 5.2 דופק (ראו מודול 07-05)

בדקו: שלוש עמדות, שלושה עומקים, קצב, איכות, השוואת ידיים.

### 5.3 מישוש נוסף

- **נקודות שו בגב** — רגישות?
- **נקודות מו בבטן** — רגישות?
- **ערוצים רלוונטיים** — גושים, מתח, רגישות?

---

## 6. שלב 6: דגלים אדומים (Hong Qi)

### 6.1 מתי להפנות לרפואה מערבית **מיידית**

| דגל אדום | חשד |
|---|---|
| כאב חזה חד עם קוצר נשימה | אוטם שריר הלב |
| כאב ראש פתאומי "הכי חזק בחיים" | דימום מוחי |
| חולשה/שיתוק חד-צדדי פתאומי | שבץ |
| כאב בטן חמור עם הגנה שרירית | בטן חריפה |
| חום גבוה (>39°) ממושך עם בלבול | זיהום חמור |
| ירידה פתאומית ולא מוסברת במשקל | ממאירות |
| דימום לא מוסבר | רפואי/כירורגי |

### 6.2 מתי להפנות לבירור (לא דחוף)

- ממצאים שלא מתאימים לאף דפוס ברפואה סינית
- מסות לא מוסברות
- תסמינים שלא משתפרים אחרי 6-8 טיפולים
- חשד לבעיה שדורשת אבחנה מערבית (צילום, בדיקות דם)

## 7. שלב 7: ניתוח ממצאים

### 7.1 ארגון המידע

**כלל הזהב**: אל תנסו לאבחן תוך כדי איסוף מידע. **אספו קודם — נתחו אחר כך**.

רשמו את כל הממצאים:
1. **מה המטופל אומר** (סימפטומים)
2. **מה אתם רואים** (התבוננות)
3. **מה הלשון אומרת**
4. **מה הדופק אומר**
5. **מה המישוש חושף**

### 7.2 חיפוש "החוט האדום"

בדרך כלל, רוב הממצאים מצביעים על **כיוון אחד**. חפשו את ה"חוט האדום" שמקשר בין הממצאים:

- האם רוב הממצאים מצביעים על חום או קור?
- האם רוב הממצאים מצביעים על עודף או חסר?
- איזה איבר חוזר בכמה ממצאים?

### 7.3 ניתוח לפי שמונת העקרונות

ארבעה צירים:
1. **פנים/חוץ** — האם הבעיה שטחית או עמוקה?
2. **חום/קור** — חם או קר?
3. **עודף/חסר** — יש יותר מדי או חסר?
4. **ין/יאנג** — סיכום כולל

## 8. שלב 8-9: מאבחנה לטיפול

(ראו מודולים 07ב-02 עד 07ב-04 לפירוט מלא)

**תהליך מקוצר**:
1. דפוס → "מה לא בסדר?"
2. עקרון טיפולי → "מה צריך לעשות?"
3. נקודות → "איפה לדקור?"

---

## 9. שלב 10: מעקב והתאמה

### 9.1 בכל טיפול חוזר, בדקו

- האם התלונה העיקרית השתפרה?
- האם הלשון השתנתה?
- האם הדופק השתנה?
- האם יש תלונות חדשות?

### 9.2 התאמת הטיפול

| מצב | פעולה |
|---|---|
| שיפור ברור | המשיכו באותו כיוון, אולי הפחיתו תדירות |
| שיפור חלקי | התאימו — אולי הדפוס העיקרי טופל, עכשיו עברו למשני |
| ללא שינוי | בדקו שוב — האבחנה נכונה? הנקודות מתאימות? |
| החמרה | עצרו. חשבו מחדש. התייעצו. |

## 10. ניהול זמן בביקור ראשון

| שלב | זמן מוצע |
|---|---|
| התבוננות ראשונית | 2 דקות |
| שאילה | 15-20 דקות |
| בדיקת לשון | 2 דקות |
| בדיקת דופק | 5 דקות |
| מישוש | 5 דקות |
| ניתוח ואבחנה | 5 דקות |
| הסבר למטופל | 3 דקות |
| טיפול (דיקור) | 20-30 דקות |
| **סה"כ** | **60-75 דקות** |

## 11. שאלות לחזרה

1. מהו סדר שלבי האבחון המומלץ?
2. מדוע חשוב לא לאבחן תוך כדי איסוף מידע?
3. פרט שלושה דגלים אדומים שמחייבים הפניה מיידית.
4. מהם שמונת העקרונות וכיצד הם עוזרים בניתוח?
5. מה לעשות כאשר מטופל לא משתפר אחרי מספר טיפולים?
