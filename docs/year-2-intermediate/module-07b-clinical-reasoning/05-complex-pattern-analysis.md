
# ניתוח דפוסים מורכבים

## Complex Pattern Analysis

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לזהות מצבים בהם קיימים שני דפוסים או יותר בו-זמנית
2. לקבוע סדרי עדיפויות בטיפול
3. לתכנן אסטרטגיה רב-שלבית
4. להתמודד עם סתירות בממצאים

---

## 1. מבוא — למה דפוסים מורכבים?

ברוב המקרים הקליניים, מטופלים **לא מציגים דפוס יחיד**. יש שילוב של 2-3 דפוסים, ולעיתים יותר. הסיבות:

- **מחלה כרונית** — עם הזמן, דפוס אחד "מוליד" דפוסים נוספים
- **חוקת חמשת האלמנטים** — איבר אחד משפיע על אחרים
- **גורמים מרובים** — מתח + תזונה לקויה + חוסר שינה
- **גיל** — עם הזמן, חסרים מצטברים

> (Jiu Bing Bi Xu, Jiu Xu Bi Yu)
> "מחלה ממושכת בהכרח יוצרת חסר; חסר ממושך בהכרח יוצר סטגנציית דם"

---

## 2. סוגי מורכבות

### 2.1 דפוסים בשרשרת

דפוס אחד **גורם** לאחר:

```
סטגנציית צ'י כבד ──→ אש כבד ──→ אש כבד פוגע בקיבה
     (שלב 1) (שלב 2) (שלב 3)
```

```
חסר צ'י טחול ──→ לחות ──→ פלגמה ──→ פלגמה חוסמת ריאות
    (שורש) (תוצאה) (התפתחות) (סימפטום)
```

### 2.2 דפוסים מקבילים

שני דפוסים **עצמאיים** שקיימים בו-זמנית:

```
חסר ין כליות ─┐
                ├──→ מטופל אחד עם שתי בעיות
חסר צ'י טחול ─┘
```

### 2.3 דפוסים מנוגדים

ממצאים **סותרים** — חום בחלק אחד וקור בחלק אחר:

- **חום למעלה, קור למטה**: פנים אדומות וכיבים בפה (חום) + רגליים קרות ושלשול (קור)
- **חום בפנים, קור בחוץ**: חום פנימי עם צמרמורת חיצונית
- **עודף וחסר מעורבים**: חסר טחול (חסר) + סטגנציית מזון (עודף)

---

## 3. קביעת סדרי עדיפויות

### 3.1 שאלות מנחות

1. **מה הדפוס העיקרי?** — מה הגורם הראשוני?
2. **מה הדפוס המשני?** — מה נגרם כתוצאה?
3. **מה דחוף?** — מה מפריע למטופל ביותר?
4. **מה השורש?** — טיפול בו ישפר הכל

### 3.2 כלל תעדוף

| עדיפות | מתי | דוגמה |
|---|---|---|
| **1 — דחוף** | סימפטום חריף מסכן חיים או מפריע מאוד | כאב חריף, שלשול חמור, חום גבוה |
| **2 — שורש** | הגורם הראשוני שמוליד את השאר | חסר טחול שגורם ללחות |
| **3 — משני** | דפוס שנגרם מהשורש | לחות שנגרמה מחסר טחול |
| **4 — שיורי** | ממצא קל שייפתר כשהשורש יטופל | סימפטומים קלים |

## 4. דפוסים מורכבים נפוצים

### 4.1 כבד תוקף טחול

**מנגנון**: סטגנציית צ'י כבד → הכבד "תוקף" את הטחול → הטחול נחלש

**ממצאים**:
- סימפטומי כבד: כאב צלעות, עצבנות, אנחות, מחזור לא סדיר
- סימפטומי טחול: נפיחות אחרי אכילה, שלשול (במיוחד בזמן מתח), עייפות
- **לשון**: צדדים אדומים, מרכז חיוור
- **דופק**: מתוח בגואן שמאל, חלש בגואן ימין

**עקרון**: החלק כבד + חזק טחול
**נקודות**: LR3, LR14, SP6, ST36, BL20, REN12

### 4.2 חסר ין כליות + אש כבד עולה (+)

**מנגנון**: ין כליות חסר → הכליות לא מזינות ין כבד → יאנג כבד עולה ללא שליטה

**ממצאים**:
- כליות: גב תחתון, ברכיים חלשות, הזעה לילית, חום אחה"צ
- כבד: כאב ראש, סחרחורת, כעס, עיניים אדומות, טעם מר
- **לשון**: אדומה, רזה, צדדים אדומים
- **דופק**: דק ומהיר, מתוח בגואן שמאל, חלש בצ'י

**עקרון**: תזן ין כליות + הרגע כבד, הורד אש
**נקודות**: KI3, KI6, SP6, LR3, GB20, GV20

### 4.3 חסר צ'י טחול + לחות-פלגמה (+)

**מנגנון**: טחול חלש → לא מנהל נוזלים → לחות מצטברת → מתעבה לפלגמה

**ממצאים**:
- טחול: עייפות, חוסר תיאבון, צואה רכה
- לחות/פלגמה: כבדות, ראש כבד, ליחה, סחרחורת, בחילות
- **לשון**: חיוורת, נפוחה, סימני שיניים, ציפוי לבן עבה שמנוני
- **דופק**: חלקלק, חלש

**עקרון**: חזק טחול + ייבש לחות + המס פלגמה
**נקודות**: ST36, SP6, BL20, REN12, ST40, REN9, SP9

### 4.4 חסר דם + סטגנציית דם

**מנגנון**: חסר דם כרוני → הדם הנותר אינו זורם היטב → סטגנציה

**ממצאים**:
- חסר דם: חיוורון, סחרחורת, עייפות, ראייה מטושטשת
- סטגנציית דם: כאב חד קבוע, שפתיים סגלגלות, מחזור עם קרישים כהים
- **לשון**: חיוורת עם כתמים סגולים
- **דופק**: דק, מחוספס

**עקרון**: תזן דם + הנע דם
**נקודות**: SP6, ST36, BL17 (הוי של הדם), SP10, LR3, BL18

### 4.5 חום למעלה + קור למטה

**מנגנון**: ין ויאנג לא מתקשרים → יאנג עולה (חום למעלה), ין שוקע (קור למטה)

**ממצאים**:
- למעלה: פנים אדומות, כיבים בפה, דפיקות לב, חרדה
- למטה: רגליים קרות, שלשול מימי, שתן תכוף בהיר
- **לשון**: קצה אדום, שורש חיוור עם ציפוי לבן
- **דופק**: מהיר בצון, איטי/חלש בצ'י

**עקרון**: נקה חום למעלה + חמם למטה, קשר ין ויאנג
**נקודות**: HT7 (נקה אש לב), KI3 (חזק כליות), REN4 + מוקסה (חמם למטה), PC6

### 4.6 חסר ין וחסר יאנג כליות

**מנגנון**: חסר ין כרוני → בסופו של דבר גם היאנג נפגע (או להפך)

**ממצאים**:
- ין: חום אחה"צ, הזעה לילית, גב תחתון כואב
- יאנג: רגליים קרות, שתן תכוף, שלשול בוקרי
- **לשון**: חיוורת עם קצות אדומות, או אדומה עם שורש רטוב
- **דופק**: חלש בצ'י בשתי הידיים

**עקרון**: תזן ין ויאנג כליות
**נקודות**: KI3, BL23, REN4, GV4, SP6

## 5. אסטרטגיות טיפול רב-שלביות

### 5.1 טיפול רציף

| שלב | מיקוד | משך |
|---|---|---|
| שלב 1 | טפלו בדחוף/ענף | 1-3 טיפולים |
| שלב 2 | עברו לשורש | 4-8 טיפולים |
| שלב 3 | תחזוקה | ביקורת חודשית |

### 5.2 טיפול סימולטני

כאשר אפשר, **טפלו בשניהם בו-זמנית**:
- חלק מהנקודות לדפוס העיקרי
- חלק מהנקודות לדפוס המשני
- דוגמה: כבד תוקף טחול → LR3 (כבד) + ST36 (טחול) באותו טיפול

### 5.3 טיפול מתחלף

בטיפול אחד — מתמקדים בדפוס א'. בטיפול הבא — בדפוס ב'. וחוזר חלילה.

## 6. סתירות בממצאים — כיצד להתמודד?

| סתירה | אפשרויות |
|---|---|
| לשון חיוורת + דופק מהיר | חסר דם (לשון) עם חום מחסר (דופק) |
| לשון אדומה + דופק איטי | חום פנימי כרוני עם קור שטחי |
| סימפטומי חום + סימפטומי קור | חום-קור מעורבים — נפוץ! |
| מטופל אומר "חם" + לשון חיוורת | תחושה סובייקטיבית ≠ מצב אובייקטיבי |

**כלל**: אל תתעלמו מסתירות. הן **מפתח** להבנת המורכבות.

## 7. שאלות לחזרה

1. מדוע רוב המטופלים מציגים יותר מדפוס אחד?
2. תארו שני סוגים שונים של מורכבות (שרשרת, מקבילים, מנוגדים).
3. כיצד נקבעים סדרי עדיפויות בטיפול בדפוסים מורכבים?
4. מהו המנגנון של "כבד תוקף טחול"? מה העקרון הטיפולי?
5. מטופל עם לשון חיוורת, נפוחה, סימני שיניים, אך גם צדדים אדומים ודופק מתוח — זהו שני הדפוסים.
