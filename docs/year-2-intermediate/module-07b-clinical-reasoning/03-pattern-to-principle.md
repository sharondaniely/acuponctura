
# מדפוס לעקרון טיפולי

## From Pattern to Treatment Principle

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לגזור עקרון טיפולי מדפוס מזוהה
2. לנסח עקרונות טיפול בשפה המקצועית של הרפואה הסינית
3. להבין את ההיגיון מאחורי כל עקרון
4. להבחין בין טיפול בשורש (בן) לטיפול בענף (ביאו)

---

## 1. מבוא — מדפוס לעקרון

לאחר שזיהינו את הדפוס, השלב הבא הוא לקבוע **עקרון טיפולי** (Zhi Ze). העקרון קובע את **הכיוון** של הטיפול — מה אנחנו רוצים להשיג.

**חוק יסוד**:
> (Han Zhe Re Zhi, Re Zhe Han Zhi)
> "את הקר — חממו, את החם — קררו"

> (Xu Ze Bu Zhi, Shi Ze Xie Zhi)
> "את החסר — חזקו, את העודף — פנו"

## 2. עקרונות טיפול כלליים

### 2.1 לפי שמונת העקרונות

| דפוס | עקרון טיפולי |
|---|---|
| חיצוני | שחרר את השטח |
| פנימי | טפל בפנים (לפי הדפוס הספציפי) |
| חום | נקה חום |
| קור | חמם |
| עודף | פנה/הפחת |
| חסר | חזק/תזן |

| עקרון | מתי |

| תזן ין | חסר ין |
| חמם יאנג | חסר יאנג |

## 3. עקרונות לפי דפוסי איברים — טבלה מקיפה

### 3.1 דפוסי לב

| דפוס | עקרון טיפולי |
|---|---|
| חסר דם לב | תזן דם, הרגע רוח הלב |
| חסר ין לב | תזן ין לב, נקה חום מחסר |
| חסר צ'י לב | חזק צ'י הלב |
| חסר יאנג לב | חמם יאנג הלב |
| אש לב | נקה אש לב, הרגע רוח |
| סטגנציית דם לב | הנע דם, פתח חסימות |
| פלגמה-אש חוסמת לב | נקה חום, המס פלגמה, פתח פתחי הלב |

### 3.2 דפוסי ריאות

| דפוס | עקרון טיפולי |
|---|---|
| רוח-קור בריאות | שחרר שטח, פזר קור, פתח ריאות |
| רוח-חום בריאות | שחרר שטח, נקה חום, פתח ריאות |
| חסר צ'י ריאות | חזק צ'י ריאות |
| חסר ין ריאות | תזן ין ריאות, הלח |
| פלגמה-לחות בריאות | ייבש לחות, המס פלגמה, הורד צ'י ריאות |

### 3.3 דפוסי כבד

| דפוס | עקרון טיפולי |
|---|---|
| סטגנציית צ'י כבד | החלק כבד, הנע צ'י |
| אש כבד | נקה אש כבד |
| רוח פנימית כבד | הרגע כבד, הרגע רוח |
| חסר דם כבד | תזן דם כבד |
| חסר ין כבד | תזן ין כבד |
| יאנג כבד עולה | הרגע כבד, הורד יאנג |
| לחות-חום כבד/שק מרה | נקה חום, ייבש לחות בכבד/שק מרה |

### 3.4 דפוסי טחול

| דפוס | עקרון טיפולי |
|---|---|
| חסר צ'י טחול | חזק צ'י טחול |
| חסר יאנג טחול | חמם יאנג טחול, חזק טחול |
| צ'י טחול שוקע | העלה יאנג טחול |
| טחול לא מחזיק דם | חזק טחול, החזק דם |
| לחות-קור טחול | חמם ויבש, חזק טחול |
| לחות-חום טחול | נקה חום, ייבש לחות |

### 3.5 דפוסי כליות

| דפוס | עקרון טיפולי |
|---|---|
| חסר ין כליות | תזן ין כליות |
| חסר יאנג כליות | חמם יאנג כליות |
| חסר ג'ינג כליות | חזק ג'ינג, תזן כליות |
| כליות לא אוחזות צ'י | חזק כליות, אחוז צ'י |
| כליות לא שולטות במים | חמם כליות, הנע מים |

## 4. שורש וענף (Ben Biao)

### 4.1 עיקרון

- **בן** = השורש, הסיבה העמוקה
- **ביאו** = הענף, הסימפטום הגלוי

**כלל**: בדרך כלל, טפלו **בשורש** (בן). אך במצב חריף — טפלו **בענף** (ביאו) קודם.

### 4.2 דוגמאות

| מצב | שורש (בן) | ענף (ביאו) | מה לטפל קודם? |
|---|---|---|---|
| חסר טחול + שלשול חריף | חסר טחול | שלשול | ענף — עצרו שלשול, אז חזקו טחול |
| חסר ין כליות + כאב גב כרוני | חסר ין | כאב | שורש — תזנו ין, הכאב ייפסק |
| סטגנציית כבד + כאב ראש חריף | סטגנציית כבד | כאב ראש | ענף — הקלו כאב ראש, אז טפלו בכבד |
| חסר דם + נדודי שינה כרוניים | חסר דם | נדודי שינה | שורש — תזנו דם, השינה תשתפר |

### 4.3 כלל מעשי

> (Ji Ze Zhi Biao, Huan Ze Zhi Ben)
> "בחריף — טפלו בענף; בכרוני — טפלו בשורש"

לעיתים: **טפלו בשניהם בו-זמנית** (Biao Ben Tong Zhi) — כאשר גם השורש וגם הענף דחופים, או כאשר הם קשורים הדדית.

## 5. ניסוח עקרון טיפולי — מבנה מומלץ

**פורמט**: [פעולה] + [מה] + [של מה]

דוגמאות:
- **חזק** צ'י **של הטחול**
- **תזן** ין **של הכליות**
- **החלק** צ'י **של הכבד**
- **נקה** חום **מהקיבה**
- **הנע** דם, **פתח** חסימות

**עקרון מורכב**: כאשר יש שני דפוסים:
- **חזק** צ'י טחול + **החלק** צ'י כבד
- **תזן** ין כליות + **הורד** אש כבד

---

## 6. דוגמאות שלמות

### דוגמה 1
- **דפוס**: סטגנציית צ'י כבד
- **עקרון**: החלק כבד, הנע צ'י
- **הסבר**: הצ'י של הכבד "תקוע" → צריך לגרום לו לזרום שוב

### דוגמה 2
- **דפוס**: חסר צ'י טחול + לחות (+)
- **עקרון**: חזק טחול, ייבש לחות
- **הסבר**: הטחול חלש → לחות מצטברת → חזק את הטחול והוא ינהל את הלחות

### דוגמה 3
- **דפוס**: חסר ין כליות עם אש עולה
- **עקרון**: תזן ין כליות, נקה חום מחסר, הורד אש
- **הסבר**: ין חסר → אין מה לאזן את האש → האש עולה → תזנו ין וגם הורידו אש

## 7. שאלות לחזרה

1. מהו ה"חוק היסודי" של עקרונות הטיפול ברפואה סינית?
2. מהו ההבדל בין שורש (בן) לענף (ביאו)?
3. מתי נטפל בענף לפני השורש?
4. נסחו עקרון טיפולי לכל אחד: א) חסר יאנג כליות, ב) אש כבד, ג) פלגמה-לחות בריאות.
5. כיצד ננסח עקרון טיפולי כאשר יש שני דפוסים?
