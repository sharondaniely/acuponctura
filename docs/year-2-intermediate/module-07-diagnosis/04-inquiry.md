
# שאילה - וון - עשר השאלות

## Inquiry - Wen Zhen - The Ten Questions

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לנהל ראיון אבחוני שיטתי על פי "עשר השאלות" (Shi Wen)
2. לפרש את תשובות המטופל ולקשרן לדפוסים פתולוגיים
3. להבחין בין ממצאים של עודף לממצאי חסר בכל שאלה
4. לבנות תמונה אבחונית מקיפה משילוב התשובות

---

## 1. מבוא — שיטת השאילה

השאילה (וון) היא השיטה **המקיפה ביותר** מבין ארבע שיטות האבחון. היא מאפשרת לנו לגשת למידע שלא ניתן להשיג בהתבוננות, האזנה או מישוש — תחושות סובייקטיביות, היסטוריה רפואית, והרגלי חיים.

ג'אנג ג'ינג יואה (Zhang Jing Yue) מתקופת שושלת מינג סיכם את השאילה בשיר:

>
>
>
>

"שאלה ראשונה — קור וחום, שנייה — הזעה, שלישית — ראש וגוף, רביעית — הפרשות..."

---

## 2. השאלה הראשונה: קור וחום (Han Re)

### 2.1 צמרמורת וחום

| ממצא | משמעות |
|---|---|
| **צמרמורת + חום בו-זמנית** | התקפה חיצונית (ביאו ג'נג) |
| **צמרמורת חזקה + חום קל** | רוח-קור חיצוני (פנג האן) |
| **חום חזק + צמרמורת קלה** | רוח-חום חיצוני (פנג רה) |
| **חום בלבד, ללא צמרמורת** | חום פנימי (ליי רה) |
| **צמרמורת בלבד, ללא חום** | קור פנימי (ליי האן) |
| **חום לסירוגין** | שאו יאנג, או מלריה |
| **חום אחר הצהריים/ערב** | חסר ין |
| **חום נמוך כרוני** | חסר צ'י |

### 2.2 תחושת קור/חום סובייקטיבית

- **תחושת קור שלא מוקלת בכיסוי** → קור פנימי (חסר יאנג)
- **תחושת קור שמוקלת בכיסוי** → התקפה חיצונית
- **חום בכפות ידיים ורגליים + חזה** ("חום חמש לבבות") → חסר ין

## 3. השאלה השנייה: הזעה (Han)

| ממצא | משמעות |
|---|---|
| **הזעה עם התקפה חיצונית** | חסר ווי צ'י, רוח-קור של טאי יאנג "חסר" |
| **ללא הזעה עם התקפה חיצונית** | רוח-קור חסם את פני השטח, טאי יאנג "עודף" |
| **הזעה ספונטנית ביום** (Zi Han) | חסר צ'י, חסר יאנג |
| **הזעה לילית** (Dao Han) | חסר ין |
| **הזעה בראש בלבד** | לחות-חום, עלייה יאנג |
| **הזעה בכפות ידיים ורגליים** | חסר ין, לחות-חום בטחול |
| **הזעה בחצי גוף בלבד** | חסימת ערוצים (צד אחד) — סכנת שבץ |
| **הזעה מרובה + קריסה** | קריסת יאנג |
| **זיעה שמנונית וקרה** | קריסת יאנג — מצב חירום |

## 4. השאלה השלישית: כאבי ראש וגוף (Tou Shen Tong)

### 4.1 כאבי ראש

| מיקום כאב ראש | ערוץ | איבר |
|---|---|---|
| מצח | ערוץ יאנג מינג | קיבה |
| צדדים (רקות) | ערוץ שאו יאנג | שק מרה |
| קודקוד | ערוץ ג'ואה ין | כבד |
| עורף | ערוץ טאי יאנג | שלפוחית |
| כל הראש | — | רוח חיצונית, או עליית יאנג כבד |

### 4.2 אופי הכאב

| אופי | משמעות |
|---|---|
| כאב חד, קבוע במקום | סטגנציית דם |
| כאב נודד | רוח (פנג) |
| כאב כבד, "כמו מטלית רטובה" | לחות (שי) |
| כאב עם תחושת מתיחה | סטגנציית צ'י |
| כאב שמוקל בלחיצה | חסר |
| כאב שמחמיר בלחיצה | עודף |

## 5. השאלה הרביעית: שתן וצואה (Er Bian)

### 5.1 צואה

| ממצא | משמעות |
|---|---|
| עצירות יבשה | חום, חסר ין, חסר דם |
| עצירות ללא יובש | סטגנציית צ'י כבד |
| שלשול עם ריח חריף | לחות-חום |
| שלשול מימי, ללא ריח | חסר יאנג טחול, קור |
| שלשול בוקרי (5:00 בבוקר) | חסר יאנג כליות ("שלשול חמישית המשמרת") |
| צואה עם מזון לא מעוכל | חסר צ'י טחול |
| מיתחלף עצירות-שלשול | סטגנציית צ'י כבד תוקף טחול |

### 5.2 שתן

| ממצא | משמעות |
|---|---|
| שתן בהיר ומרובה | קור, חסר יאנג |
| שתן כהה ומועט | חום, לחות-חום |
| הטלת שתן תכופה | חסר צ'י כליות, חסר יאנג |
| הטלה כואבת וקשה | לחות-חום בשלפוחית |
| הטלת שתן בלילה | חסר יאנג כליות |
| דם בשתן | חום בדם, חום לחות בשלפוחית |

## 6. השאלה החמישית: מזון, שתייה וטעם (Yin Shi Kou Wei)

### 6.1 תיאבון

| ממצא | משמעות |
|---|---|
| תיאבון טוב, עיכול תקין | טחול/קיבה בריאים |
| חוסר תיאבון | חסר צ'י טחול, לחות |
| רעב מתמיד, אוכל הרבה | חום בקיבה |
| רעב אבל לא רוצה לאכול | חסר ין קיבה |
| סלידה מאוכל | סטגנציית מזון, הריון |
| תשוקה למאכלים מסוימים | רמז לאיבר (מתוק=טחול, חמוץ=כבד) |

### 6.2 צמא ושתייה

| ממצא | משמעות |
|---|---|
| ללא צמא | קור, ללא חום |
| צמא למים קרים | חום פנימי |
| צמא למים חמים בלגימות קטנות | חסר ין, או לחות |
| שותה הרבה ומשתין הרבה | סוכרת, חום בריאות/קיבה |
| צמא אך לא רוצה לשתות | לחות-חום (המים אינם נספגים) |

### 6.3 טעם בפה

| טעם | משמעות |
|---|---|
| מר | חום, אש כבד/לב |
| מתוק/דביק | לחות-חום טחול |
| חמוץ | סטגנציית צ'י כבד, חום כבד |
| מלוח | חסר כליות |
| חסר טעם | חסר צ'י טחול |

## 7. השאלה השישית: חזה ובטן (Xiong Fu)

| ממצא | משמעות |
|---|---|
| לחץ/מלאות בחזה | סטגנציית צ'י, פלגמה |

| כאב בטן שמוקל בחום | קור |
| כאב בטן שמוקל בקור | חום |

## 9. השאלה השמינית: היסטוריה רפואית (Jiu Bing)

שאלות חשובות:
- מחלות ילדות?
- ניתוחים?
- מחלות כרוניות?
- תרופות קבועות?
- היסטוריה משפחתית?
- חיסונים?

**חשיבות**: מחלות עבר משפיעות על הדפוס הנוכחי. למשל — מחלת חום חמורה בילדות עלולה לפגוע בין של הכליות לכל החיים.

---

## 10. השאלה התשיעית: גורם המחלה הנוכחית (Bing Yin)

שאלות מפתח:
- **מתי התחילה הבעיה?**
- **מה קדם לה?** (אירוע רגשי? חשיפה למזג אוויר? שינוי תזונה? פציעה?)
- **מה מחמיר ומה מקל?** (מנוחה? תנועה? חום? קור? אכילה?)
- **כיצד התפתחה?** (הדרגתית? פתאומית?)

### טבלת גורמים אבחוניים

| גורם מקל/מחמיר | משמעות |
|---|---|
| מחמיר בקור, מוקל בחום | דפוס קור |
| מחמיר בחום, מוקל בקור | דפוס חום |
| מחמיר במנוחה | סטגנציה |
| מוקל במנוחה | חסר |
| מחמיר בלחץ | עודף |
| מוקל בלחץ | חסר |
| מחמיר אחרי אכילה | עודף, סטגנציית מזון |
| מוקל אחרי אכילה | חסר |
| מחמיר במתח | סטגנציית צ'י כבד |

## 11. השאלה העשירית: לנשים — מחזור והפרשות (Jing Dai)

### 11.1 מחזור חודשי

| ממצא | משמעות |
|---|---|
| מחזור מוקדם (מקדים) | חום בדם, חסר צ'י (לא מחזיק דם) |
| מחזור מאוחר (מאחר) | קור בדם, סטגנציית דם, חסר דם |
| מחזור לא סדיר | סטגנציית צ'י כבד |
| דימום מרובה | חום בדם, חסר צ'י |
| דימום מועט | חסר דם, סטגנציית דם, קור |
| צבע אדום כהה + קרישים | סטגנציית דם + חום |
| צבע ורוד בהיר | חסר דם, חסר צ'י |
| כאבי מחזור לפני הדימום | סטגנציית צ'י/דם (עודף) |
| כאבי מחזור אחרי הדימום | חסר דם/צ'י (חסר) |

### 11.2 הפרשה נרתיקית (Dai Xia)

| הפרשה | משמעות |
|---|---|
| לבנה, דלילה, ללא ריח | חסר טחול, לחות-קור |
| צהובה, סמיכה, עם ריח | לחות-חום |
| אדומה-חומה | חום בדם, חום רעיל |

## 13. שאלות לחזרה

1. פרט את "עשר השאלות" וציין מה כל שאלה חושפת.
2. מה ההבדל בין הזעה ספונטנית להזעה לילית?
3. מטופל עם כאב ראש בצדדים, טעם מר בפה, ומחזור לא סדיר — מה הדפוס הסביר?
4. מה המשמעות של "שלשול חמישית המשמרת"?
5. מדוע חשוב לשאול "מה מקל ומה מחמיר"?
