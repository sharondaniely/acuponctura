# מקרים רב-דפוסיים

## Multi-Pattern Cases (复杂辨证)

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לזהות ולנתח מקרים קליניים עם 2-3 דפוסים סימולטניים
2. לקבוע סדר עדיפויות: טיפול בשורש, בענף, או בשניהם
3. ליישם אסטרטגיות טיפול רצפי מול סימולטני
4. לנתח 5 מקרי בוחן מורכבים ולהציע פרוטוקול טיפולי

---

## 1. עקרונות באבחון רב-דפוסי

### 1.1 מדוע מופיעים מספר דפוסים?

בפרקטיקה הקלינית, רוב המטופלים מציגים יותר מדפוס אחד:
- **התפתחות טבעית** — דפוס ראשוני יוצר דפוסים משניים (למשל: סטגנציית צ'י כבד → חום כבד → חסר ין כבד)
- **חוקתיות + פתוגן** — חולשה חוקתית + גורם חיצוני/רגשי
- **גיל** — עם הזמן מצטברים מספר חוסרי איזון
- **מחלות כרוניות** — מייצרות "שכבות" של פתולוגיה

### 1.2 כלל השורש והענף (标本 Biao Ben)

| מצב | אסטרטגיה | דוגמה |
|---|---|---|
| **ענף חריף** | טפל בענף קודם | כאב חד בגב + חסר כליות כרוני → קודם שחרר כאב |
| **שורש חמור** | טפל בשורש קודם | חסר צ'י חמור + הצטננות קלה → קודם חזק צ'י |
| **שניהם חמורים** | טפל בשניהם בו-זמנית | דמם רחמי חריף + חסר דם → עצור דמם + חזק דם |
| **שניהם כרוניים** | טפל בשורש, התייחס לענף | סטגנציית צ'י + ליחה → הזז צ'י (שורש) + המס ליחה (ענף) |

### 1.3 קביעת סדר עדיפויות

שאלות מנחות:
1. **מה מסכן?** — מצב חירום תמיד קודם
2. **מה הכי מפריע למטופל?** — שילוב של צורך קליני ורצון המטופל
3. **מה השורש?** — מה גרם למה? הגורם הראשוני חשוב יותר
4. **מה מזין מה?** — אם דפוס A מחזיק דפוס B, טפל ב-A

## 2. אסטרטגיות טיפול

### 2.1 טיפול רצפי (Sequential)

- טיפול בדפוס אחד בכל פגישה
- מתאים כשהדפוסים עצמאיים
- יתרון: ממוקד, ברור
- חיסרון: איטי

### 2.2 טיפול סימולטני (Simultaneous)

- התייחסות למספר דפוסים באותה פגישה
- בחירת נקודות ש"עובדות כפול"
- יתרון: יעיל, מהיר
- חיסרון: פחות ממוקד, דורש ניסיון

### 2.3 נקודות רב-תפקודיות

נקודות שפועלות על מספר דפוסים בו-זמנית:

| נקודה | דפוסים שמטפלת בהם |
|---|---|
| **SP6** (三阴交) | חסר דם + סטגנציית צ'י כבד + חסר ין כליות |
| **ST36** (足三里) | חסר צ'י טחול + לחות + חולשה כללית |
| **LV3** (太冲) | סטגנציית צ'י כבד + עליית יאנג כבד + חום כבד |
| **KI3** (太溪) | חסר ין כליות + חסר יאנג כליות + חסר ג'ינג |
| **CV4** (关元) | חסר צ'י + חסר יאנג + חסר דם |

## 3. מקרה בוחן 1 — מיגרנה עם עייפות

### 3.1 מטופלת

אישה, 42, מיגרנות חוזרות + עייפות כרונית + מחזור לא סדיר

### 3.2 ניתוח דפוסים

| דפוס | סימנים | משמעות |
|---|---|---|
| **עליית יאנג כבד** | כאבי ראש צידיים, עצבנות, סחרחורת | ענף — הסימפטום העיקרי |
| **חסר דם כבד** | עייפות, חיוורון, מחזור דל | שורש — כבד לא מוזן |
| **חסר צ'י טחול** | עייפות, עיכול חלש, נפיחות | שורש עמוק — לא מייצר מספיק דם |

### 3.3 אסטרטגיה

**שלב 1** (4-6 טיפולים): הרגעת יאנג כבד + חיזוק דם
- GB20, Taiyang, LV3 (יאנג כבד), SP6, ST36 (חיזוק דם וטחול)

**שלב 2** (6-8 טיפולים): חיזוק טחול ודם
- ST36, SP6, BL20, BL17, CV4

## 8. עקרונות מעשיים

### 8.1 טיפים לטיפול רב-דפוסי

- **אל תנסו לטפל בהכל בטיפול אחד** — עדיף ממוקד מאשר מפוזר
- **השתמשו בנקודות רב-תפקודיות** — SP6, ST36, LV3 פועלות על מספר דפוסים
- **הקשיבו למטופל** — מה הכי מפריע לו? התחילו משם
- **תעדו** — רשמו את הדפוסים, הנקודות והתגובות בכל טיפול
- **בדקו מחדש** — כל 4-6 טיפולים, העריכו מחדש את הדפוסים
- **שאלו**: "מה ה-80/20?" — 20% מהנקודות עושים 80% מהעבודה

## 9. שאלות לחזרה

1. הסבר את עקרון "שורש וענף" — מתי מטפלים בכל אחד?
2. מהן נקודות רב-תפקודיות? תן 3 דוגמאות.
3. בחר אחד ממקרי הבוחן ותאר את הרציונל הטיפולי.
4. מה ההבדל בין טיפול רצפי לסימולטני? מתי תעדיף כל גישה?
5. כיצד קובעים איזה דפוס הוא "השורש" כשיש מספר דפוסים?
