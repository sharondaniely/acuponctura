# מנגנוני פעולה של אקופונקטורה

## Mechanisms of Action of Acupuncture (针灸作用机制)

---

## מטרות למידה

בסיום שיעור זה, הסטודנט יוכל:
1. לתאר את המנגנונים הנוירולוגיים של שיכוך כאב בדיקור
2. להסביר את תורת רקמת החיבור/פאסיה
3. לתאר את ההשפעות על המערכת האוטונומית
4. להכיר מנגנוני אנטי-דלקתיים
5. לפרש ממצאי fMRI ומחקרי הדמיה
6. להסביר את תופעת דה-צ'י ומתאמים עצביים

---

## 1. מנגנונים נוירולוגיים

### 1.1 תיאוריית השער (Gate Control Theory)

מלוויין ווול (Melzack & Wall, 1965):
- **העיקרון**: גירוי סיבי עצב עבים (Aβ) חוסם את העברת כאב דרך סיבים דקים (Aδ, C)
- **רלוונטיות**: דקירת מחט מפעילה סיבי Aβ ו-Aδ → "סוגרת את השער" לכאב
- **מגבלה**: מסביר רק שיכוך כאב מקומי ומיידי

### 1.2 מנגנון אנדורפינים

ההאן (Han Ji-Sheng) ומחקרים נוספים:
- דיקור גורם לשחרור **אנדורפינים** (Endorphins) במוח ובחוט השדרה
- **אנקפלינים** (Enkephalins): שיכוך כאב מהיר
- **בטא-אנדורפין**: שיכוך כאב ממושך + תחושת רווחה
- **דינורפין** (Dynorphin): שיכוך כאב עמוק
- **ראייה**: נלוקסון (חוסם אופיואידי) מבטל חלקית את אפקט השיכוך → מוכיח מעורבות אופיואידית

### 1.3 תלות תדר באלקטרו-דיקור

| תדר | נוירוטרנסמיטר | אפקט |
|---|---|---|
| **2 Hz (נמוך)** | אנקפלין, בטא-אנדורפין | שיכוך כאב כרוני, הרגעה |
| **15 Hz (בינוני)** | שילוב | אפקט מעורב |
| **100 Hz (גבוה)** | דינורפין | שיכוך כאב חריף, ספסטיות |
| **2/100 Hz (מעורב)** | כל הנוירוטרנסמיטרים | שיכוך כאב מיטבי |

### 1.4 סרוטונין ונוראדרנלין

- דיקור מעלה רמות **סרוטונין** (5-HT) במוח
- מעלה **נוראדרנלין** במסלולי עיכוב כאב יורדים
- רלוונטי לטיפול בדיכאון, חרדה ומצבי כאב כרוני

### 1.5 DNIC - עיכוב כאב מפוזר (Diffuse Noxious Inhibitory Control)

- גירוי כואב בנקודה אחת → עיכוב כאב באזור אחר
- מוחי → מפעיל מסלולי עיכוב יורדים מגזע המוח
- מסביר מדוע נקודות רחוקות (כמו LI4 לכאב שיניים) עובדות

## 2. תיאוריית רקמת החיבור והפאסיה

### 2.1 הלאנגוין (Helene Langevin)

פרופ' הלן לאנגוויין מאוניברסיטת ורמונט:
- גילתה שנקודות דיקור ממוקמות לרוב **במפגשי רקמות חיבור**
- סיבוב המחט → רקמת חיבור נכרכת סביב המחט (Winding)
- גורם ל**מתיחה מכנית** (Mechanical Stretch) של תאי פיברובלסט

### 2.2 תגובת הפיברובלסט

- מתיחה מכנית → שחרור **ATP** (אדנוזין טריפוספט)
- ATP מפורק ל**אדנוזין** → אפקט אנטי-דלקתי ושיכוך כאב
- הפיברובלסטים משנים צורה ומשחררים גורמי צמיחה
- **רמודלינג**: שינוי מבני ברקמת חיבור לאורך זמן

### 2.3 רשת הפאסיה

- הפאסיה (Fascia) היא רשת רציפה בכל הגוף
- ערוצי הדיקור עשויים להתאים למישורי פאסיה
- גירוי בנקודה אחת → העברת אות מכני לאורך הפאסיה
- מסביר פוטנציאלית את תופעת "התפשטות התחושה" לאורך ערוץ

## 3. מערכת עצבים אוטונומית

### 3.1 השפעה פאראסימפתטית

דיקור מפעיל את **עצב הוואגוס** (Vagus Nerve):
- הורדת קצב לב
- שיפור עיכול (פריסטלטיקה)
- ירידה בלחץ דם
- הפעלת "מצב מנוחה ועיכול" (Rest & Digest)

### 3.2 הרפלקס האנטי-דלקתי הכולינרגי

| שלב | תיאור |
|---|---|
| 1 | דקירה מגרה סיבי עצב אפרנטיים |
| 2 | אות מגיע לגזע המוח (NTS) |
| 3 | הפעלת עצב הוואגוס (אפרנטי) |
| 4 | שחרור אצטילכולין בטחול ובאיברים |
| 5 | עיכוב שחרור ציטוקינים דלקתיים (TNF-α, IL-6) |

### 3.3 אפקטים סגמנטליים

דקירה באזור ספציפי → השפעה על **סגמנט** חוט השדרה המתאים:
- דקירה בגב (BL נקודות) → השפעה על איבר פנימי באותו סגמנט
- מסביר חלקית את מנגנון נקודות שו בגב (Back Shu Points)
- T1-T5 → לב, ריאות | T7-T10 → כבד, קיבה | L1-L3 → כליה, מעיים

## 4. מנגנונים אנטי-דלקתיים

### 4.1 ציטוקינים

מחקרים מראים שדיקור:
- **מפחית**: TNF-α, IL-1β, IL-6 (ציטוקינים פרו-דלקתיים)
- **מעלה**: IL-10, IL-4 (ציטוקינים אנטי-דלקתיים)
- מנגנון: הפעלת ציר HPA + רפלקס כולינרגי

### 4.2 ציר HPA

- דיקור מפעיל את ציר **ההיפותלמוס-היפופיזה-אדרנל**
- שחרור ACTH → קורטיזול → אפקט אנטי-דלקתי
- במיוחד בנקודות: ST36, LI4, LI11

### 4.3 אדנוזין

- גולדמן (Goldman, 2010): דיקור ב-ST36 → עלייה של **24 מונים** באדנוזין מקומי
- אדנוזין → פעולה אנטי-דלקתית ומשככת כאב
- מסביר את ההשפעה המקומית של הדיקור

---

## 5. מחקרי הדמיה מוחית

### 5.1 fMRI — הדמיה פונקציונלית

מחקרי fMRI גילו שדיקור:
- **מפעיל**: אזורים במוח הקשורים לעיבוד כאב ולוויסות אוטונומי
- **מכבה**: אזורים הקשורים לתפיסת כאב (מטריקס הכאב)
- **ספציפיות נקודות**: נקודות שונות מפעילות אזורי מוח שונים

### 5.2 ממצאים ספציפיים

| נקודה | אזורי מוח מופעלים | רלוונטיות |
|---|---|---|
| **LI4** | קורטקס סומטוסנסורי, אינסולה, PAG | שיכוך כאב |
| **LV3** | היפותלמוס, אמיגדלה | ויסות רגשי |
| **ST36** | היפותלמוס, גזע מוח, אינסולה | ויסות אוטונומי |
| **GB34** | קורטקס מוטורי, גרעיני בסיס | תפקוד מוטורי |

### 5.3 Default Mode Network

- דיקור משפיע על ה-Default Mode Network (DMN)
- רלוונטי למצבי כאב כרוני, דיכאון, חרדה
- שינויים בקישוריות מוחית (Brain Connectivity)

## 6. דה-צ'י ומתאמים עצביים

### 6.1 מהי דה-צ'י (得气)?

תחושה ייחודית של הדיקור:
- כבדות, לחץ, נימול, כאב עמום, חום, זרם
- נחשבת בעיני TCM כסימן ש"הצ'י הגיע"

### 6.2 בסיס נוירולוגי

| תחושה | סיב עצבי | סוג |
|---|---|---|
| כבדות/לחץ | Aδ (III) | מיילינציה דקה |
| נימול | Aβ (II) | מיילינציה עבה |
| כאב עמום | C (IV) | ללא מיילין |
| חום | C, Aδ | תרמורצפטורים |
| זרם/התפשטות | — | העברה לאורך רקמת חיבור? |

### 6.3 האם דה-צ'י הכרחי?

- **גישת TCM**: כן — "אם הצ'י לא הגיע, הטיפול אינו יעיל"
- **מחקר**: תמונה מעורבת — יש מחקרים שמראים יעילות גם ללא דה-צ'י
- **דקירה רדודה**: אפקטיבית בחלק מהמקרים (שיטה יפנית, בטנית)

## 7. סיכום אינטגרטיבי

### 7.1 מודל רב-שכבתי

הדיקור פועל בו-זמנית על מספר רמות:

| רמה | מנגנון |
|---|---|
| **מקומי** | אדנוזין, ATP, דלקת מקומית, פאסיה |
| **סגמנטלי** | רפלקס חוט שדרה, עצבים אוטונומיים |
| **מרכזי** | אנדורפינים, סרוטונין, DNIC, fMRI |
| **מערכתי** | ציר HPA, רפלקס כולינרגי, ציטוקינים |

### 7.2 מסר מרכזי

אין מנגנון אחד שמסביר את כל אפקטי הדיקור — זוהי מערכת רב-שכבתית המשלבת אותות עצביים, הורמונליים, מכניים ואימונולוגיים.

---

## נקודות מפתח לזכור

- אנדורפינים: 2 Hz = אנקפלין + בטא-אנדורפין; 100 Hz = דינורפין
- לאנגוויין: סיבוב מחט → כריכת רקמת חיבור → שחרור ATP → אדנוזין
- רפלקס כולינרגי: דיקור → וואגוס → אצטילכולין → ירידה בדלקת
- fMRI: נקודות שונות מפעילות אזורי מוח ספציפיים ושונים
- מודל רב-שכבתי: מקומי + סגמנטלי + מרכזי + מערכתי
